01
Electrical Nature · 전기적 성질
화학 결합의 전기적 성질
공유 결합·이온 결합 모두 전자가 관여 · 물 전기분해 · 염화나트륨 전기분해
▼
화학 결합이란?
🔵 화학 결합 (Chemical Bond)
세상에는 셀 수 없이 많은 물질이 존재하지만, 물질을 구성하는 원소는 90여 종에 불과합니다. 이 원소들이 결합을 형성해 다양한 물질을 만듭니다.

• 비금속 원자끼리: 전자쌍을 공유 → 공유 결합
• 금속 + 비금속: 금속 원자가 전자를 줌 → 이온화 → 이온 결합
물의 전기 분해 — 공유 결합의 전기적 성질
💧 2H₂O(l) → 2H₂(g) + O₂(g)
물(H₂O)에 전류를 흘려 주면 각 전극에서 기포가 발생합니다.

• (+)극: 산소(O₂) 기체 발생
• (−)극: 수소(H₂) 기체 발생
• 부피비: 산소(O₂) : 수소(H₂) = 1 : 2

전기 에너지를 가하면 수소 원자와 산소 원자 사이의 결합이 끊어집니다. 이는 화학 결합이 형성될 때 전자가 관여하며 결합에 전기적 성질이 있기 때문입니다.
⚠️ 황산 나트륨(Na₂SO₄)을 넣는 이유
순수한 증류수는 전기 전도성이 거의 없어 전기 분해가 잘 일어나지 않습니다. 황산 나트륨을 소량 녹이면 이온이 생겨 전기 전도성이 증가해 전기 분해가 원활하게 일어납니다.
염화 나트륨 전기 분해 — 이온 결합의 전기적 성질
🧂 2NaCl(l) → 2Na(l) + Cl₂(g)
이온 결합 물질인 염화 나트륨(NaCl):

• 고체 상태: 이온이 고정 → 전류 흐르지 않음
• 액체 상태(용융액): 이온이 자유롭게 이동 → 전류 흐름 → 전기 분해 가능
• (−)극: Na(나트륨) 생성 / (+)극: Cl₂(염소 기체) 생성

이를 통해 이온 결합에도 전자가 관여함을 알 수 있습니다.
📌 핵심 결론
화학 결합이 형성될 때는 전기적 성질을 띠는 전자가 관여하며, 공유 결합과 이온 결합 모두 화학 결합에는 전기적 성질이 있습니다.
✏️ 확인 문제 1
물의 전기 분해 실험에서 (+)극과 (−)극에서 발생하는 기체의 종류와 부피비를 각각 쓰시오.
정답:

(+)극: 산소(O₂)
(−)극: 수소(H₂)
부피비: 산소 : 수소 = 1 : 2

2H₂O → 2H₂ + O₂에서 H₂ : O₂ = 2 : 1이므로 (−)극의 수소가 (+)극의 산소보다 2배 더 발생합니다.

02
Electronegativity · 전기음성도
전기음성도와 주기적 변화

▼
전기음성도란?
⚡ 전기음성도 (Electronegativity)
전기음성도: 공유 결합을 이루는 각 원자가 공유 전자쌍을 끌어당기는 정도를 상대적인 수치로 나타낸 것.

폴링(Pauling) 척도: 공유 전자쌍을 끌어당기는 정도가 가장 큰 플루오린(F)의 전기음성도를 4.0으로 정하고 상대적 수치 사용.

18족 원소는 다른 원자와 결합 형성이 어려워 전기음성도를 보통 다루지 않습니다.

03
Bond Polarity · 결합의 극성
결합의 극성과 쌍극자 모멘트
무극성 공유 결합 vs 극성 공유 결합 · δ+, δ− 부분 전하 · 쌍극자 모멘트 화살표
▼
무극성 공유 결합 vs 극성 공유 결합
구분무극성 공유 결합극성 공유 결합
정의같은 종류의 원자 사이 결합
→ 공유 전자쌍 치우침 없음		다른 종류의 원자 사이 결합
→ 공유 전자쌍이 한쪽으로 치우침
부분 전하원자가 부분 전하 안 띰전기음성도 큰 원자: δ−
전기음성도 작은 원자: δ+
예시H₂, O₂, N₂, Cl₂, F₂HCl, HF, H₂O, NH₃
극성의 크기와 전기음성도 차이
📊 전기음성도 차이 → 극성 크기
화학 결합을 형성하는 두 원자의 전기음성도 차이가 커질수록 결합의 극성이 대체로 커집니다.

H−F: 전기음성도 차이 = 4.0 − 2.2 = 1.8 (치우침 큼)
H−Cl: 전기음성도 차이 = 3.2 − 2.2 = 1.0 (치우침 작음)

따라서 H−F 결합이 H−Cl 결합보다 극성이 더 큽니다.
⬅ 결합과 이온 결합의 연속성
전기음성도 차이 0 → 무극성 공유 결합
전기음성도 차이 증가 → 극성 공유 결합
전기음성도 차이 매우 큼 → 전자가 완전 이동 → 이온 결합

무극성 공유 결합과 이온 결합은 화학 결합의 양 끝이며, 대부분의 결합은 그 사이에 위치합니다.
쌍극자 모멘트 (Dipole Moment)
→ 쌍극자 모멘트
쌍극자: 크기가 같고 부호가 반대인 두 전하가 일정한 거리를 두고 분리된 것.

쌍극자 모멘트: 공유 결합에서 극성의 크기를 나타내는 척도.
화살표로 나타낼 때 → 전기음성도가 작은 원자에서 큰 원자 방향으로 표시 (+ → −)

무극성 공유 결합: 쌍극자 모멘트 = 0
극성 공유 결합: 쌍극자 모멘트 ≠ 0
✏️ 확인 문제 5
공유 결합은 결합하는 두 원자의 전기음성도의 차이에 따라 ( ) 공유 결합과 ( ) 공유 결합으로 나눌 수 있다. 괄호에 알맞은 말을 쓰시오.
정답: 무극성, 극성
✏️ 확인 문제 6
다음 세 가지 결합을 무극성 공유 결합과 극성 공유 결합으로 구분하시오.

N−H    N≡N    C−F
정답:

무극성 공유 결합: N≡N (같은 종류의 원자 간 결합)
극성 공유 결합: N−H, C−F (서로 다른 종류의 원자 간 결합)

N−H: N(3.0) ≠ H(2.2) → 극성
C−F: C(2.6) ≠ F(4.0) → 극성
✏️ 확인 문제 7
대체로 결합을 이루는 원자의 전기음성도 차이가 커질수록 결합의 쌍극자 모멘트가 ( 커지고, 작아지고 ) 결합의 극성이 ( 커진다, 작아진다 ).
정답: 커지고, 커진다

전기음성도 차이가 클수록 공유 전자쌍의 치우침이 커지고 부분 전하의 크기가 증가하므로 쌍극자 모멘트와 결합의 극성이 대체로 커집니다.

01
Lewis Dot Structure · 루이스 구조
루이스 전자점식과 루이스 구조
원자가 전자를 점으로 표시 · 공유 전자쌍·비공유 전자쌍 · 단일·이중·삼중 결합
▼
루이스 전자점식이란?
🔵 루이스 전자점식 (Lewis Electron Dot Structure)
루이스 전자점식: 원자가 전자를 원소 기호 주위에 점으로 표시해 공유 결합을 나타내는 방법.

그리는 규칙:
① 원자가 전자를 나타내는 점을 원소 기호의 상하좌우에 하나씩 배치
② 원자가 전자가 5개 이상이면 다섯 번째부터 다시 상하좌우에 하나씩 그려 쌍을 이루게 함
③ 상하좌우에 점을 찍는 순서는 상관 없음
공유 전자쌍과 비공유 전자쌍

비공유 전자쌍결합에 참여하지 않는 전자쌍점 2개(:..)로 표시하거나 생략

⚠️ 옥텟 규칙 (Octet Rule)
원자가 전자를 잃거나 얻거나 공유하면서 가장 바깥 전자 껍질의 전자 배치를 18족 원소처럼(8개) 만들려는 경향성.

분자를 루이스 전자점식으로 나타낼 때 모든 원자가 옥텟 규칙을 만족하도록 전자쌍을 배치해야 합니다. (단, H는 2개)

CH₄(109.5°) > NH₃(107°) > H₂O(104.5°)

비공유 전자쌍이 많을수록 공유 전자쌍을 더 세게 밀어내므로 결합각이 감소합니다.
이중·삼중 결합의 취급
📌 다중 결합은 하나의 결합으로 취급
전자쌍 반발 이론으로 분자 구조를 결정할 때 이중 결합이나 삼중 결합은 하나의 결합으로 취급합니다.

예) CO₂: C와 O 사이 이중 결합 2개 → 공유 전자쌍 2개로 취급 → 선형(180°)

전자레인지 원리

🛢 기름과 물이 섞이지 않는 이유
기름은 무극성 물질, 물은 극성 물질이므로 서로 섞이지 않습니다.

비누(계면 활성제)는 극성 부분(머리)과 무극성 부분(꼬리)을 모두 가지고 있어 기름때와 물 사이를 연결해 세탁이 가능하게 합니다.
✏️ 확인 문제 12
분자 내에 극성 공유 결합을 갖는 분자는 모두 극성 분자이다. (○, ×)
정답: × (틀림)

극성 공유 결합으로 이루어진 분자라도 분자 구조가 대칭이면 결합의 쌍극자 모멘트가 상쇄되어 무극성 분자가 됩니다.
예) CO₂(선형), CH₄(정사면체), BCl₃(평면삼각형): 모두 극성 결합이지만 무극성 분자
✏️ 확인 문제 13
다음 분자가 극성 분자인지 무극성 분자인지 쓰시오.
(1) HCN    (2) CH₃Cl    (3) BCl₃
정답:

(1) HCN: 선형이지만 양쪽에 다른 원자(H와 N) → 쌍극자 모멘트 상쇄 안 됨 → 극성 분자

(2) CH₃Cl: 정사면체형이지만 한쪽에만 Cl → 비대칭 → 극성 분자

(3) BCl₃: 평면 삼각형(대칭) → 쌍극자 모멘트 상쇄 → 무극성 분자
✏️ 확인 문제 14
물에 잘 녹을 것으로 예상되는 물질을 있는 대로 고르시오.

에탄올(C₂H₅OH)    흑연(C)    산소(O₂)    염화 수소(HCl)
정답: 에탄올(C₂H₅OH), 염화 수소(HCl)

물(H₂O)은 극성 용매이므로 극성 물질이나 이온 결합 물질이 잘 용해됩니다.

에탄올(C₂H₅OH): OH기로 인해 극성 → 물에 잘 녹음
HCl: 극성 분자 → 물에 잘 녹음
흑연(C): 무극성 → 잘 안 녹음
O₂: 무극성 분자 → 잘 안 녹음
🔥 심화 문제
다음은 세 가지 분자의 분자식이다.

(가) HCN    (나) BCl₃    (다) H₂S

이에 대한 설명으로 옳은 것은?
① 극성 분자는 세 가지이다.
② (가)~(다) 중에서 구조가 같은 분자가 존재한다.
③ 분자의 쌍극자 모멘트가 0인 분자는 한 가지이다.
④ 극성 공유 결합으로 이루어진 분자는 두 가지이다.
⑤ 모든 원자가 같은 평면에 위치하는 분자는 두 가지이다.
정답: ③

(가) HCN: 선형, 극성 분자, 모든 원자 같은 직선에 위치
(나) BCl₃: 평면 삼각형, 무극성 분자 (대칭), 모든 원자 같은 평면에 위치
(다) H₂S: 굽은 형, 극성 분자

① 극성 분자: HCN, H₂S → 2가지 (틀림)
② 구조가 같은 분자: HCN(선형), BCl₃(평면삼각형), H₂S(굽은형) → 없음 (틀림)
③ 쌍극자 모멘트=0: BCl₃만 → 1가지 (옳음)
④ 극성 공유 결합: 세 가지 모두 다른 원자간 결합 → 3가지 (틀림)
⑤ 같은 평면: HCN(직선=평면), BCl₃(평면) → 2가지인데 H₂S는 굽은 형으로 모든 원자가 같은 평면에 있음 → 세 가지 모두 (틀림)
