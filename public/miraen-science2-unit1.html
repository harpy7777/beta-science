🦒 자연선택과 진화
자연선택과 진화 메커니즘
다윈의 자연선택설 · 변이 → 생존경쟁 → 자연선택 → 진화
핵심 학습 내용
  • 과잉 생산: 생물은 환경이 부양할 수 있는 것보다 많은 자손 생산
  • 변이: 같은 종에서도 개체마다 형질 차이가 존재하며 유전됨
  • 생존경쟁: 제한된 환경에서 살아남기 위한 경쟁 발생
  • 자연선택: 생존에 유리한 변이를 가진 개체가 살아남아 더 많은 자손을 남김
  • 사례 1: 기린 목 길이 — 목이 긴 개체가 먹이 획득에 유리 → 자연선택
  • 사례 2: 갈라파고스 핀치 — 섬별 먹이 환경에 따라 부리 모양 다양화
  • 사례 3: 낫모양 적혈구 빈혈증 — 말라리아 유행 지역에서 일부 보유자 유리
수능·내신 핵심 키워드
자연선택변이 생존경쟁과잉생산 갈라파고스적응진화 낫모양적혈구
세특 서술 예시
"다윈의 자연선택설을 갈라파고스 핀치의 사례에 적용하여 과잉 생산→변이→생존경쟁→자연선택의 4단계 과정을 논리적으로 서술하였으며, 이를 현대 진화론(유전자 관점)과 연결하여 심화 탐구를 수행함."

  • 발생학: 척추동물 초기 배 발생 과정의 유사성 → 공통 조상 존재 암시

  • 생물다양성과 생태계 안정성: 먹이그물의 복잡성 ↑ → 외부 충격 회복력 ↑

  • 외래종 침입: 뉴트리아·황소개구리·붉은귀거북·가시박 등 생태계 교란
  • 환경 오염: 농약·중금속·플라스틱으로 인한 생물 집단 사망
  • 보전 방안: 생물다양성협약(CBD)·람사르협약·생태 통로·서식지 복원·종자 은행
  • 우리나라 사례: 반달가슴곰·황새·수달 복원 프로젝트
세특 서술 예시
"생물다양성 감소의 4가지 원인(서식지 파괴·남획·외래종·환경오염)을 구체적 사례와 함께 분석하였으며, 생물다양성협약 및 우리나라의 멸종위기종 복원 사례(반달가슴곰)를 연결하여 실질적인 보전 방안을 제안하는 탐구 보고서를 완성함."

🔥 산화와 환원
산화·환원의 두 가지 정의
산소 이동 관점 vs 전자 이동 관점 완전 비교
핵심 학습 내용
  • 산소 이동 관점 — 산화: 산소 얻음 / 환원: 산소 잃음 (예: 철 + 산소 → 산화철 = 산화)
  • 전자 이동 관점 — 산화: 전자 잃음 / 환원: 전자 얻음 (OIL RIG 암기법)
  • 산화와 환원은 항상 동시에 일어남 (산화환원 반응은 짝반응)
  • 생활 속 사례: 광합성(CO₂ 환원), 연소(탄소 산화), 철의 제련(Fe₂O₃ 환원), 철의 부식, 표백제
  • 환원제: 자신이 산화되면서 상대를 환원 / 산화제: 자신이 환원되면서 상대를 산화
수능·내신 핵심 키워드
OIL RIG산화제 환원제전자이동 철의 제련광합성
실험 탐구 포인트
탐구 1: 구리선을 질산은 수용액에 넣는 실험 → 구리가 산화(전자 잃음), 은 이온이 환원(전자 얻음) 관찰

탐구 2: 철못을 황산구리 수용액에 넣어 구리 석출 관찰 → 산화환원 동시성 확인

🧲 산화수
산화수와 산화환원 반응식
산화수 결정 규칙 · 반응 전후 산화수 변화 분석
핵심 학습 내용
  • 산화수 규칙 1: 단원소 물질의 산화수 = 0 (O₂, Fe, Cu 등)
  • 산화수 규칙 2: 이온의 산화수 = 이온의 전하 (Na⁺ = +1, Cl⁻ = -1)
  • 산화수 규칙 3: 화합물에서 O는 -2, H는 +1 (과산화물 제외)
  • 산화수 규칙 4: 화합물에서 각 원소의 산화수 합 = 0
  • 산화수 증가 → 산화 / 산화수 감소 → 환원
  • 반응식 완성: 산화된 원소의 산화수 증가량 = 환원된 원소의 산화수 감소량
탐구 & 수행평가 포인트
수행평가 포인트: KMnO₄ + FeSO₄ 반응에서 각 원소의 산화수를 결정하고, 산화되는 원소와 환원되는 원소를 구분하여 반응식 완성하기

🧪 산·염기 정의
산·염기의 정의와 성질
아레니우스 정의 · 지시약 · pH 완전 정복
핵심 학습 내용
  • 아레니우스의 산: 수용액에서 H⁺ 내놓는 물질 (HCl, H₂SO₄, CH₃COOH)
  • 아레니우스의 염기: 수용액에서 OH⁻ 내놓는 물질 (NaOH, KOH, Ca(OH)₂)
  • 지시약 색 변화: BTB — 산(노랑), 중성(초록), 염기(파랑) / 페놀프탈레인 — 산(무색), 염기(붉은색)
  • pH: 수용액의 H⁺ 농도를 나타내는 척도, 0~14, 7 미만=산, 7=중성, 7 초과=염기
  • 생활 속 사례: 레몬(산, pH≈2), 우유(약산, pH≈6), 소다(염기, pH≈8), 표백제(강염기, pH≈13)
실험 탐구 포인트
탐구: 다양한 일상 용액(식초·탄산음료·비눗물·베이킹 소다)의 pH 측정 → BTB 지시약으로 색 변화 관찰 → 산성·염기성 분류 표 완성

⚗️ 중화 반응
중화 반응과 이온 변화
H⁺ + OH⁻ → H₂O · 중화열 · 이온 수 그래프 분석
핵심 학습 내용
  • 중화 반응: 산의 H⁺와 염기의 OH⁻가 만나 물(H₂O) 생성 + 중화열 발생
  • 중화점: H⁺과 OH⁻의 몰 수가 같아지는 지점 → 온도 최고점, pH = 7(강산+강염기)
  • 이온 수 변화: HCl에 NaOH 첨가 시 — H⁺↓, OH⁻↑, Na⁺↑(계속), Cl⁻ 일정
  • 중화 반응 그래프: x축=NaOH 부피, y축=온도 → 포물선 형태, 꼭짓점=중화점
  • 생활 속 중화: 위장약(제산제 — 위산 중화), 생선 비린내(아민+레몬), 치약(구강 산성화 방지)
수능·내신 핵심 키워드
중화점중화열 알짜이온반응식구경꾼이온 이온수변화제산제
실험 탐구 포인트
탐구: 염산(HCl) 수용액에 수산화나트륨(NaOH) 용액을 조금씩 넣으며 온도와 pH를 측정 → 이온 수 변화 그래프 작성 → 중화점 결정

🌡️ 에너지 출입
발열 반응과 흡열 반응
에너지 다이어그램 · 일상 사례 · 수능 필수 개념
핵심 학습 내용
  • 발열 반응: 반응물의 에너지 > 생성물의 에너지 → 주변으로 열 방출 → 온도 상승
  • 발열 반응 사례: 연소(장작), 중화 반응, 산화 칼슘+물, 손난로(철 산화), 체내 세포 호흡
  • 흡열 반응: 반응물의 에너지 < 생성물의 에너지 → 주변의 열 흡수 → 온도 하강
  • 흡열 반응 사례: 광합성, 질산암모늄+물(냉찜질), 탄산수소나트륨 분해, 얼음 녹음
  • 에너지 다이어그램: 반응물·생성물 에너지 크기 비교로 발열/흡열 구분
수능·내신 핵심 키워드
발열반응흡열반응 에너지다이어그램손난로 냉찜질세포호흡광합성

산화 칼슘 탐구 & 생활 속 에너지 반응
발열·흡열 통합 탐구 보고서 완성
탐구 실험 내용
  • 실험 1 (발열): 산화 칼슘(CaO) + 물 → Ca(OH)₂, 열 방출 → 온도계로 온도 상승 측정
  • 실험 2 (흡열): 질산암모늄(NH₄NO₃) + 물 → 흡열 반응, 온도 하강 측정
  • 결과 분석: 두 반응의 온도 변화 비교 표·그래프 작성
  • 응용: 산화 칼슘 발열 원리 → 군용 자가발열 도시락, 즉석 온열 음식
  • 응용: 질산암모늄 흡열 원리 → 냉찜질 팩, 스포츠 의료용 쿨링 팩
