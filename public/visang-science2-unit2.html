생태계를 구성하는 요소
🌿 생태계(Ecosystem)란?
생태계: 생물과 환경이 영향을 주고받으며 살아가는 체계.

조직 수준: 개체 → 개체군 → 군집 → 생태계
• 개체군: 일정한 지역에 같은 종의 개체들이 무리를 이룬 것
• 군집: 여러 개체군이 같은 서식지에 모여 살아가는 것
• 생태계: 군집을 구성하는 개체군이 다른 개체군 및 환경과 영향을 주고받는 체계
생물요소와 비생물요소
구분정의종류 / 예시
생물요소 생태계에 존재하는 모든 생물 생산자: 광합성으로 양분 스스로 제조 (식물, 식물 플랑크톤)
소비자: 다른 생물을 먹어 양분 획득 (고라니, 멧돼지 등)
분해자: 죽은 생물·배설물 분해하여 양분 획득 (버섯, 세균, 곰팡이)
비생물요소 생물을 둘러싸고 있는 환경요인 빛, 온도, 물, 토양, 공기
생물과 환경의 상호 관계
☀️ 빛과 생물
• 강한 빛을 받는 잎: 두꺼움 (울타리조직 발달 → 광합성 활발)
• 약한 빛을 받는 잎: 얇고 넓음 (빛을 넓게 받기 위해)

일조 시간과 번식:
• 꾀꼬리·종달새: 일조 시간이 길어지는 봄에 번식
• 송어·노루: 일조 시간이 짧아지는 가을에 번식
🌡️ 온도와 생물
온도는 생물의 물질대사에 영향을 미침.

여우의 온도 적응 (베르크만 법칙):
• 사막여우 (고온): 몸집 작고 귀 큼 (열 방출 유리)
• 온대여우: 중간
• 북극여우 (저온): 몸집 크고 귀 작음 (열 보존 유리)

식물의 온도 적응: 툰드라의 털송이풀 → 잎·꽃에 털이 나 체온 저하 방지
💧 물·토양·공기와 생물
물과 생물: 선인장 → 잎이 가시로 변해 수분 증발 최소화. 연꽃 → 통기조직 발달
곤충 → 키틴질로 수분 손실 방지. 뱀·도마뱀 → 비늘로 수분 손실 방지

토양과 생물: 지렁이·두더지 → 토양 통기성 향상. 미생물 → 물질 순환 역할

공기와 생물: 고산지대 사람 → 평지보다 적혈구 수 증가 (산소 효율적 운반)

먹이 관계와 생태피라미드
🔗 먹이사슬과 먹이그물
먹이사슬: 생태계에서 생산자부터 최종 소비자까지 먹고 먹히는 관계를 사슬 모양으로 나타낸 것
예) 식물 → 메뚜기 → 개구리 → 뱀 → 매

먹이그물: 여러 개의 먹이사슬이 서로 얽혀 그물처럼 복잡하게 나타나는 것

→ 먹이 관계가 복잡할수록 생태계평형이 잘 유지됨
🔺 생태피라미드(Ecological Pyramid)
각 영양단계의 에너지양, 개체수, 생체량의 상대적인 양을 하위 영양단계에서 상위 영양단계로 쌓아 올리면 위로 갈수록 줄어드는 피라미드 형태를 이루는 것.

이유: 하위 영양단계의 생물이 가진 에너지 일부가 생명활동에 쓰이거나 열에너지로 방출 → 상위 단계로 전달되는 에너지양 감소

영양단계: 생산자 → 1차 소비자 → 2차 소비자 → 3차 소비자
예) 식물 → 곤충·달팽이 → 비둘기·오리 → 매

생태계평형과 회복
⚖️ 생태계평형(Ecosystem Equilibrium)
생태계에서 생물군집의 구성이나 개체수, 물질의 양, 에너지의 흐름이 균형을 이루면서 안정된 상태를 유지하는 것.

회복 과정 (1차 소비자 일시 증가 사례):
1차 소비자 증가 → 생산자 감소 + 2차 소비자 증가 → 1차 소비자 감소 → 생산자 회복 → 평형 회복
⚠️ 환경 변화와 생태계 파괴
자연적 요인: 지진, 화산, 태풍, 홍수 등 자연재해
인위적 요인: 무분별한 개발, 벌목, 환경 오염

→ 생태계평형이 깨지면 인간을 비롯한 모든 생물의 생존이 위협받으며, 회복에 오랜 시간과 노력이 필요

생태계 보전 방안:
• 천연기념물 및 국립 공원으로 지정·관리
• 환경보전을 위한 특별법 제정 및 시행
• 환경영향평가 실시
✏️ 확인 문제 3
다음 두 생태계 (가)와 (나)에서 쥐가 사라졌을 때, 생태계평형이 더 잘 유지될 수 있는 생태계를 고르고 그 이유를 서술하시오.

(가): 옥수수 → 쥐 → 뱀 → 매 (단순한 먹이사슬)
(나): 옥수수·당근·벼 → 쥐·토끼·메뚜기 → 뱀·개구리 → 매·늑대 (복잡한 먹이그물)
정답: (나)

(나)에서 쥐가 사라지더라도 뱀은 토끼, 메뚜기 등 다른 먹이를 먹을 수 있기 때문에 생태계에 미치는 영향이 작아 평형을 유지할 수 있습니다.

반면 (가)에서 쥐가 사라지면 뱀의 먹이가 없어져 뱀도 감소하고, 매도 줄어드는 연쇄적인 문제가 발생합니다.

→ 먹이 관계가 복잡할수록 생태계평형이 잘 유지됩니다.
✏️ 확인 문제 4
어떤 국립 공원에서 늑대(최상위 포식자)가 사냥으로 사라진 후 사슴의 개체수가 급증하였다. 이후 늑대를 다시 도입했더니 생태계가 회복되기 시작했다. 늑대 도입 후 생태계가 평형을 회복하는 과정을 서술하시오.
정답:

늑대가 다시 도입되면 → 사슴의 개체수 감소 (포식 증가) → 사슴이 먹어치우던 풀과 나무가 회복 → 식생이 살아나면서 강둑이 안정되고 다른 생물들의 서식지도 회복 → 비버가 댐을 만들 나무 확보 → 수생 생태계 회복 → 전체 생태계평형 회복

이처럼 최상위 포식자 한 종이 생태계 전반에 연쇄적인 영향을 미치는 것을 영양 단계 연쇄 효과(Trophic Cascade)라고 합니다.

복사 평형과 지구 열수지
🌍 지구의 에너지 균형
복사 평형: 지구가 태양으로부터 흡수한 에너지와 같은 양의 에너지를 우주로 방출하면서 에너지 평형을 이루는 상태.

지구 열수지: 지구의 복사 평형 상태에서 나타나는 지표·대기·우주 간의 열출입 관계

태양 복사 에너지 100 기준:
• 반사·산란 (우주로): 30
• 지표면과 대기에 흡수: 70
• 지구 복사 에너지로 우주에 방출: 70 (평형)
지구 온난화 메커니즘
🌡️ 온실 효과와 지구 온난화
온실 기체: 이산화 탄소(CO₂, 75%), 메테인(CH₄, 18%), 산화 이질소(N₂O, 4%), 기타

온실 효과 원리: 온실 기체가 지표에서 방출되는 지구 복사 에너지를 흡수했다가 지표로 재복사 → 지구 평균 온도 유지

지구 온난화: 대기 중 온실 기체의 양이 증가함에 따라 온실 효과가 강화되어 지구의 평균 온도가 높아지는 현상

사막화(Desertification)
🏜️ 사막화의 원인과 영향
사막화: 사막이 아니던 지역의 토지가 황폐해지면서 사막으로 변해가는 현상.

발생 지역: 주로 강수량이 적고 증발량이 많은 위도 30° 부근의 건조한 곳

원인:
• 자연적 요인: 대기 대순환 변화 → 증발량 증가·강수량 감소 → 장기간 가뭄
• 인위적 요인: 과잉 경작, 과잉 방목, 무분별한 삼림 파괴

영향: 농경지 감소, 식량 부족, 물 부족, 황사 발생 증가

대책: 사막화 방지 협약 (국제 협약), 조림 사업

→ 빙하 면적 감소 → 해수면 상승 → 해안 지역 침수

소단원 01 · Solar Energy

▼
태양 에너지의 생성
☀️ 수소 핵융합 반응(Hydrogen Fusion Reaction)
태양 에너지는 약 1500만 K의 초고온 상태인 태양 중심부에서 생성됩니다.

수소 핵융합 반응: 수소 원자핵이 단계별로 충돌하여 헬륨 원자핵이 만들어지는 반응

에너지가 발생하는 이유:
핵융합으로 만들어진 헬륨 원자핵 1개의 질량 < 반응에 참여한 양성자 2개 + 중성자 2개의 질량 합계
→ 핵융합 반응 과정에서 질량이 줄어듦
→ 줄어든 질량만큼 에너지로 전환 (아인슈타인: 질량과 에너지는 서로 변환될 수 있는 물리량)
⚠️ 핵심 공식 요약
수소 핵융합: 4H → He + 에너지
반응 전 수소 4개의 질량 > 반응 후 헬륨 1개의 질량
→ 감소한 질량 = 방출되는 에너지 (E = mc²)
태양 에너지의 전환과 흐름
🔄 지구에서의 태양 에너지 전환
태양에서 생성된 에너지는 지구에 도달하여 다양한 에너지로 전환됩니다.

전환 사례:
• 대기 흡수 → 열에너지 → 바람 → 운동 에너지 (대기·해수 이동)
• 태양 전지 → 빛에너지 → 전기 에너지 (태양광 발전)
• 식물 광합성 → 빛에너지 → 화학 에너지 (포도당 저장)
• 화석 연료 연소 → 화학 에너지 → 운동/열에너지
• 바다 증발 → 구름 → 비 → 댐 → 위치 에너지 → 운동 에너지 → 전기 에너지 (수력 발전)
🌊 태양 에너지의 연속적 흐름
태양 에너지의 전환은 연속적인 과정으로 이루어지며 에너지 흐름을 일으킵니다.

태양 → 바다에 흡수 (열에너지) → 물 증발 → 구름 → 비·눈 → 강·댐 (위치 에너지) → 수력 발전 (전기 에너지) → 다시 바다 → 순환

대기 CO₂ → 광합성 (화학 에너지) → 유해 매립 → 화석 연료 → 연소 → CO₂ → 순환

→ 지구시스템의 각 권역을 이동하면서 다양한 에너지 흐름을 일으킵니다.
✏️ 확인 문제 7
태양의 수소 핵융합 반응에 대한 설명으로 옳은 것을 모두 고르시오.
ㄱ. 수소 핵융합 반응으로 생성되는 원자핵은 헬륨이다.
ㄴ. 수소 핵융합 반응 과정에서 질량이 감소한다.
ㄷ. 태양에서 발생한 에너지는 지구에 도달하여 다양한 에너지로 전환된다.
ㄹ. 핵융합 반응 전후의 총 질량은 보존된다.
정답: ㄱ, ㄴ, ㄷ

ㄱ ✅: 수소 원자핵 → 헬륨 원자핵 생성
ㄴ ✅: 반응 과정에서 질량이 감소하고, 그만큼 에너지가 방출됩니다.
ㄷ ✅: 지구에서 열에너지, 운동 에너지, 화학 에너지, 전기 에너지 등으로 전환됩니다.
ㄹ ✗: 핵융합에서는 감소한 질량이 에너지로 전환되므로 총 질량은 보존되지 않습니다.

전자기 유도(Electromagnetic Induction)
🧲 전자기 유도란?
전자기 유도: 코일 근처에서 자석을 움직였을 때 코일을 통과하는 자기장이 시간에 따라 변하여 코일에 전류가 유도되어 흐르는 현상

이때 코일에 흐르는 전류를 유도 전류라고 합니다.

발견자: 패러데이(Faraday, 1791~1867) — 자기장으로부터 전류를 만드는 것이 가능하다는 아이디어로 발견
유도 전류의 방향과 세기
조건검류계 바늘 방향
N극을 코일에 가까이 할 때오른쪽 (예시)
N극을 코일에서 멀리 할 때왼쪽 (반대)
S극을 코일에 가까이 할 때왼쪽 (N극 반대)
S극을 코일에서 멀리 할 때오른쪽 (반대)
자석이 코일 속에서 정지해 있을 때0 (움직임 없음)
⚡ 유도 전류의 세기를 크게 하는 방법
① 자석의 세기: 자석을 겹쳐서 움직이면 세기 증가
② 자석의 속도: 자석을 빠르게 움직이면 세기 증가
③ 코일의 감은 수: 코일을 많이 감을수록 세기 증가

→ 세 가지 모두 코일을 통과하는 자기장의 변화를 크게 만들기 때문
발전기에서의 전기 에너지 생성
🏭 발전기의 원리
발전기: 자석과 코일로 구성. 바깥쪽에 코일이 든 철심(고정), 안쪽에 자석(회전)

터빈이 회전 → 자석이 코일 속에서 회전 → 자기장 변화 → 전자기 유도 → 유도 전류 발생 → 전기 에너지 생성

즉 터빈의 운동 에너지 → 전기 에너지로 전환
발전 방식에너지원에너지 전환 과정장점단점
화력 발전 석탄, 석유, 천연가스 (화석 연료) 화학에너지 → 열에너지 → 증기 → 터빈 운동에너지 → 전기에너지 건설비용 적음, 빠른 건설, 에너지 공급 안정성 높음 CO₂ 다량 배출 → 지구 온난화 심화
핵발전 우라늄 (핵연료) 핵에너지 → 열에너지 → 증기 → 터빈 운동에너지 → 전기에너지 적은 연료로 대량 전력 생산, CO₂ 배출 거의 없음 방사성 폐기물 처리 어려움, 방사능 누출 위험
수력 발전 물의 위치에너지 위치에너지 → 운동에너지 → 터빈 → 전기에너지 CO₂ 미배출, 자원 무한 적합한 지역 한정, 생태계 영향

에너지 전환과 보존
⚖️ 에너지 보존 법칙
에너지 보존 법칙: 에너지는 여러 가지 형태로 전환될 수 있지만 새롭게 생겨나거나 소멸되지 않고, 에너지가 전환되는 과정에서 전환 전과 후의 총량은 항상 같다.

예) 스마트폰 충전 전기에너지 = 소리에너지 + 빛에너지 + 운동에너지 + 열에너지 (합계)
에너지 효율(Energy Efficiency)
📊 에너지 효율의 정의
에너지 효율: 공급된 에너지 중에서 유용하게 사용된 에너지의 비율(%)

에너지 효율(%) = (유용하게 사용된 에너지 / 공급된 에너지) × 100

왜 에너지 효율은 항상 100% 미만인가?
에너지가 전환될 때 항상 열에너지가 발생하여 버려지기 때문. 이 열에너지는 다시 사용하기 어렵습니다.

예) 선풍기를 오래 사용하면 모터가 뜨거워지는 것 = 전기에너지 일부가 열에너지로 버려지는 것
구분화석 연료 자동차전기 자동차
에너지원화석 연료 (화학에너지)배터리 (전기에너지)
에너지 효율약 15~25%약 87~91%
이유연소 과정에서 많은 열에너지가 버려짐전동기 사용으로 버려지는 열에너지 적음, 제동 시 에너지 일부 재사용
신재생 에너지(New & Renewable Energy)
🌿 신재생 에너지란?
신재생 에너지: 기존의 화석 연료를 변환하여 이용하거나 햇빛, 바다, 바람 등의 재생 가능한 에너지를 변환하여 이용하는 에너지

특징: 자원 고갈 염려 적음, 친환경적, 지구 환경 문제 해결에 기여
한계: 설치 비용 많음, 발전 효율 낮음

✏️ 확인 문제 9

② 에너지 효율은 항상 100%보다 작다.
③ 에너지가 전환되기 전과 전환된 후의 총량은 같다.
④ 에너지 효율 = 유용하게 사용된 에너지 / 공급된 에너지 × 100(%)

정답:

화석 연료의 연소로 발생한 열에너지로 물을 끓여 고온·고압의 증기를 만듭니다. 이 증기로 터빈이 회전하면 발전기의 자석도 함께 회전하여 전자기 유도 현상이 일어나고, 코일에 유도 전류가 발생하여 전기 에너지가 만들어집니다.
✏️ 확인 문제 11
신재생 에너지에 대한 설명으로 옳은 것을 모두 고르시오.
ㄱ. 화석 연료에 비해 자원 고갈의 염려가 없다.
ㄴ. 화력 발전에 비해 발전 효율이 높아 대규모 전력 공급이 용이하다.
ㄷ. 재생 가능한 에너지를 사용하므로 지속적인 발전이 가능하다.
ㄹ. 신재생 에너지 발전 과정에서도 환경 문제가 발생할 수 있다.
정답: ㄱ, ㄷ, ㄹ

ㄱ ✅: 태양, 바람, 물 등 재생 가능한 자원을 이용하므로 고갈 염려 적음
ㄴ ✗: 신재생 에너지는 발전 효율이 낮고, 날씨 등 환경 조건에 크게 영향 받음
ㄷ ✅: 재생 가능한 에너지이므로 지속가능한 발전 가능
ㄹ ✅: 예를 들어 풍력 발전 시 소음·조류 충돌, 태양광 패널 제조 시 폐기물 등 환경 문제 발생 가능
