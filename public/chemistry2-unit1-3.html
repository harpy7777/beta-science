소단원 01 · Enthalpy & Thermochemical Equations
엔탈피와 열화학 반응식
반응열·엔탈피·반응 엔탈피의 개념을 이해하고, 열화학 반응식 작성 규칙과 열량계를 이용한 반응 엔탈피 측정법을 학습합니다.
01
Enthalpy · 엔탈피와 반응열
반응열·엔탈피·반응 엔탈피
발열 반응(ΔH<0) · 흡열 반응(ΔH>0) · ΔH = 생성물 엔탈피 합 − 반응물 엔탈피 합
▼
반응열과 발열·흡열 반응
🌡️ 반응열이란?
반응열: 화학 반응에서 반응물과 생성물 사이에 출입하는 열에너지

• 발열 반응 (Exothermic): 열에너지를 방출하는 반응 → 주위 온도 상승
  예) 연소 반응, 손난로(철의 산화), 산화칼슘 + 물, 중화 반응

• 흡열 반응 (Endothermic): 열에너지를 흡수하는 반응 → 주위 온도 하강
  예) 광합성, 질산 암모늄 용해, 탄산수소 나트륨 분해(베이킹파우더)
엔탈피(H)와 반응 엔탈피(ΔH)
⚡ 엔탈피와 반응 엔탈피의 정의
엔탈피(H): 어떤 물질이 특정 온도와 압력에서 가지는 고유한 에너지. 직접 측정 불가.

반응 엔탈피(ΔH): 일정한 압력에서 화학 반응이 일어날 때의 엔탈피 변화
= 생성물의 엔탈피 합 − 반응물의 엔탈피 합
ΔH = H생성물 − H반응물
발열 반응: H반응물 > H생성물 → ΔH < 0    흡열 반응: H반응물 < H생성물 → ΔH > 0
발열·흡열 반응과 ΔH 부호
구분엔탈피 변화ΔH 부호열에너지 방향주위 온도
발열 반응 반응물 > 생성물 (엔탈피 감소) ΔH < 0 (음수) 계 → 주위 방출 ↑ 상승
흡열 반응 반응물 < 생성물 (엔탈피 증가) ΔH > 0 (양수) 주위 → 계 흡수 ↓ 하강
✏️ 확인 문제 1
다음 반응을 발열 반응과 흡열 반응으로 구분하고 ΔH의 부호를 써 보자.
(가) CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(l) (메테인 연소)
(나) 2NaHCO₃(s) → Na₂CO₃(s) + H₂O(g) + CO₂(g) (탄산수소나트륨 분해)
정답:

(가) 발열 반응 — ΔH < 0 (연소 반응은 열에너지 방출)
(나) 흡열 반응 — ΔH > 0 (가열해야 반응이 일어나므로 열에너지 흡수, 빵을 부풀리는 원리)
02
Thermochemical Equations · 작성 규칙
열화학 반응식 작성법
반응 엔탈피를 함께 표기 · 물질 상태 표기 필수 · 계수 비례 · 역반응 시 부호 반전
▼
열화학 반응식 작성 규칙
📝 열화학 반응식이란?
화학 반응식에 반응 엔탈피(ΔH)를 함께 나타낸 것. 열화학 반응식을 통해 반응 시 출입하는 열에너지의 양을 알 수 있습니다.
📌 열화학 반응식 작성 시 주의 사항 3가지
① 물질의 상태 표기 필수: 같은 물질이라도 상태에 따라 엔탈피가 다릅니다.
  → (s) 고체, (l) 액체, (g) 기체, (aq) 수용액

② 온도·압력 조건 표기: ΔH는 온도·압력에 따라 달라지므로 명시. (기본값: 25℃, 1기압)

③ 계수 변화 시 ΔH도 비례해서 변함:
  N₂(g) + O₂(g) → 2NO(g)   ΔH = 180.6 kJ
  ½N₂(g) + ½O₂(g) → NO(g)   ΔH = 90.3 kJ (계수 ½배 → ΔH도 ½배)

④ 역반응 시 ΔH 부호 반전:
  H₂(g) + ½O₂(g) → H₂O(l)   ΔH = −285.8 kJ
  H₂O(l) → H₂(g) + ½O₂(g)   ΔH = +285.8 kJ
물질 상태에 따른 엔탈피 차이
💧 상태별 엔탈피: 고체 < 액체 < 기체
같은 종류의 물질이라도 고체 → 액체 → 기체로 될수록 엔탈피가 증가합니다.

따라서 고체→액체, 액체→기체는 모두 흡열 반응(ΔH > 0)입니다.

물의 열화학 반응식 비교:
H₂(g) + ½O₂(g) → H₂O(s)   ΔH = −291.5 kJ
H₂(g) + ½O₂(g) → H₂O(l)   ΔH = −285.8 kJ
H₂(g) + ½O₂(g) → H₂O(g)   ΔH = −241.8 kJ

→ 생성물 상태: 고체 < 액체 < 기체 순으로 엔탈피 높음 → |ΔH|: 고체 > 액체 > 기체
✏️ 확인 문제 2
고체 탄산 칼슘(CaCO₃) 1몰이 고체 산화 칼슘(CaO)과 이산화 탄소(CO₂) 기체로 분해될 때 179.2 kJ의 열에너지를 흡수한다. 이 반응을 열화학 반응식으로 나타내 보자.
정답:

CaCO₃(s) → CaO(s) + CO₂(g)   ΔH = +179.2 kJ

흡열 반응이므로 ΔH = 양수(+). 물질의 상태(s, g)를 반드시 표기해야 합니다.
03
Calorimetry · 열량계
반응 엔탈피 측정 — 열량계
Q = c×m×Δt · 간이 열량계 · 통열량계 · 발열 반응 Q>0, ΔH<0
▼
열량 계산 공식
Q = c × m × Δt
Q: 열량(J) · c: 용액의 비열(J/(g·℃)) · m: 용액의 질량(g) · Δt: 온도 변화(℃)
⚠️ 반응열(Q)과 반응 엔탈피(ΔH)의 부호 관계
발열 반응: 온도 상승 → Δt > 0 → Q > 0 → 열화학 반응식에서 ΔH < 0
흡열 반응: 온도 하강 → Δt < 0 → Q < 0 → 열화학 반응식에서 ΔH > 0

∴ ΔH의 부호 = Q의 부호와 반대
열량계의 종류
종류구조적합한 반응특징
간이 열량계 스타이로폼 컵 용기에 반응물 투입 중화 반응, 고체 용해 반응 구조 간단, 열 손실 커서 정밀도 낮음
통열량계 단열 용기 내 강철 용기에 시료 태움 + 물 온도 측정 연소 반응 (고체·액체 연료) 열 손실 최소화, 정밀한 측정 가능
✏️ 확인 문제 3
간이 열량계에 25℃의 1 M 염산(HCl(aq)) 50 mL와 25℃의 1 M 수산화 나트륨(NaOH) 수용액 50 mL를 넣어 반응시켰더니 수용액의 온도는 31.5℃가 되었다. HCl 1몰과 NaOH 1몰이 반응할 때의 반응 엔탈피(ΔH)를 구해 보자. (단, 수용액의 비열은 4.2 J/(g·℃)이고, 밀도는 1 g/mL이다.)
정답: ΔH = −54.6 kJ

1단계: HCl 0.05 mol + NaOH 0.05 mol이 반응 시 발생한 열량(Q) 계산
혼합 후 용액 질량 = 50 + 50 = 100 g
Q = 4.2 × 100 × (31.5 − 25) = 4.2 × 100 × 6.5 = 2730 J = 2.73 kJ

2단계: 발열 반응이므로 ΔH < 0
0.05 mol : −2.73 kJ = 1 mol : x
x = −2.73 ÷ 0.05 = −54.6 kJ
✏️ 확인 문제 4
H₂O(g) 18 g이 분해되어 H₂(g)와 O₂(g)가 생성될 때 241.8 kJ의 열을 흡수한다. 이 반응을 열화학 반응식으로 나타내 보자. (단, H, O의 원자량은 각각 1, 16이다.)
정답:

H₂O(g) 18 g = 1 mol (분자량: 18)
흡열 반응이므로 ΔH > 0

H₂O(g) → H₂(g) + ½O₂(g)   ΔH = +241.8 kJ

소단원 02 · Hess's Law
헤스 법칙
반응 경로에 무관하게 반응 엔탈피 총합이 일정한 헤스 법칙, 표준 생성 엔탈피, 결합 에너지를 이용한 반응 엔탈피 계산을 학습합니다.
01
Hess's Law · 경로 무관 원칙
헤스 법칙 (Hess's Law)
반응물·생성물 같으면 반응 경로 무관 · 반응 엔탈피 총합 일정 · 측정 불가 반응에 응용
▼
헤스 법칙이란?
🛝 헤스 법칙 (1840년, 헤스 제안)
헤스 법칙: 화학 반응이 일어날 때 반응물의 종류와 상태, 생성물의 종류와 상태가 같으면 반응 경로에 관계없이 반응 엔탈피의 총합은 일정하다.

엔탈피는 물질의 상태 함수이므로 경로가 아닌 처음 상태와 나중 상태에만 의존합니다.
ΔH전체 = ΔH₁ + ΔH₂ + ΔH₃ + ···
반응 경로를 여러 단계로 나눠도 각 단계 ΔH의 합은 전체 ΔH와 같다
헤스 법칙 적용 — NaOH와 HCl 반응
📊 두 가지 경로로 같은 결과
경로 1 (2단계):
① NaOH(s) → NaOH(aq)   ΔH₁ = −44.5 kJ
② NaOH(aq) + HCl(aq) → NaCl(aq) + H₂O(l)   ΔH₂ = −55.8 kJ
합: ΔH₁ + ΔH₂ = −100.3 kJ

경로 2 (1단계):
NaOH(s) + HCl(aq) → NaCl(aq) + H₂O(l)   ΔH₃ = −100.3 kJ

→ ΔH₁ + ΔH₂ = ΔH₃ = −100.3 kJ ✅ 헤스 법칙 성립
헤스 법칙으로 측정 어려운 반응 엔탈피 계산
🏭 CO 생성 반응 엔탈피 계산 (직접 측정 불가)
C(s, 흑연) + ½O₂(g) → CO(g) 의 ΔH를 직접 측정하기 어려운 이유: 탄소 연소 시 대부분 CO₂가 생성됨

헤스 법칙 적용:
경로 1: C(s) + O₂(g) → CO₂(g)   ΔH₁ = −393.5 kJ
경로 2: ① C(s) + ½O₂(g) → CO(g)   ΔH₂ = ?
          ② CO(g) + ½O₂(g) → CO₂(g)   ΔH₃ = −283.0 kJ

헤스 법칙: ΔH₁ = ΔH₂ + ΔH₃
ΔH₂ = ΔH₁ − ΔH₃ = −393.5 − (−283.0) = −110.5 kJ
✏️ 확인 문제 5
다음 열화학 반응식을 이용하여 H₂O(l) → H₂O(g) 반응의 반응 엔탈피(ΔH)를 구해 보자.
• 2H₂(g) + O₂(g) → 2H₂O(l)   ΔH = −571.6 kJ
• 2H₂(g) + O₂(g) → 2H₂O(g)   ΔH = −483.6 kJ
정답: ΔH = +44.0 kJ

헤스 법칙 적용: 목표 반응 = H₂O(l) → H₂O(g)

2H₂O(g) → 2H₂(g) + O₂(g)   ΔH = +483.6 kJ (역반응)
2H₂(g) + O₂(g) → 2H₂O(l)   ΔH = −571.6 kJ
더하면: 2H₂O(g) → 2H₂O(l)   ΔH = −88.0 kJ
½배하면: H₂O(g) → H₂O(l)   ΔH = −44.0 kJ

역반응: H₂O(l) → H₂O(g)   ΔH = +44.0 kJ
✏️ 확인 문제 6
다음 열화학 반응식을 이용하여 N₂(g) + 2H₂(g) → N₂H₄(g) 반응의 반응 엔탈피(ΔH)를 구해 보자.
① N₂(g) + 3H₂(g) → 2NH₃(g)   ΔH₁ = −91.8 kJ
② N₂H₄(g) + H₂(g) → 2NH₃(g)   ΔH₂ = −187.6 kJ
정답: ΔH = +95.8 kJ

식 ①을 그대로 사용:
N₂(g) + 3H₂(g) → 2NH₃(g)   ΔH₁ = −91.8 kJ

식 ②의 역반응 사용:
2NH₃(g) → N₂H₄(g) + H₂(g)   −ΔH₂ = +187.6 kJ

두 식을 더하면:
N₂(g) + 3H₂(g) + 2NH₃(g) → 2NH₃(g) + N₂H₄(g) + H₂(g)
약분: N₂(g) + 2H₂(g) → N₂H₄(g)
ΔH = −91.8 + 187.6 = +95.8 kJ
02
Standard Enthalpy of Formation · ΔHf°
표준 생성 엔탈피와 반응 엔탈피
ΔHf°: 성분 원소 → 물질 1 mol 생성 · 원소 단체의 ΔHf° = 0 · ΔH = Σ생성물 ΔHf° − Σ반응물 ΔHf°
▼
표준 생성 엔탈피(ΔHf°)
🌡️ 표준 생성 엔탈피란?
생성 엔탈피: 어떤 물질 1 mol이 성분 원소의 가장 안정한 원소 물질로부터 생성될 때의 엔탈피 변화. 단위: kJ/mol

표준 생성 엔탈피(ΔHf°): 표준 상태(25℃, 1기압)에서의 생성 엔탈피

핵심: 탄소(흑연), H₂(g), O₂(g) 등 가장 안정한 원소 단체의 ΔHf° = 0
주요 물질의 표준 생성 엔탈피
물질ΔHf° (kJ/mol)물질ΔHf° (kJ/mol)
C(s, 흑연)0CO(g)−110.5
O₂(g)0CO₂(g)−393.5
H₂(g)0CH₄(g)−74.6
H₂O(g)−241.8NH₃(g)−45.9
H₂O(l)−285.8C(s, 다이아몬드)+1.9
표준 생성 엔탈피로 반응 엔탈피 계산
ΔH = Σ(생성물의 ΔHf°) − Σ(반응물의 ΔHf°)
각 물질의 ΔHf°에 반응식의 계수를 곱하여 더한다
📊 메테인 연소 반응 엔탈피 계산 예제
CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(l)   ΔH = ?

ΔH = [ΔHf°(CO₂) + 2×ΔHf°(H₂O(l))] − [ΔHf°(CH₄) + 2×ΔHf°(O₂)]
= [−393.5 + 2×(−285.8)] − [(−74.6) + 2×0]
= (−393.5 − 571.6) − (−74.6)
= −965.1 + 74.6 = −890.5 kJ
✏️ 확인 문제 7
다음 표준 생성 엔탈피를 이용하여 포도당(C₆H₁₂O₆(s)) 1 mol로부터 에탄올(C₂H₅OH(l))을 얻는 발효 반응의 반응 엔탈피(ΔH)를 구해 보자.
C₆H₁₂O₆(s) → 2C₂H₅OH(l) + 2CO₂(g)
ΔHf°: 포도당 −1260 kJ/mol, 에탄올 −277.7 kJ/mol, CO₂ −393.5 kJ/mol
정답: ΔH = −72.4 kJ

ΔH = [2×ΔHf°(C₂H₅OH) + 2×ΔHf°(CO₂)] − [ΔHf°(C₆H₁₂O₆)]
= [2×(−277.7) + 2×(−393.5)] − (−1260)
= (−555.4 − 787.0) + 1260
= −1342.4 + 1260 = −82.4 kJ

※ 교과서 수치에 따라 −72.4 kJ ~ −82.4 kJ 범위일 수 있음

Bond Energy · 결합 에너지
결합 에너지와 반응 엔탈피
결합 끊기 = 흡열 · 결합 형성 = 발열 · ΔH = 반응물 결합 에너지 합 − 생성물 결합 에너지 합
▼
결합 에너지란?
🔗 결합 에너지 (Bond Energy)
기체 상태의 분자를 구성하는 두 원자 사이의 공유 결합 1 mol을 끊는 데 필요한 에너지. 단위: kJ/mol

• 결합을 끊는 것 = 에너지 흡수 (흡열)   → ΔH > 0
• 결합이 형성되는 것 = 에너지 방출 (발열) → ΔH < 0

예) H₂(g) → H(g) + H(g)   ΔH = 436 kJ/mol (H-H 결합 에너지 = 436 kJ/mol)

결합 에너지로 반응 엔탈피 계산
ΔH = Σ(반응물의 결합 에너지) − Σ(생성물의 결합 에너지)
반응물의 결합 끊기(흡열) + 생성물의 결합 형성(발열) = 전체 반응 엔탈피
📊 H₂ + Cl₂ → 2HCl 반응 엔탈피 계산
과정 1 (결합 끊기): H-H 1 mol + Cl-Cl 1 mol 끊기
ΔH₁ = 436 + 243 = +679 kJ (흡열)

과정 2 (결합 형성): H-Cl 2 mol 형성
ΔH₂ = −(2 × 432) = −864 kJ (발열)

전체: ΔH = 679 + (−864) = −185 kJ (발열 반응)
✏️ 확인 문제 8
다음 결합 에너지를 이용하여 N₂(g) + 3H₂(g) → 2NH₃(g) 반응의 반응 엔탈피를 구해 보자.
N≡N = 945 kJ/mol, H-H = 436 kJ/mol, N-H = 391 kJ/mol
정답: ΔH = −87 kJ

반응물 결합 에너지 (끊기):
N≡N 1 mol = 945 kJ
H-H 3 mol = 3 × 436 = 1308 kJ
합계 = 2253 kJ

생성물 결합 에너지 (형성):
N-H 결합: NH₃ 1분자당 3개, 2 mol NH₃ → N-H 6 mol
6 × 391 = 2346 kJ

ΔH = 2253 − 2346 = −93 kJ
(※ 교과서 수치에 따라 약간의 차이 있음)

소단원 03 · Spontaneity of Chemical Change
화학 변화의 자발성
자발적 변화의 특징과 엔트로피(S)의 개념, 계와 주위의 엔트로피 변화, 그리고 엔탈피·엔트로피를 통합한 반응 자발성 판단법을 학습합니다.
01
Entropy · 엔트로피
자발적 변화와 엔트로피
자발적 변화 = 무질서도 증가 · 엔트로피(S) = 무질서한 정도 · S: 고체<액체≪기체
▼
자발적 변화란?
🌊 자발적 변화 (Spontaneous Change)
자발적 변화: 외부의 영향(에너지 공급) 없이 스스로 일어나는 변화

자발적 변화의 특징 — 무질서도 증가:
• 물에 잉크를 떨어뜨리면 → 잉크가 물 전체로 퍼짐 (자발) / 다시 뭉치지 않음 (비자발)
• 향수 뿌리면 → 분자들이 공기 중으로 퍼짐 (자발)
• 드라이아이스가 기체로 승화 (자발) / 기체가 다시 고체가 되지 않음 (비자발)

공통점: 질서 있는 상태 → 무질서한 상태로 변화
엔트로피(S)와 엔트로피 변화(ΔS)
🌀 엔트로피 (Entropy, S)
엔트로피(S): 분자의 무질서한 정도를 나타내는 물리량

대부분의 자발적인 과정은 엔트로피가 낮은(질서 있는) 상태에서 높은(무질서한) 상태로 진행됩니다.
ΔS = S최종 − S초기
ΔS > 0: 무질서도 증가 (엔트로피 증가)    ΔS < 0: 무질서도 감소 (엔트로피 감소)
물질의 상태와 엔트로피
상태분자 배열엔트로피 크기상태 변화 시 ΔS
고체 규칙적, 고정 위치 가장 작음 고체→액체: ΔS > 0 (엔트로피 증가)
액체 어느 정도 이동 가능, 불규칙 중간 액체→기체: ΔS > 0 (엔트로피 증가)
기체 자유롭게 운동, 매우 무질서 가장 큼 기체→액체: ΔS < 0 (엔트로피 감소)
화학 반응에서 엔트로피 변화 예측
📊 기체 분자 수 변화로 ΔS 예측 (반응물·생성물 모두 기체인 경우)
기체 분자 수 증가 → ΔS > 0:
N₂O₄(g) → 2NO₂(g)   반응 전 1 mol → 반응 후 2 mol

기체 분자 수 감소 → ΔS < 0:
N₂(g) + 3H₂(g) → 2NH₃(g)   반응 전 4 mol → 반응 후 2 mol

고체/액체가 기체로 변하는 반응:
CaCO₃(s) → CaO(s) + CO₂(g)   기체 생성 → ΔS > 0
✏️ 확인 문제 9
다음 변화에서 엔트로피 변화(ΔS)의 부호를 예측하고 이유를 설명해 보자.
(1) H₂O(l) → H₂O(g)    (2) 2SO₂(g) + O₂(g) → 2SO₃(g)
정답:

(1) ΔS > 0 — 액체가 기체로 상태 변화. 기체는 액체보다 분자가 훨씬 무질서하게 배열되므로 엔트로피 증가

(2) ΔS < 0 — 반응 전 기체 분자 수: SO₂ 2 mol + O₂ 1 mol = 3 mol / 반응 후: SO₃ 2 mol. 기체 분자 수 감소(3→2) → 무질서도 감소 → 엔트로피 감소
02
System & Surroundings · 자발성 판단
계와 주위의 엔트로피 변화와 자발성
계·주위·우주 개념 · ΔS전체 = ΔS계 + ΔS주위 · ΔS전체>0: 자발 · 열역학 제2법칙
▼
계·주위·우주의 정의
🌍 계(System)와 주위(Surroundings)
계 (System): 우리가 관심을 갖는 대상 = 반응물과 생성물
주위 (Surroundings): 계를 제외한 모든 것 = 반응 용기, 주변 공간 등
우주 (Universe): 계 + 주위

계의 종류:
• 열린계: 주위와 물질·에너지 모두 교환 가능
• 닫힌계: 주위와 에너지는 교환 가능, 물질은 불가
• 고립계: 주위와 물질·에너지 모두 교환 불가
전체 엔트로피 변화와 자발성
ΔS전체 = ΔS계 + ΔS주위
열역학 제2법칙: ΔS전체 > 0 → 자발적 반응 / ΔS전체 < 0 → 비자발적 반응
계의 엔탈피 변화와 주위의 엔트로피 변화
반응 종류계의 ΔH계열 이동 방향주위 온도ΔS주위
발열 반응 ΔH계 < 0 계 → 주위 (열 방출) ↑ 상승 ΔS주위 > 0 (증가)
흡열 반응 ΔH계 > 0 주위 → 계 (열 흡수) ↓ 하강 ΔS주위 < 0 (감소)

✏️ 확인 문제 10
다음 반응은 발열 반응(ΔH = −91.8 kJ)이고 고온에서 자발적으로 일어난다. 고온에서 자발적으로 일어나는 이유를 계와 주위의 엔트로피 변화로 설명해 보자.
N₂(g) + 3H₂(g) → 2NH₃(g)   ΔH = −91.8 kJ
정답:

이 반응은 기체 분자 수가 4 mol → 2 mol로 감소하므로 ΔS계 < 0입니다. 발열 반응이므로 ΔS주위 > 0입니다.

고온에서 자발적으로 일어나는 이유: 고온에서는 발열 반응으로 방출되는 열에너지가 주위에 미치는 영향이 더 크기 때문에 ΔS주위의 크기가 ΔS계의 크기보다 더 커집니다. 따라서 ΔS전체 = ΔS계 + ΔS주위 > 0이 되어 반응이 자발적으로 일어납니다.
✏️ 확인 문제 11
다음 반응이 실온에서 자발적으로 일어나는 까닭을 계와 주위의 엔트로피 변화로 설명해 보자.
CO₂(s) → CO₂(g) (드라이아이스의 승화)
정답:

CO₂(s) → CO₂(g)는 고체가 기체로 승화하는 상태 변화로 ΔS계 > 0(엔트로피 크게 증가)입니다. 승화는 흡열 과정이므로 ΔS주위 < 0입니다.

그러나 고체→기체의 상태 변화는 엔트로피가 매우 크게 증가하므로 |ΔS계| > |ΔS주위|가 됩니다. 따라서 ΔS전체 = ΔS계 + ΔS주위 > 0이 되어 실온에서도 자발적으로 일어납니다.
✏️ 확인 문제 12
다음 (가), (나) 반응의 계의 엔트로피 변화(ΔS계)를 예측하고, 자발성 여부를 엔탈피와 엔트로피 변화로 설명해 보자.
(가) CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(l)   ΔH = −890.8 kJ (자발)
(나) 2H₂O(l) → 2H₂(g) + O₂(g)   ΔH = +571.6 kJ (비자발)
정답:

(가) CH₄ 연소:
ΔS계: 반응 전 기체 3 mol(CH₄ 1 + O₂ 2) → 반응 후 CO₂(g) 1 mol + H₂O(l) 2 mol. 기체 분자 수 감소 → ΔS계 < 0
ΔS주위: 강한 발열 반응 → ΔS주위 > 0 (크게 증가)
∴ |ΔS주위| > |ΔS계| → ΔS전체 > 0 → 자발적

(나) 물 전기 분해:
ΔS계: 액체 → 기체 분자 수 증가(기체 3 mol 생성) → ΔS계 > 0
ΔS주위: 강한 흡열 반응 → ΔS주위 < 0 (크게 감소)
∴ |ΔS주위| > |ΔS계| → ΔS전체 < 0 → 비자발적 (전기 에너지 공급 필요)
