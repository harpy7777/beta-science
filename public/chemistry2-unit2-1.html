소단원 01 · Brønsted–Lowry Acid & Base
브뢴스테드·라우리 산 염기
아레니우스 산 염기의 한계와 브뢴스테드·라우리의 새로운 정의, 짝산·짝염기 관계, 양쪽성 물질, 이온화 평형을 학습합니다.
01
Acid–Base Definitions · 산 염기 정의의 발전
아레니우스 정의의 한계와 브뢴스테드·라우리 정의
H⁺ 주는 물질 = 산 · H⁺ 받는 물질 = 염기 · 수용액 이외에도 적용 가능
▼
아레니우스 산 염기 정의와 한계
🏺 아레니우스 정의 (1887년)
산: 물에 녹아 수소 이온(H⁺)을 내놓는 물질 (예: HCl, HNO₃, H₂SO₄)
염기: 물에 녹아 수산화 이온(OH⁻)을 내놓는 물질 (예: NaOH, KOH)
⚠️ 아레니우스 정의의 한계 2가지
한계 1 — 수용액에서만 적용: 비수용액 반응에는 적용 불가
  예) HCl(g) + NH₃(g) → NH₄Cl(s) 반응은 아레니우스로 설명 불가

한계 2 — NH₃의 염기성 설명 불가: NH₃는 OH⁻를 갖지 않지만 수용액에서 염기성을 나타냄
  → H₂O과 반응하여 OH⁻를 생성하기 때문

한계 3 — H⁺ 단독 존재 불가: 실제로 H⁺은 H₂O와 결합한 하이드로늄 이온(H₃O⁺)으로 존재
브뢴스테드·라우리 산 염기 정의 (1923년)
🌟 브뢴스테드·라우리 정의
산 (Acid): 다른 물질에게 양성자(H⁺)를 주는 물질
염기 (Base): 다른 물질로부터 양성자(H⁺)를 받는 물질

→ 수용액 이외의 반응에도 적용 가능
→ NH₃의 염기성도 설명 가능
반응별 브뢴스테드·라우리 산 염기 적용
반응물 1역할반응물 2역할생성물 예시
HCl 산 (H⁺ 줌) H₂O 염기 (H⁺ 받음) H₃O⁺ + Cl⁻
NH₃ 염기 (H⁺ 받음) H₂O 산 (H⁺ 줌) NH₄⁺ + OH⁻
CH₃COOH 산 (H⁺ 줌) H₂O 염기 (H⁺ 받음) CH₃COO⁻ + H₃O⁺
💡 H₂O의 양쪽성 물질
물(H₂O)은 HCl과 반응할 때는 염기로, NH₃와 반응할 때는 산으로 작용합니다.
이처럼 조건에 따라 산과 염기 모두로 작용할 수 있는 물질을 양쪽성 물질이라고 합니다.

물 외에도 HCO₃⁻(탄산수소 이온)도 양쪽성 물질입니다.
✏️ 확인 문제 1
다음 반응에서 브뢴스테드·라우리 산과 염기로 작용한 물질을 각각 찾아보자.
(가) CH₃COOH + H₂O ⇌ CH₃COO⁻ + H₃O⁺
(나) CH₃NH₂ + H₂O ⇌ CH₃NH₃⁺ + OH⁻
정답:

(가) 산: CH₃COOH (H⁺ 줌), 염기: H₂O (H⁺ 받음)
→ CH₃COOH는 H⁺를 H₂O에게 주어 H₃O⁺ 생성

(나) 염기: CH₃NH₂ (H⁺ 받음), 산: H₂O (H⁺ 줌)
→ CH₃NH₂는 H₂O로부터 H⁺를 받아 CH₃NH₃⁺ 생성. H₂O는 산으로 작용
02
Conjugate Acid–Base Pairs · 짝산짝염기
짝산·짝염기 쌍
산 → 짝염기 (H⁺ 잃음) · 염기 → 짝산 (H⁺ 얻음) · H⁺ 하나 차이
▼
짝산·짝염기 개념
🔗 짝산·짝염기 (Conjugate Acid–Base Pair)
H⁺의 이동으로 산과 염기로 되는 한 쌍의 물질을 짝산·짝염기 쌍이라고 합니다.

• 짝산: 염기가 H⁺를 받아 생성된 물질 (염기 + H⁺)
• 짝염기: 산이 H⁺를 잃어 생성된 물질 (산 − H⁺)

→ 짝산과 짝염기는 H⁺ 1개 차이
HF 수용액 이온화 평형의 짝산·짝염기
📊 HF(aq) + H₂O(l) ⇌ F⁻(aq) + H₃O⁺(aq)
짝산·짝염기 쌍 ①: HF (산) ↔ F⁻ (짝염기) — HF가 H⁺를 잃으면 F⁻
짝산·짝염기 쌍 ②: H₂O (염기) ↔ H₃O⁺ (짝산) — H₂O가 H⁺를 얻으면 H₃O⁺

정반응: HF(산) + H₂O(염기) → F⁻ + H₃O⁺
역반응: F⁻(염기) + H₃O⁺(산) → HF + H₂O
NH₃ 수용액 이온화 평형의 짝산·짝염기
📊 NH₃(aq) + H₂O(l) ⇌ NH₄⁺(aq) + OH⁻(aq)
짝산·짝염기 쌍 ①: NH₃ (염기) ↔ NH₄⁺ (짝산) — NH₃가 H⁺를 얻으면 NH₄⁺
짝산·짝염기 쌍 ②: H₂O (산) ↔ OH⁻ (짝염기) — H₂O가 H⁺를 잃으면 OH⁻
양쪽성 물질 HCO₃⁻
⚖️ HCO₃⁻ — 산으로도, 염기로도 작용
반응 (가): HCO₃⁻(aq) + H₂O(l) ⇌ H₂CO₃(aq) + OH⁻(aq)
  → HCO₃⁻가 H₂O로부터 H⁺를 받음 → HCO₃⁻는 염기로 작용

반응 (나): HCO₃⁻(aq) + NH₃(aq) ⇌ CO₃²⁻(aq) + NH₄⁺(aq)
  → HCO₃⁻가 NH₃에게 H⁺를 줌 → HCO₃⁻는 산으로 작용

∴ HCO₃⁻ = 양쪽성 물질
✏️ 확인 문제 2
다음 이온화 평형에서 짝산·짝염기 쌍을 모두 찾아보자.
HCN(aq) + H₂O(l) ⇌ CN⁻(aq) + H₃O⁺(aq)
정답:

짝산·짝염기 쌍 ①: HCN(산) — CN⁻(짝염기)
  HCN이 H⁺를 잃으면 CN⁻

짝산·짝염기 쌍 ②: H₂O(염기) — H₃O⁺(짝산)
  H₂O가 H⁺를 얻으면 H₃O⁺

∴ 짝산·짝염기 쌍: (HCN, CN⁻), (H₃O⁺, H₂O)
✏️ 확인 문제 3
물(H₂O)의 자동 이온화 반응에서 짝산·짝염기 쌍을 찾고, 빈칸을 채워 보자.
H₂O + H₂O ⇌ H₃O⁺ + OH⁻
"H₂O는 H₃O⁺의 ( )이고, H₂O는 OH⁻의 ( )이다."
정답: H₂O는 H₃O⁺의 (짝염기)이고, H₂O는 OH⁻의 (짝산)이다.

자동 이온화 반응에서:
• 산으로 작용한 H₂O: H⁺를 주어 OH⁻ 생성 → H₂O는 OH⁻의 짝산
• 염기로 작용한 H₂O: H⁺를 받아 H₃O⁺ 생성 → H₂O는 H₃O⁺의 짝염기

소단원 02 · Ionization Constants Ka & Kb
이온화 상수와 산 염기의 세기
강산·약산·강염기·약염기의 이온화 정도 차이, 이온화 상수(Ka, Kb)의 정의와 의미, 이온화 상수를 이용한 약산·약염기 수용액의 pH 계산법을 학습합니다.
01
Strong & Weak Acids/Bases · 강산 약산 구분
산과 염기의 세기 — 강산·약산·강염기·약염기
이온화 정도에 따른 구분 · 강산은 거의 완전 이온화 · 약산은 일부만 이온화
▼
이온화 정도에 따른 분류
💪 강산 vs 약산
강산 (Strong Acid): 수용액에서 대부분 이온화하는 산 → H₃O⁺ 농도 높음
  예) HCl(염산), H₂SO₄(황산), HNO₃(질산), HBr, HI

약산 (Weak Acid): 수용액에서 일부만 이온화하는 산 → H₃O⁺ 농도 낮음
  예) CH₃COOH(아세트산), HF(플루오린화 수소), HCN(사이안화 수소), H₂CO₃(탄산)
💪 강염기 vs 약염기
강염기 (Strong Base): 수용액에서 대부분 이온화하는 염기
  예) NaOH(수산화 나트륨), KOH, Ca(OH)₂(수산화 칼슘)

약염기 (Weak Base): 수용액에서 일부만 이온화하는 염기
  예) NH₃(암모니아), CH₃NH₂(메틸아민)
이온화 정도 비교 (농도가 같은 경우)
구분이온화 정도H₃O⁺ 또는 OH⁻ 농도pH 비교 (산의 경우)
강산 거의 100% 이온화 매우 높음 pH 낮음 (산성 강함)
약산 일부만 이온화 (Ka 크기에 의존) 낮음 pH 높음 (산성 약함)
🧪 실험으로 확인 — 0.1 M HCl vs 0.1 M CH₃COOH
농도가 같은 염산과 아세트산 수용액에 pH 시험지를 넣어보면:
• HCl(aq): pH ≈ 1 (거의 모두 이온화)
• CH₃COOH(aq): pH ≈ 3 (일부만 이온화)

→ pH가 다른 이유: H₃O⁺ 생성량이 다르기 때문
✏️ 확인 문제 4
25℃에서 농도가 같은 2가지 산 HX, HY 수용액의 pH가 HX > HY이다. 산 HX와 HY의 이온화 상수(Ka)의 크기를 비교해 보자.
정답: Ka(HX) < Ka(HY)

pH가 클수록 산성이 약함 → HX의 산성이 더 약함 → HX의 이온화 정도 더 작음
→ HX의 H₃O⁺ 농도 더 낮음

Ka가 클수록 이온화 정도가 크고 산이 강하므로:
Ka(HX) < Ka(HY)
02
Ka & Kb · 이온화 상수 정의
산의 이온화 상수 Ka와 염기의 이온화 상수 Kb
Ka = [A⁻][H₃O⁺]/[HA] · Kb = [BH⁺][OH⁻]/[B] · 온도 일정하면 농도 무관
▼
산의 이온화 상수 Ka
🔢 Ka (Acid Ionization Constant)
산 HA의 이온화 평형: HA(aq) + H₂O(l) ⇌ A⁻(aq) + H₃O⁺(aq)

평형 상수 K = [A⁻][H₃O⁺] / [HA][H₂O]
수용액에서 물의 농도는 일정 → K × [H₂O] = Ka
Ka = [A⁻][H₃O⁺] / [HA]
Ka 클수록 이온화 많이 됨 = 강산    Ka 작을수록 이온화 적게 됨 = 약산
염기의 이온화 상수 Kb
🔢 Kb (Base Ionization Constant)
염기 B의 이온화 평형: B(aq) + H₂O(l) ⇌ BH⁺(aq) + OH⁻(aq)
Kb = [BH⁺][OH⁻] / [B]
Kb 클수록 강염기    Kb 작을수록 약염기
주요 이온화 상수 값 (25℃)
산 (Acid)Ka염기 (Base)Kb
HCl (염산)~10⁷ (강산)NaOH~10²⁰ (강염기)
CH₃COOH (아세트산)1.8 × 10⁻⁵NH₃ (암모니아)1.8 × 10⁻⁵
HF (플루오린화 수소)6.8 × 10⁻⁴CH₃NH₂ (메틸아민)4.6 × 10⁻⁴
HCN (사이안화 수소)6.2 × 10⁻¹⁰C₆H₅NH₂ (아닐린)7.4 × 10⁻¹⁰
H₂O1.0 × 10⁻¹⁴H₂O1.0 × 10⁻¹⁴
✅ 이온화 상수의 특징
① Ka, Kb는 온도가 일정하면 농도에 관계없이 일정한 값
② 같은 온도에서 농도가 달라도 Ka, Kb는 변하지 않음
③ Ka × Kb = Kw (물의 이온화 상수) → 짝산·짝염기 관계에서 성립
④ 온도가 높아지면 Ka, Kb 값 변화 가능
03
pH Calculation · 약산·약염기 pH 계산
이온화 상수를 이용한 pH 계산
평형표 작성 → x≪C 근사 → [H₃O⁺] 또는 [OH⁻] 계산 → pH 산출
▼
약산 수용액 pH 계산 순서
📋 3단계 계산법
1단계: 이온화 평형식 + 평형 농도표 작성 (ICE Table)
  처음 농도(I) / 농도 변화(C) / 평형 농도(E)

2단계: Ka식에 평형 농도 대입 → x 계산
  약산이므로 x≪C → (C − x) ≈ C 근사 적용 가능

3단계: pH = −log[H₃O⁺] 계산
예제 — 1.0 M CH₃COOH 수용액의 pH (25℃, Ka = 1.8 × 10⁻⁵)
🧮 계산 과정
1단계: CH₃COOH + H₂O ⇌ CH₃COO⁻ + H₃O⁺
  처음: 1.0 / 0 / 0 (M)
  변화: −x / +x / +x
  평형: (1.0−x) / x / x

2단계: Ka = x² / (1.0 − x) = 1.8 × 10⁻⁵
  x≪1.0이므로 → x² / 1.0 = 1.8 × 10⁻⁵
  x = √(1.8 × 10⁻⁵) = 4.2 × 10⁻³
  [H₃O⁺] = 4.2 × 10⁻³ M

3단계: pH = −log(4.2 × 10⁻³) = 3 − log4.2 ≈ 3 − 0.6 = 2.4
✏️ 확인 문제 5
25℃에서 0.1 M HA(aq)의 pH를 구해 보자. (단, 25℃에서 HA의 이온화 상수 Ka = 1 × 10⁻⁵)
정답: pH = 3

HA + H₂O ⇌ A⁻ + H₃O⁺ 에서
Ka = x² / (0.1 − x) ≈ x² / 0.1 = 1 × 10⁻⁵
x² = 1 × 10⁻⁶, x = 1 × 10⁻³
[H₃O⁺] = 1 × 10⁻³ M
pH = −log(10⁻³) = 3
✏️ 확인 문제 6
25℃에서 0.01 M NH₃(aq)의 pH를 구해 보자. (단, 25℃에서 NH₃의 이온화 상수 Kb = 1.8 × 10⁻⁵)
정답: pH ≈ 10.6

NH₃ + H₂O ⇌ NH₄⁺ + OH⁻
처음: 0.01 / 0 / 0 → 평형: (0.01−x) / x / x
Kb = x² / (0.01 − x) ≈ x² / 0.01 = 1.8 × 10⁻⁵
x² = 1.8 × 10⁻⁷, x = 4.2 × 10⁻⁴
[OH⁻] = 4.2 × 10⁻⁴ M
pOH = −log(4.2 × 10⁻⁴) = 4 − log4.2 ≈ 3.4
pH = 14 − pOH = 14 − 3.4 = 10.6

중화 적정 곡선
중화 적정의 원리, 강산·약산을 강염기로 적정할 때의 pH 곡선 차이, 중화점에서의 액성, 적절한 지시약 선택법을 학습합니다.
01
Neutralization Titration · 중화 적정 기본
중화 적정의 원리와 지시약
표준 용액 · 중화점 = 중화 반응 완결 지점 · 종말점 · 지시약 변색 범위
▼
중화 적정이란?
⚗️ 중화 적정 (Neutralization Titration)
중화 반응의 양적 관계를 이용하여 농도를 모르는 산이나 염기의 농도를 구하는 방법

중화 반응: H⁺ + OH⁻ → H₂O (1:1 몰비)

• 표준 용액: 농도를 정확히 아는 용액
• 중화점: 산의 H⁺ mol = 염기의 OH⁻ mol이 되는 지점 (이론적)
• 종말점: 지시약의 색이 변하는 지점 (실험적) → 중화점으로 간주
지시약의 변색 범위
지시약산성(색)변색 pH 범위염기성(색)적용 적정
메틸 오렌지 빨강 3.1 ~ 4.4 노랑 강산 + 강염기
메틸 레드 빨강 4.4 ~ 6.2 노랑 강산 + 강염기
BTB (브로모티몰 블루) 노랑 6.0 ~ 7.6 파랑 강산 + 강염기
페놀프탈레인 무색 8.3 ~ 10.0 분홍 강산/약산 + 강염기 ✅
🌸 약산을 강염기로 적정할 때 페놀프탈레인을 써야 하는 이유
약산(예: CH₃COOH)을 강염기(NaOH)로 적정할 때 중화점에서 pH > 7 (염기성) → pH ≈ 8~9

• 메틸 레드(변색 pH 4.4~6.2): 중화점 도달 전 색 변화 → 사용 불가 ❌
• 페놀프탈레인(변색 pH 8.3~10.0): 중화점 근처에서 색 변화 → 사용 가능 ✅
✏️ 확인 문제 7
25℃에서 강산을 강염기로 중화 적정할 때 중화점의 pH를 구해 보자. 또한 이때 사용할 수 있는 지시약을 모두 골라 보자. (메틸 오렌지, 메틸 레드, BTB, 페놀프탈레인)
정답: 중화점 pH = 7.0

강산과 강염기의 중화 반응으로 생성되는 염(NaCl 등)은 중성을 나타내므로 중화점 pH = 7.0

사용 가능한 지시약: 메틸 레드(4.4~6.2), BTB(6.0~7.6), 페놀프탈레인(8.3~10.0)

강산을 강염기로 중화 적정할 때
📉 강산(HCl) + 강염기(NaOH) 중화 적정 곡선
특징:
• 중화점 도달 전: 강산성 유지 (pH 낮음)
• 중화점: pH = 7.0 (NaCl 생성, 가수 분해 없음)
• 중화점 근처: pH 급격히 증가 (거의 수직)
• 중화점 이후: 강염기에 의해 pH 결정 (pH 높음)
약산을 강염기로 중화 적정할 때
📈 약산(CH₃COOH) + 강염기(NaOH) 중화 적정 곡선

• 초기 pH: 강산보다 높음 (같은 농도라도 이온화 덜 됨)
• 중화점 전: pH 완만하게 증가
• 중화점: pH > 7.0 (CH₃COONa 생성 → 가수 분해로 염기성)
• 중화점 근처: pH 급격히 증가
• 중화점 이후: 강염기에 의해 pH 결정 (강산 경우와 거의 동일)

🔑 약산 적정에서 특별한 지점 — 반중화점
약산 HA를 강염기 NaOH로 중화 적정할 때,
넣어 준 NaOH의 양이 처음 HA의 절반(½)이 되는 지점: [HA] ≈ [A⁻]

이때 Ka = [A⁻][H₃O⁺] / [HA] ≈ [H₃O⁺]
→ Ka = [H₃O⁺], 즉 pKa = pH

이 지점의 pH를 측정하면 약산의 이온화 상수(Ka)를 구할 수 있습니다.
✏️ 확인 문제 8
25℃에서 0.2 M HA(aq) 50 mL에 0.1 M NaOH(aq) 50 mL를 혼합한 수용액의 액성을 써 보자. (단, 25℃에서 HA의 이온화 상수 Ka = 1 × 10⁻⁵)
정답: 산성

HA의 mol = 0.2 × 0.05 = 0.01 mol
NaOH의 mol = 0.1 × 0.05 = 0.005 mol

중화 후 남은 HA = 0.01 − 0.005 = 0.005 mol (미중화)
생성된 NaA = 0.005 mol

HA와 A⁻가 같은 mol (0.005 mol) 존재 → 반중화점
Ka = [H₃O⁺] = 1 × 10⁻⁵ → pH = 5 → 산성

소단원 04 · Hydrolysis of Salts
염의 가수 분해
짝산·짝염기의 이온화 상수 관계(Ka × Kb = Kw), 염이 물과 반응하여 산성·중성·염기성을 나타내는 원리, 다양한 염 수용액의 액성 예측법을 학습합니다.
01
Ka × Kb = Kw · 짝산짝염기 상수 관계
짝산·짝염기의 이온화 상수 관계
Ka × Kb = Kw = 1×10⁻¹⁴ (25℃) · 산 강할수록 짝염기 약함
▼
Ka × Kb = Kw 유도
🔢 짝산·짝염기의 이온화 상수 관계 유도
산 HA의 이온화: HA + H₂O ⇌ A⁻ + H₃O⁺   Ka = [A⁻][H₃O⁺] / [HA]
짝염기 A⁻의 이온화: A⁻ + H₂O ⇌ HA + OH⁻   Kb = [HA][OH⁻] / [A⁻]

Ka × Kb = ([A⁻][H₃O⁺] / [HA]) × ([HA][OH⁻] / [A⁻])
= [H₃O⁺][OH⁻] = Kw
Ka × Kb = Kw = 1 × 10⁻¹⁴ (25℃)
짝산·짝염기 쌍에서 산의 이온화 상수 × 짝염기의 이온화 상수 = 물의 이온화 상수
이 관계의 의미
💡 산의 세기 vs 짝염기의 세기
Ka가 클수록(강산) → Kb가 작음(약한 짝염기)
Ka가 작을수록(약산) → Kb가 큼(강한 짝염기)

예시:
CH₃COOH: Ka = 1.8 × 10⁻⁵ → CH₃COO⁻의 Kb = Kw/Ka = 5.6 × 10⁻¹⁰
HCN: Ka = 6.2 × 10⁻¹⁰ → CN⁻의 Kb = Kw/Ka = 1.6 × 10⁻⁵ (CH₃COO⁻보다 강한 염기)
✏️ 확인 문제 9
25℃에서 산 HA의 이온화 상수 Ka = 1 × 10⁻⁵이다. 25℃에서 HA의 짝염기 A⁻의 이온화 상수(Kb)를 구해 보자. (단, 25℃에서 Kw = 1 × 10⁻¹⁴)
정답: Kb = 1 × 10⁻⁹

Ka × Kb = Kw
Kb = Kw / Ka = (1 × 10⁻¹⁴) / (1 × 10⁻⁵) = 1 × 10⁻⁹

💧 염의 가수 분해 (Hydrolysis of Salts)
염의 가수 분해: 염을 이루는 이온이 물과 반응하여 H₃O⁺ 또는 OH⁻를 생성하는 반응

→ 염 수용액의 액성(산성·중성·염기성)은 염을 생성한 산과 염기의 세기에 따라 결정됩니다.
각 경우별 염의 가수 분해
염 생성 반응염의 예가수 분해수용액 액성

🧪 CH₃COONa 수용액 (약산 + 강염기 → 염기성)
CH₃COONa(aq) → Na⁺(aq) + CH₃COO⁻(aq) (완전 이온화)

🧪 NH₄Cl 수용액 (강산 + 약염기 → 산성)
NH₄Cl(aq) → NH₄⁺(aq) + Cl⁻(aq) (완전 이온화)

NH₄⁺는 NH₃의 짝산 → 약산으로 작용
NH₄⁺(aq) + H₂O(l) ⇌ NH₃(aq) + H₃O⁺(aq)

→ H₃O⁺ 생성 → 산성 (pH < 7)

(나) NaF = 강염기(NaOH) + 약산(HF)의 중화 반응으로 생성 → F⁻이 가수 분해

▼
완충 용액이란?
🛡️ 완충 용액 (Buffer Solution)
완충 용액: 소량의 산이나 염기를 가해도 pH가 거의 변하지 않는 용액

완충 용액의 구성:
① 약산 + 그 약산의 짝염기 혼합 (예: CH₃COOH + CH₃COONa)
② 약염기 + 그 약염기의 짝산 혼합 (예: NH₃ + NH₄Cl)
완충 작용의 원리 — CH₃COOH/CH₃COO⁻ 완충 용액
💧 CH₃COOH + CH₃COONa 완충 용액의 원리
⚖️
초기 상태: CH₃COOH(약산)와 CH₃COO⁻(짝염기)가 거의 같은 양으로 공존
CH₃COOH ⇌ CH₃COO⁻ + H⁺ 평형 유지
🔴
강산(H⁺) 첨가 시:
CH₃COO⁻ + H⁺ → CH₃COOH
→ 짝염기(CH₃COO⁻)가 H⁺를 흡수 → pH 거의 변화 없음
🔵
강염기(OH⁻) 첨가 시:
OH⁻ + CH₃COOH → CH₃COO⁻ + H₂O
→ 약산(CH₃COOH)이 OH⁻를 중화 → pH 거의 변화 없음

NH₄⁺ (암모늄 이온) NH₃ (암모니아) NH₃ + NH₄Cl 약 pH 8.3~10.3
H₂CO₃ (탄산) HCO₃⁻ (탄산수소 이온) 혈액 내 존재 혈액 pH ≈ 7.4
⚠️ 완충 용액이 아닌 것
① 강산 + 강산의 짝염기(= 강염기) — 이미 완전 이온화, 공존 불가
예) HCl + NaCl → HCl이 강산이므로 Cl⁻와 HCl의 완충 작용 없음

② 약산과 강염기를 1:1 mol로 혼합 → 완전 중화 → 완충 용액 아님
예) CH₃COOH 1 L + NaOH 1 L (같은 농도) → CH₃COONa만 생성

(다) CH₃COOH 0.01 mol에 NaOH 0.005 mol → 절반만 중화

다음은 탄산(H₂CO₃)과 탄산수소 이온(HCO₃⁻)으로 이루어진 완충 용액의 평형 반응이다.
H₂CO₃(aq) + H₂O(l) ⇌ HCO₃⁻(aq) + H₃O⁺(aq)
완충 용액에 소량의 염기를 넣을 때 일어나는 완충 작용을 평형 이동의 원리로 설명해 보자.
정답:

소량의 염기(NaOH)를 넣으면 OH⁻가 생성됩니다.

OH⁻는 H₃O⁺와 반응: OH⁻ + H₃O⁺ → 2H₂O → H₃O⁺ 농도 감소

르샤틀리에 원리에 의해 H₃O⁺ 농도 감소 → 정반응 방향으로 평형 이동
(H₂CO₃ + H₂O → HCO₃⁻ + H₃O⁺ 방향)

→ H₂CO₃이 이온화하여 H₃O⁺를 보충 → pH가 거의 일정하게 유지
✏️ 확인 문제 13 (NH₃/NH₄Cl 완충 용액)
약염기인 NH₃와 NH₄Cl로 이루어진 완충 용액에 소량의 산을 넣었을 때 일어나는 변화를 화학 반응식으로 나타내 보자.
정답:

소량의 강산(HCl)을 넣으면 H⁺가 생성됩니다.

완충 용액에 존재하는 NH₃(약염기)가 H⁺를 흡수:
NH₃(aq) + H⁺(aq) → NH₄⁺(aq)

→ H⁺가 NH₃에 의해 소비 → H₃O⁺ 농도 거의 변화 없음 → pH 거의 일정

✏️ 확인 문제 14 (단원 종합)
다음 설명 중 옳은 것을 모두 골라보자.
ㄱ. 브뢴스테드·라우리 정의에서 산은 양성자(H⁺)를 주는 물질이다.
ㄴ. Ka가 클수록 약산이다.
ㄷ. 강산과 강염기의 중화점 pH는 7.0이다.
ㄹ. NaCN 수용액은 산성을 나타낸다. (HCN은 약산)
ㅁ. NH₃와 NH₄Cl 혼합 용액은 완충 용액이다.
정답: ㄱ, ㄷ, ㅁ

ㄱ. ✅ 옳음 — 브뢴스테드·라우리 산 = H⁺ 주는 물질

ㄴ. ❌ 틀림 — Ka가 클수록 강산. (Ka 작을수록 약산)

ㄷ. ✅ 옳음 — 강산 + 강염기 → 가수 분해 없는 염 생성 → pH = 7.0

ㄹ. ❌ 틀림 — NaCN = 약산(HCN) + 강염기(NaOH)의 염 → CN⁻가 가수 분해 → 염기성

ㅁ. ✅ 옳음 — NH₃(약염기) + NH₄Cl(짝산의 염) → 완충 용액 성립
