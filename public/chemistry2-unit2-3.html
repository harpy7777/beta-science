Hydrocarbon · 탄화수소
탄화수소와 작용기·탄화수소 유도체
C·H만으로 구성 · 작용기가 성질 결정 · 시성식으로 구분
▼
탄화수소란?
⛽ 탄화수소 (Hydrocarbon)
탄화수소: 탄소(C)와 수소(H)만으로 이루어진 탄소 화합물

주요 탄화수소:
• 메테인(CH₄): 탄소 1개, LNG(액화 천연 가스) 주성분, 가정용 연료
• 에테인(C₂H₆): 탄소 2개, 초저온 냉매, 화학 물질 합성 원료
• 프로페인(C₃H₈): 탄소 3개, LPG(액화 석유 가스) 주성분, 차량용·휴대용 연료
작용기와 탄화수소 유도체
🔧 작용기 (Functional Group)
작용기: 탄화수소의 수소 원자(−H)와 치환되어 탄소 화합물의 특성을 결정하는 부분

탄화수소 유도체: 탄화수소의 수소 원자가 다른 작용기로 치환된 화합물

같은 작용기를 가진 탄화수소 유도체들은 성질이 서로 비슷하다

시성식(Structural Formula)이란?
📝 시성식 — 작용기가 잘 드러나는 화학식
탄화수소 유도체를 나타낼 때 작용기를 명확히 표현하는 화학식 표기법

에탄올: 분자식 C₂H₆O → 시성식 C₂H₅OH (−OH 표시)
아세트산: 분자식 C₂H₄O₂ → 시성식 CH₃COOH (−COOH 표시)

※ 구조 이성질체: 분자식 같아도 구조 다르면 성질 다름 (시성식으로 구분)
예) 에탄올(C₂H₅OH) vs 다이메틸 에터(CH₃OCH₃) — 분자식 C₂H₆O로 동일

▼
알코올 (Alcohol)
물질시성식탄소 수특징 및 용도
메탄올 CH₃OH 1개 독성 있음 (실명·신경손상), 경주용 연료, 연료전지, 플라스틱 원료 (공업용)
에탄올 C₂H₅OH 2개 소독용 의약품, 향수·화장품 용매, 연료 (다양한 용도)
알데하이드 (Aldehyde)
물질시성식탄소 수특징 및 용도
폼알데하이드 HCHO 1개 자극적 냄새, 독성, 새집 증후군 원인, 플라스틱·접착제·염료 원료
아세트알데하이드 CH₃CHO 2개 자극적 냄새, 독성, 숙취 원인 물질, 플라스틱·접착제 원료
카복실산 (Carboxylic Acid)
물질시성식탄소 수특징 및 용도
폼산 HCOOH 1개 개미·꿀벌 독액 성분, 피부 부어오름·가려움 원인
아세트산 CH₃COOH 2개 식초 성분(4~6 %), 플라스틱·의약품·염료 원료
✏️ 확인 문제 2
다음을 탄소 원자 수가 1개로 같은 알데하이드와 카복실산을 각각 이름과 시성식으로 써 보자.

알데하이드 (탄소 1개): 폼알데하이드 — 시성식 HCHO
(H와 −CHO로 구성, 탄소 원자 1개)

실생활 적용 — 포도주와 식초
🍇 포도 → 포도주 → 포도 식초
1단계 (포도주 생성):
C₆H₁₂O₆ 효모→ 2C₂H₅OH + 2CO₂
포도당 → 에탄올 (발효)

∴ 식초에는 아세트산(CH₃COOH)이 4~6 % 포함됨
음주 측정기 원리
🚓 알코올 산화 반응 이용 — 음주 측정기
측정기 내부: 다이크로뮴산 칼륨(K₂Cr₂O₇, 주황색) + 황산

3C₂H₅OH + 2K₂Cr₂O₇ + 8H₂SO₄ → 3CH₃COOH + 2Cr₂(SO₄)₃ + 2K₂SO₄ + 11H₂O
                      주황색 → 녹색

에탄올에 의해 K₂Cr₂O₇(주황) → Cr₂(SO₄)₃(녹색) 색 변화 → 음주 여부 판단
✏️ 확인 문제 3
어떤 알코올(㉠)을 산화시켰더니 폼알데하이드를 거쳐 카복실산(㉡)이 생성되었다. ㉠과 ㉡은 각각 무엇인지 이름과 시성식을 써 보자.
정답:

㉠ 메탄올 — 시성식: CH₃OH
메탄올이 산화되면 폼알데하이드(HCHO) 생성

㉡ 폼산 — 시성식: HCOOH
폼알데하이드(HCHO)가 추가 산화되면 폼산(HCOOH) 생성
02
Silver Mirror Reaction & Esterification
알데하이드의 은거울 반응과 에스터화 반응
−CHO + Ag⁺ → Ag(은거울) · 카복실산 + 알코올 → 에스터 + 물
▼
알데하이드의 환원성과 은거울 반응
🪞 은거울 반응 (Silver Mirror Reaction)
알데하이드는 쉽게 산화되어 카복실산이 되므로 다른 물질을 환원시키는 성질이 있습니다.

알데하이드 + 은 이온(Ag⁺) 수용액 반응 시:
−CHO (포밀기) → −COOH (카복실기) 로 산화
+ Ag⁺ → Ag (순수한 은 석출) — 투명 용기 벽에 붙어 은거울면 형성

은거울 반응 활용: 포도당에 −CHO 존재 확인 (포도당도 은거울 반응 가능)
에스터화 반응 (Esterification)
🍎 에스터화 반응 — 카복실산 + 알코올 → 에스터 + 물
카복실산이 알코올과 반응하여 에스터(ester)를 생성하는 반응

에스터기: −COO− (카복실산과 알코올이 결합 시 형성)
🧪 아세트산 + 에탄올 → 아세트산 에틸
CH₃−C(=O)−OH + H−O−C₂H₅ 진한H₂SO₄, 가열→ CH₃COO C₂H₅ + H₂O

• 아세트산의 카복실기(−COOH)의 −OH와 에탄올의 하이드록시기의 −H가 결합 → 물 빠져나감
• 시성식: CH₃COOC₂H₅ (아세트산 에틸)
• 달콤한 사과 향 → 향료로 이용
아스피린 합성 (에스터화 반응 응용)
💊 살리실산 + 아세트산 → 아세틸 살리실산(아스피린)
살리실산의 −OH + 아세트산의 −COOH → 에스터화 반응
→ 아세틸 살리실산(아스피린) + H₂O 생성

살리실산 단점: 구토·위장 장애 부작용
아세틸 살리실산: 진통·해열·소염 효과, 위장 자극 덜함 → 세계 최대 생산 진통제
✏️ 확인 문제 4
아세트산과 반응하여 아세트산 에틸을 생성하는 물질의 이름과 시성식을 써 보자.
정답: 에탄올 — C₂H₅OH

에스터화 반응: 카복실산(아세트산) + 알코올(에탄올) → 에스터(아세트산 에틸) + 물
CH₃COOH + C₂H₅OH → CH₃COOC₂H₅ + H₂O

에탄올(C₂H₅OH)의 하이드록시기(-OH)의 H가 아세트산의 -COOH의 OH와 결합하여 물이 빠져나감
✏️ 확인 문제 5
알코올이 산화되어 카복실산을 생성할 때, 분자당 수소 원자 수와 산소 원자 수의 변화를 설명해 보자.
정답:

에탄올(C₂H₅OH) → 아세트산(CH₃COOH) 변화 비교:

• 수소 원자 수: C₂H₅OH(H 6개) → CH₃COOH(H 4개) → 2개 감소
• 산소 원자 수: C₂H₅OH(O 1개) → CH₃COOH(O 2개) → 1개 증가

∴ 산화 시 수소 원자는 감소하고 산소 원자는 증가

중합 반응과 고분자
단위체와 중합 반응의 개념, 첨가 중합·축합 중합 반응의 차이, 천연 고분자(단백질·핵산)와 합성 고분자(플라스틱·합성 섬유)를 학습합니다.
01
Polymerization · 중합 반응 원리
중합 반응 — 첨가 중합과 축합 중합
단위체 반복 결합 = 고분자 · 첨가 중합: 이중결합 끊기 · 축합 중합: 물 빠짐
▼
중합 반응 기본 개념
🔗 고분자와 중합 반응
고분자: 분자량이 매우 큰 분자 (중합체라고도 함)
단위체: 고분자를 구성하는 작은 분자
중합 반응: 단위체들이 결합하여 고분자를 만드는 반응

단위체 + 단위체 + ··· → 고분자
첨가 중합 반응 (Addition Polymerization)
📎 첨가 중합 반응
단위체의 탄소 원자와 탄소 원자 사이에 이중 결합이 존재하는 경우, 이중 결합이 끊어지면서 단위체와 단위체가 결합하여 고분자를 만드는 반응

특징: 이중 결합(C=C) → 단일 결합(C−C), 부산물 없음

예시:
• 에틸렌(CH₂=CH₂) × n → 폴리에틸렌(PE)
• 염화 바이닐(CH₂=CHCl) × n → 폴리(염화 바이닐)(PVC)
축합 중합 반응 (Condensation Polymerization)
💧 축합 중합 반응
단위체에 −OH, −COOH, −NH₂ 같은 작용기가 존재할 때, 단위체들이 결합하면서 물(H₂O)과 같은 작은 분자가 빠져나오며 고분자를 만드는 반응

특징: 부산물(H₂O, HCl 등) 생성

예시:
• 테레프탈산 + 에틸렌 글리콜 → PET + H₂O
• 헥사메틸렌다이아민 + 염화 아디프산 → 나일론 + HCl
• 아미노산 + 아미노산 → 단백질 + H₂O (펩타이드결합)
첨가 중합 vs 축합 중합 비교
구분첨가 중합축합 중합
조건이중 결합(C=C) 존재−OH, −COOH, −NH₂ 등 작용기
결합 방식이중 결합 개방 후 결합탈수 반응 (H₂O 등 빠져나옴)
부산물없음H₂O, HCl 등
예폴리에틸렌(PE), PVC단백질, 나일론, PET
✏️ 확인 문제 6
폴리에틸렌(PE)과 폴리(에틸렌 테레프탈레이트)(PET)를 만드는 중합 반응의 차이를 설명해 보자.
정답:

폴리에틸렌(PE): 첨가 중합
단위체 에틸렌(CH₂=CH₂)의 C=C 이중 결합이 끊어지면서 단위체끼리 결합
→ 부산물 없이 폴리에틸렌 생성

PET: 축합 중합

천연 고분자와 합성 고분자

▼
천연 고분자 — 단백질
🧬 단백질 (Protein)
단위체: 아미노산 (20가지 종류)
중합 방식: 축합 중합 (탈수 반응)
결합: 펩타이드결합(−CONH−)

한 아미노산의 카복실기(−COOH)와 다른 아미노산의 아미노기(−NH₂) 사이에서 H₂O가 빠져나오며 펩타이드결합 형성

아미노산 20가지 × 배열 순서 다양 → 수많은 종류의 단백질 형성 가능
천연 고분자 — 핵산 (DNA/RNA)
🧬 핵산 (Nucleic Acid)
단위체: 뉴클레오타이드 (인산 + 탄소 5개 당 + 염기)
종류: DNA (이중 나선 구조), RNA (단일 나선 구조)

DNA 염기 4종류: 아데닌(A), 타이민(T), 구아닌(G), 사이토신(C)
염기 쌍: A−T, G−C (수소 결합으로 이중 나선 안정화)
염기 배열 순서에 따라 다양한 유전 정보 저장
합성 고분자 — 플라스틱
플라스틱단위체중합 방식용도
폴리에틸렌(PE)에틸렌첨가 중합투명 용기, 포장용 랩, 비닐봉지, 병마개
폴리(염화 바이닐)(PVC)염화 바이닐첨가 중합수도관, 절연재, 인조 가죽
PET테레프탈산+에틸렌 글리콜축합 중합음료수병(페트병), 폴리에스터 섬유
합성 고분자 — 합성 섬유
🧵 나일론 (Nylon) — 폴리아마이드 섬유
단위체: 헥사메틸렌다이아민(−NH₂) + 염화 아디프산(−COCl)
중합 방식: 축합 중합 (HCl 빠져나옴)
결합: 아마이드 결합(−CONH−)

특징: 질기고 탄성 좋음, 물 흡수 후에도 강도 유지, 오랫동안 강도 유지
용도: 의류, 스타킹, 로프 등

정답:

공통점 1: 둘 다 축합 중합으로 만들어짐

▼
합성 의약품 — 아스피린(아세틸 살리실산)
💊 최초의 합성 의약품 — 아스피린
개발 배경: 살리실산(버드나무 잎 추출물)은 진통 효과 우수하나 구토·위장 장애 부작용 있음

호프만의 합성 (에스터화 반응):
살리실산 + 아세트산 → 아세틸 살리실산(아스피린) + H₂O

특징: 진통·해열·소염 효과, 위장 자극 감소
→ 세계적으로 가장 많이 생산되는 먹는 진통제
비누의 합성과 세척 원리
🧼 비누화 반응 (Saponification)
동물성 지방이나 식물성 기름 + NaOH(또는 KOH) → 비누 + 글리세롤

비누 분자의 구조:
• 친유성기: 긴 탄화수소 사슬 (−기름과 잘 섞임)
• 친수성기: 카복실산 이온 부분 (−COO⁻ Na⁺, 물과 잘 섞임)
🫧 비누의 세척 과정
① 비눗물 속 비누 분자의 친유성기가 빨래의 기름때에 달라붙음
② 친수성기는 바깥쪽을 향하면서 기름때를 둘러쌈
③ 작은 공 모양으로 기름때를 섬유로부터 떼어 냄
④ 물속으로 퍼져 나가 제거됨

→ 기름때에 달라붙는 부분: 친유성기

정답: 수산화 나트륨(NaOH) 또는 수산화 칼륨(KOH)

비누화 반응: 동물성 지방·식물성 기름 + NaOH(수산화 나트륨) 또는 KOH(수산화 칼륨)를 넣고 가열 → 비누 생성

식품 첨가물
🍭 식품 첨가물 (Food Additives)
가공식품에서 식품의 변질 방지, 모양·향 개선을 위해 사용되는 물질

특징: 가볍고 고온에서 강도 매우 큼, 원하는 모양으로 만들기 쉬움

생분해성 플라스틱
🌱 PLA와 PHA — 미래의 플라스틱

→ 생산 비용 높아 대중화 미흡

✏️ 확인 문제 10 (단원 종합)
다음 설명 중 옳은 것을 모두 골라보자.
ㄱ. 알코올의 수용액은 산성을 띤다.
ㄴ. 알데하이드는 은거울 반응을 통해 은 이온(Ag⁺)을 환원시킨다.
ㄷ. 에스터화 반응은 카복실산과 알코올이 반응하여 에스터와 물을 생성한다.
ㄹ. 첨가 중합 반응에서는 H₂O가 빠져나온다.
ㅁ. 단백질은 아미노산의 축합 중합으로 생성되는 천연 고분자이다.
정답: ㄴ, ㄷ, ㅁ

ㄱ. ❌ 틀림 — 알코올(−OH)의 수용액은 중성임 (카복실산이 산성)

ㄴ. ✅ 옳음 — 알데하이드의 −CHO → −COOH로 산화되면서 Ag⁺를 Ag로 환원

ㄷ. ✅ 옳음 — 에스터화 반응: 카복실산 + 알코올 → 에스터 + H₂O

ㄹ. ❌ 틀림 — H₂O가 빠져나오는 것은 축합 중합. 첨가 중합은 부산물 없음

ㅁ. ✅ 옳음 — 단백질 = 아미노산의 축합 중합 (펩타이드결합) = 천연 고분자
