소단원 01 · Geological Time

▼
지질시대와 화석
🪨 지질시대와 화석의 정의
지질시대: 지구가 탄생한 약 46억 년 전부터 현재까지의 기간.

화석(Fossil): 지질시대에 살았던 생물의 유해나 흔적이 지층에 보존된 것.

화석으로 환경 추정:
• 고사리 화석 → 온난하고 습한 육상 환경
• 산호 화석 → 수온이 높고 수심이 얕은 따뜻한 바다 환경
• 사슴 발자국 화석 → 당시 육상 생물이 살았음을 의미
지질시대 구분 기준
📊 지질시대를 나누는 방법
지질시대는 넓은 지역에 걸쳐 일어난 지구 환경의 급격한 변화를 기준으로 구분합니다.

4개 지질시대:
• 선캄브리아 시대 (88.2%): 화석 거의 없음
• 고생대 (6.3%): 약 5억 3900만 년 전~2억 5200만 년 전
• 중생대 (4.1%): 약 2억 5200만 년 전~6600만 년 전
• 신생대 (1.4%): 약 6600만 년 전~현재
각 지질시대의 환경과 생물
지질시대환경 특징생물 특징대표 화석
선캄브리아 시대강한 자외선, 오존층 없음. 남세균 광합성으로 산소 증가 시작단세포 생물 → 남세균 → 다세포 동물(말기)스트로마톨라이트
고생대오존층 형성 → 생물 육상 진출 가능. 말기 판게아 형성바다: 삼엽충·어류. 육상: 양치식물·양서류. 말기 대멸종삼엽충, 완족류
중생대판게아 분리, 기후 온난. 말기 대멸종바다: 암모나이트. 육지: 공룡·겉씨식물암모나이트, 공룡 화석
신생대현재와 비슷한 수륙 분포. 말기 빙하기·간빙기 반복포유류·속씨식물 번성. 화폐석. 인류 조상 출현화폐석, 매머드 화석
생물의 육상 진출
☀️ 왜 고생대 중기에 생물이 육상으로 진출했나?
남세균의 광합성 → 산소(O₂) 축적 → 오존(O₃) 형성 → 오존층이 강한 자외선 차단 → 육상 생활 가능 → 고생대 중기 식물 먼저 육상 진출, 동물 뒤따름

(다) 공룡이 번성하고 암모나이트가 바다에 서식하였다.   (라) 포유류가 번성하고 인류의 조상이 출현하였다.

▼
대멸종이란?
☄️ 대멸종(Mass Extinction)의 정의
짧은 지질학적 기간 동안 전체 생물종의 상당 비율이 동시에 사라지는 대규모 사건.
지질시대 동안 총 5번 발생. 원인: 해양 환경 변화, 소행성 충돌, 화산 활동 등
중생대 말 대멸종 원인 가설 비교
가설주요 내용뒷받침 증거
소행성 충돌설거대 소행성 충돌 → 먼지 구름이 햇빛 차단 → 기온 하강, 광합성 불가 → 먹이사슬 붕괴중생대 말 지층에서 이리듐 고농도 검출. 멕시코 칙술루브 크레이터(직경 180km)
대규모 화산 폭발설화산재·가스 방출 → 햇빛 차단, 산성비 → 기후 변화인도 데칸 고원의 대규모 현무암 분출 흔적
⚠️ 현재 학계의 견해
소행성 충돌이 방아쇠 역할을 하고, 화산 활동과 해수면 변화가 복합적으로 작용했다는 견해가 우세합니다.
🌱 대멸종 이후 생물다양성 변화
대멸종 → 생태 공간 비워짐 → 살아남은 생물들이 새로운 환경에 적응하며 폭발적으로 다양화
예) 공룡 멸종 → 포유류 전성시대 → 신생대 포유류 다양화
✏️ 확인 문제 2
중생대 말 대멸종에 대한 설명으로 틀린 것을 찾아 올바르게 고치시오.
(나) 대멸종은 매우 짧은 몇 년 사이에 발생하는 현상이다.
정답: (나)가 틀림
수정: 대멸종은 수만 년~수백만 년에 걸쳐 일어난 지질학적 사건이다.

진화와 변이의 정의
🔄 진화(Evolution)와 변이(Variation)
진화: 생물 집단의 특성이 오랜 세월 동안 여러 세대를 거치면서 변화하는 과정. 집단(개체군) 수준에서 일어남.

변이: 같은 종의 개체마다 형질이 조금씩 다른 것. 진화의 재료.
변이의 두 가지 종류
구분원인유전 여부예시
유전적 변이유전자의 차이 (돌연변이, 유성생식)유전됨 ✅흰색 깃털 공작, 토끼 털색 다양성
환경적 변이환경 요인 차이 (먹이, 온도, 빛)유전 안 됨 ✗어린 홍학의 회색 몸
⚠️ 핵심: 자연선택에서 중요한 변이
자연선택에 의한 진화는 유전적 변이를 통해서만 일어납니다.

소단원 01 · Oxidation & Reduction
산화와 환원
산소의 이동 · 전자의 이동 · 철의 제련 · 생활 속 산화환원
▼
산소의 이동으로 본 산화와 환원
용어산소 관점예시
산화(Oxidation)물질이 산소를 얻는 반응C + O₂ → CO₂, 철이 녹스는 것
환원(Reduction)물질이 산소를 잃는 반응2CuO → 2Cu + O₂
⚠️ 산화와 환원은 항상 동시에 일어난다
2CuO + C → 2Cu + CO₂
• 탄소(C): 산소를 얻음 → 산화
• 산화구리(CuO): 산소를 잃음 → 환원
전자의 이동으로 본 산화와 환원
⚡ 전자 이동 정의
산화: 물질이 전자(e⁻)를 잃는 반응
환원: 물질이 전자(e⁻)를 얻는 반응
철의 제련
🏭 철의 제련 과정 (용광로)
1단계: 2C + O₂ → 2CO (탄소 산화)
2단계: Fe₂O₃ + 3CO → 2Fe + 3CO₂
 • CO: 산화 / Fe₂O₃: 환원
✏️ 확인 문제 5
반응식 Cu + 2Ag⁺ → Cu²⁺ + 2Ag에서 산화된 물질과 환원된 물질을 각각 쓰고, 전자의 이동 방향을 설명하시오.
정답:
산화된 물질: Cu (전자를 잃어 Cu²⁺로 산화)
환원된 물질: Ag⁺ (전자를 얻어 Ag로 환원)
전자 이동: Cu → Ag⁺ 방향으로 이동

소단원 02 · Neutralization
중화 반응
산·염기 이온화 · H⁺+OH⁻→H₂O · 중화열 그래프 · 생활 속 중화 반응
▼
산과 염기의 정의
🧪 아레니우스(Arrhenius)의 산·염기 정의
산(Acid): 물에 녹아 H⁺을 내놓는 물질
예) HCl → H⁺ + Cl⁻, H₂SO₄ → 2H⁺ + SO₄²⁻

염기(Base): 물에 녹아 OH⁻을 내놓는 물질
예) NaOH → Na⁺ + OH⁻, Ca(OH)₂ → Ca²⁺ + 2OH⁻
중화 반응의 원리
💧 중화 반응(Neutralization)
핵심 반응식: H⁺ + OH⁻ → H₂O
예) HCl + NaOH → NaCl + H₂O

• H⁺ 남으면 → 산성
• OH⁻ 남으면 → 염기성
• 완전 반응 → 중성
생활 속 중화 반응 사례
사례문제중화 방법
생선 비린내 제거트리메틸아민(염기성)레몬즙(산성)으로 중화
충치 예방세균이 산성 물질 생성 → 치아 부식염기성 치약으로 중화
위산 과다위산(HCl) 과다염기성 제산제 복용
토양 산성화산성비·질소 비료석회 가루(CaO) 살포
✏️ 확인 문제 6
같은 농도의 염산과 NaOH 수용액에서 중화열이 가장 많이 발생하는 경우와 그 이유를 설명하시오.
(가) 염산 20mL + NaOH 40mL   (나) 염산 40mL + NaOH 40mL   (다) 염산 40mL + NaOH 20mL
정답: (나)에서 중화열이 가장 많이 발생
중화열은 반응하는 H⁺와 OH⁻의 수에 비례합니다. (나)에서 40mL 분량이 완전 중화되어 가장 많은 중화열이 발생합니다.

발열 반응과 흡열 반응
구분에너지 방향주변 온도원리
발열 반응(Exothermic)반응계 → 주변으로 열에너지 방출↑ 온도 상승반응물 에너지 > 생성물 에너지
흡열 반응(Endothermic)주변 → 반응계로 열에너지 흡수↓ 온도 하강반응물 에너지 < 생성물 에너지
발열 반응 사례
🔥
연소 반응
숯·가스 연소. 화학에너지 → 열+빛에너지
🤲
손난로
Fe + O₂ → Fe₂O₃. 철의 산화 반응으로 열 방출
💧
산화칼슘 + 물
CaO + H₂O → Ca(OH)₂ + 열. 100℃ 이상 발열
흡열 반응 사례
🧊
냉찜질 주머니
NH₄NO₃이 물에 용해 시 주변 열에너지 흡수 → 온도 하강
🌿
광합성
빛에너지 흡수하여 화학에너지(포도당)로 전환
✏️ 확인 문제 7
다음을 발열 반응과 흡열 반응으로 분류하시오.
(가) 광합성   (나) 철 가루와 산소의 반응   (다) 산화칼슘과 물의 반응   (라) 질산암모늄의 물 용해
정답:
발열 반응: (나) 철의 산화, (다) CaO + H₂O
흡열 반응: (가) 광합성, (라) 질산암모늄 용해
