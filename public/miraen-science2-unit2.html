☀️ 핵융합
태양 에너지의 생성 원리
수소 핵융합 · 질량 결손 · E=mc² · KSTAR
핵심 학습 내용
  • 태양의 핵융합: 4개의 수소 원자핵(양성자) → 1개의 헬륨 원자핵 (p-p 연쇄 반응)
  • 질량 결손: 반응 후 질량 < 반응 전 질량 → 감소한 질량이 에너지로 전환
  • E=mc²: 에너지 = 질량 × 빛의 속도² → 매우 적은 질량 = 엄청난 에너지
  • KSTAR: 한국 핵융합 연구소의 인공 태양 프로젝트, 1억℃ 초고온 플라즈마 유지
  • 핵융합 vs 핵분열: 핵융합(수소→헬륨, 방사성 폐기물 없음) vs 핵분열(우라늄 분열)

🌿 에너지 전환
지구에서 태양 에너지 전환
광합성·화석연료·풍력·태양광 에너지 흐름
핵심 학습 내용
  • 광합성: 태양 에너지(빛) → 화학 에너지(포도당) 저장
  • 화석연료: 고대 생물의 광합성 에너지가 지층에 저장 → 연소 시 열에너지로 방출
  • 풍력: 태양열 → 대기 순환 → 바람(역학 에너지) → 발전기 → 전기 에너지
  • 태양광: 태양 빛(빛 에너지) → 태양전지 → 전기 에너지 (광전 효과)
  • 수력: 태양열 → 증발 → 구름 → 강수 → 위치 에너지 → 전기 에너지
세특 서술 예시
"태양 에너지가 지구 생태계와 인류 문명의 다양한 에너지원으로 전환되는 경로를 에너지 흐름도로 제작하고, 각 전환 단계에서의 효율 손실을 분석하는 탐구 활동을 수행함."

전자기 유도와 발전기
패러데이 법칙 · 발전기 구조 · 교류 생성
핵심 학습 내용
  • 전자기 유도: 코일 주위의 자기장이 변화하면 유도 기전력이 발생(패러데이 법칙)
  • 렌츠의 법칙: 유도 전류는 자기장 변화를 방해하는 방향으로 흐름
  • 발전기 구조: 자석(회전자)+코일(고정자), 자석 회전 → 코일에 유도 전류 발생
  • 교류(AC): 시간에 따라 방향이 바뀌는 전류, 발전기에서 자연 발생
  • 변압기: 1차코일(입력)과 2차코일(출력)의 권수비로 전압 변환 → 장거리 송전
  • 고압 송전 이유: 전력(P=IV) 같을 때 전압↑ → 전류↓ → 열 손실(P=I²R)↓
실험 탐구 포인트
탐구: 코일과 자석을 이용해 LED에 불 켜기 실험 → 자석 삽입 속도, 코일 감은 수에 따른 유도 전류 세기 비교 → 전자기 유도 결과 분석 보고서

🏭 발전 방식 비교
화력·핵발전 에너지 전환 비교
에너지 전환 단계 · 환경 영향 · 우리나라 에너지 현황
핵심 학습 내용
  • 화력 발전: 화석연료 연소(화학 에너지) → 수증기(열 에너지) → 터빈 회전(역학 에너지) → 발전기(전기 에너지)
  • 핵발전: 우라늄 핵분열(핵에너지) → 수증기(열 에너지) → 터빈 → 발전기(전기 에너지)
  • 화력 발전 문제: CO₂·미세먼지·황산화물 등 온실기체·대기오염 물질 배출
  • 핵발전 문제: 방사성 폐기물 처리, 사고 위험(체르노빌·후쿠시마)
  • 우리나라 에너지 현황: 화력+핵발전 비중 높음 → 신재생 에너지 전환 필요
탐구 포인트
수행평가: 화력·핵발전·태양광·풍력의 에너지 효율, 환경 영향, 경제성을 비교하는 표 작성 → "우리나라 최적 에너지 믹스"를 근거 있게 제안하는 보고서

⚡ 에너지 효율
에너지 전환 효율과 계산
효율 공식 · 전기차 vs 내연기관 · 회생 제동
핵심 학습 내용
  • 에너지 보존 법칙: 에너지는 생성·소멸되지 않고 형태만 전환됨
  • 에너지 효율(%) = 유용하게 전환된 에너지 / 공급된 전체 에너지 × 100
  • 내연기관차 효율: 약 20~30% (나머지는 열 손실)
  • 전기차 효율: 약 80~90% (전기 에너지 → 역학 에너지 전환 효율 높음)
  • 회생 제동: 감속 시 역학 에너지 → 전기 에너지 재전환·배터리 충전
  • 열병합 발전: 발전 시 발생하는 폐열을 난방에 재활용 → 전체 효율 80% 이상

🌞 신재생 에너지
신재생 에너지와 미래 에너지
태양광·풍력·수소·ESS·스마트 그리드
핵심 학습 내용
  • 태양광: 태양전지(실리콘 반도체)로 빛→전기, 설치 쉽고 소음 없음, 날씨 영향 받음
  • 풍력: 바람→날개 회전→발전, 넓은 부지 필요, 소음·조류 충돌 문제
  • 수소 에너지: 연료전지(수소+산소→전기+물), 그린수소(재생에너지)·그레이수소(화석연료) 구분
  • 조력: 조수 간만의 차이 이용, 우리나라 시화호 조력발전소
  • 지열: 지구 내부 열 이용, 아이슬란드 활발히 활용
  • ESS(에너지 저장 시스템): 신재생 에너지의 간헐성 보완, 잉여 전력 저장
  • 스마트 그리드: AI 기반 전력 수요·공급 최적화 시스템
세특 서술 예시
"태양광·풍력·수소 등 신재생 에너지의 발전 원리와 장단점을 비교 분석하고, ESS와 스마트 그리드를 활용한 미래 에너지 시스템 설계를 제안하는 창의적 탐구 보고서를 완성함."
