생태계의 정의와 구성
🌿 생태계(Ecosystem)의 정의
생태계란 일정한 지역에서 생물과 생물, 생물과 비생물 환경이 서로 영향을 주고받으며 하나의 시스템을 이루는 것을 말합니다. 작은 연못도 생태계이고, 아마존 열대우림도 하나의 거대한 생태계입니다.
생물 요소 — 유기물을 얻는 방식에 따른 3가지 분류
분류유기물을 얻는 방법생태계 내 역할대표 예시
생산자광합성으로 스스로 유기물 합성무기물 → 유기물로 에너지 고정식물, 다시마, 식물 플랑크톤
소비자다른 생물의 유기물 섭취에너지를 상위 영양단계로 전달초식동물(1차), 육식동물(2~4차)
분해자사체·배설물 등 분해 후 흡수유기물→무기물 환원, 물질 순환세균, 곰팡이, 버섯
⚠️ 분해자가 없다면?
분해자가 없으면 생태계의 유기물이 축적되어 생산자가 필요한 무기 영양소(질소, 인 등)를 얻지 못합니다. 결국 생산자가 살 수 없게 되어 생태계 전체가 무너집니다. 분해자는 물질 순환의 마지막 고리이자 생태계 유지의 핵심입니다.
생물의 조직 단계 — 개체 → 생태계
개체
생물을 이루는 낱낱의 단위
→
개체군
같은 지역에 사는 같은 종의 개체 집단
→
군집
같은 지역에 사는 모든 개체군의 집합
→
생태계
군집 + 비생물적 환경의 상호작용 전체
비생물 요소가 생물에 미치는 영향 (비생물 → 생물: 작용)
비생물 요소생물에 미치는 영향적응 사례
온도생물의 형태·분포 결정, 화학 반응 속도 조절북극여우 귀 작음(보온), 사막여우 귀 큼(방열), 단풍·낙엽
물(수분)생물체 구성, 물질 이동, 생명 반응에 필수선인장의 굵은 줄기(수분 저장), 연꽃 줄기 기공
빛(광량·광주기)식물의 성장·방향 결정, 동물 수면·생식 주기 조절광굴성(빛 쪽으로 줄기 성장), 일조 시간에 따른 꽃 개화
토양식물 영양 공급, 동물 서식지·피신처 제공지렁이 활동 → 토양 비옥, 두더지 굴 생활
공기O₂ 호흡, CO₂ 광합성 원료, N₂ 고정고산 식물은 낮은 O₂ 농도에 적응
생물이 비생물 환경을 변화시키는 예 (생물 → 비생물: 반작용)
🌿 생물이 환경을 바꾸는 사례들
• 식물 광합성 → 대기 중 O₂ 농도 증가, CO₂ 농도 감소
• 지렁이의 토양 분해 활동 → 토양 비옥 (흙 속 통기성, 수분 보유력 향상)
• 소·양 등 가축의 트림 → 메테인(CH₄) 배출 → 온실 효과 강화
• 숲의 증산 작용 → 주변 습도 증가, 기온 낮아짐
→ 생물과 비생물 환경은 서로 영향을 주고받으며 순환합니다.
✏️ 확인 문제 1
다음 중 비생물 환경이 생물에 영향을 준 사례를 모두 고르시오.
ㄱ. 지렁이가 땅을 파서 토양을 비옥하게 만든다.
ㄴ. 기온이 낮아지면 개구리가 겨울잠을 잔다.
ㄷ. 건조한 사막 환경에서 선인장은 굵은 줄기에 수분을 저장한다.
ㄹ. 식물의 증산 작용으로 주변 기온이 낮아진다.
정답: ㄴ, ㄷ

ㄱ: 생물(지렁이) → 비생물(토양) 반작용
ㄴ ✅: 비생물(기온) → 생물(개구리 겨울잠) 작용
ㄷ ✅: 비생물(건조한 환경) → 생물(선인장 적응) 작용
ㄹ: 생물(식물 증산) → 비생물(기온) 반작용

생태계 평형
먹이사슬·먹이그물 · 에너지 효율 10% · 생태 피라미드 · 평형 유지 · 포식자-피식자 진동
▼
먹이사슬과 먹이그물
🍃 먹이사슬과 먹이그물의 차이
먹이사슬: 생물 사이의 먹고 먹히는 관계를 일직선으로 나타낸 것
예) 풀 → 메뚜기 → 개구리 → 뱀 → 매

먹이그물: 여러 먹이사슬이 복잡하게 연결된 그물 모양의 관계

먹이그물이 복잡할수록 생태계가 안정적인 이유:
특정 종이 감소하거나 사라져도 다른 먹이 또는 포식자가 있어 생태계 전체 균형이 유지됩니다.
에너지 효율과 생태 피라미드
📊 에너지 효율 약 10%의 법칙
한 영양단계에서 다음 영양단계로 전달되는 에너지는 약 10%에 불과합니다. 나머지 약 90%는 호흡·열에너지로 소실됩니다.

계산 예시: 생산자(식물)가 100의 에너지를 가지면
→ 1차 소비자(초식동물): 약 10
→ 2차 소비자(육식동물): 약 1
→ 3차 소비자: 약 0.1

→ 따라서 상위 영양단계로 갈수록 이용 가능한 에너지가 급격히 감소합니다.

종류내용역피라미드 가능 여부사례
에너지 피라미드각 영양단계가 보유한 에너지량불가능 ✗

생태계 평형 유지 원리
⚖️ 생태계 평형(Ecosystem Equilibrium)
생태계 평형: 생태계를 구성하는 생물의 종류와 수, 에너지와 물질의 양이 일정하게 유지되는 상태

평형 유지 메커니즘 — 포식자-피식자 진동:
피식자(토끼) 증가 → 포식자(스라소니) 먹이 풍부 → 포식자 증가
→ 피식자 감소 → 포식자 먹이 부족 → 포식자 감소
→ 피식자 다시 회복 → 약 10년 주기로 진동

→ 이처럼 먹고 먹히는 관계가 개체수를 자동으로 조절하여 생태계 평형을 유지합니다.
🌿 생태계 평형이 깨지는 경우와 회복
일시적 교란: 산불·홍수 등 자연재해 → 생물 일시 감소 → 시간이 지나면 스스로 회복 가능

심각한 교란:
• 외래종 침입 → 천적 없어 개체수 폭발 → 토착 생물 서식지·먹이 빼앗김
예) 황소개구리(우리나라), 뉴트리아, 가시박

• 서식지 파괴 → 특정 종 멸종 → 먹이그물 단순화 → 회복 어려워짐

핵심: 먹이그물이 복잡할수록 회복력(탄성)이 높아집니다.
✏️ 확인 문제 2
생태계에서 어떤 종류의 피라미드가 역피라미드(뒤집힌 삼각형)가 될 수 없는지 쓰고, 그 이유를 설명하시오.
정답: 에너지 피라미드

에너지 피라미드는 절대 역피라미드가 될 수 없습니다. 이유는 에너지 흐름이 일방향적이기 때문입니다. 한 영양단계에서 다음 영양단계로 에너지가 전달될 때 약 90%가 호흡열로 소실되므로, 상위 영양단계의 에너지는 반드시 하위 영양단계보다 적어야 합니다. 에너지는 순환하지 않고 생태계에서 소실되므로 역전이 불가능합니다.
✏️ 확인 문제 3
어느 생태계에서 초식동물이 갑자기 크게 증가했을 때, 이후 생태계 내 포식자(육식동물)와 식물의 개체수 변화를 순서대로 서술하시오.
정답:

① 초식동물 증가 → ② 포식자(육식동물)의 먹이 풍부 → 포식자 증가
③ 초식동물에게 먹히는 식물 감소 → 식물 감소
④ 포식자 증가 + 식물 감소 → 초식동물 감소
⑤ 초식동물 감소 → 포식자 먹이 부족 → 포식자 감소 / 식물 다시 회복
→ 이 과정이 반복되며 전체적으로 개체수가 진동하면서 평형 상태로 수렴됩니다.

소단원 03 · Global Environmental Change
지구 환경의 변화
온실 효과 · 지구 온난화 · CO₂ 농도 그래프 · 대기 대순환 · 기후 변화 대응
▼
온실 효과와 지구의 에너지 균형
🌡️ 온실 효과(Greenhouse Effect)의 원리
태양 에너지(단파 복사): 대기를 통과하여 지표면에 흡수됨
지구 복사(장파 복사): 지표면이 흡수한 에너지를 다시 적외선으로 방출
온실 기체의 역할: 방출된 적외선을 흡수하여 지구 표면으로 재방출 → 지구 표면 온도 상승

온실 효과는 지구를 따뜻하게 유지하는 자연적 현상입니다. 온실 효과가 없다면 지구 평균 기온은 약 −18℃로 떨어져 생명체가 살기 어렵습니다. 문제는 산업화로 온실 기체가 과도하게 증가하면서 지구 온난화가 심화되고 있다는 것입니다.
온실 기체의 종류와 특성
온실 기체주요 발생원특징
이산화탄소(CO₂)화석연료 연소, 산림 벌채온실 효과 기여도 가장 큼. 대기 중 농도 산업화 이후 급격히 증가
메테인(CH₄)가축 소화 활동, 논, 매립지CO₂보다 온실 효과 약 28배 강함
아산화질소(N₂O)비료 사용, 화석연료 연소CO₂보다 약 265배 강한 온실 효과
수증기(H₂O)자연적 증발자연 온실 효과의 가장 큰 부분 담당
마우나로아 CO₂ 농도 그래프 분석
📈 마우나로아 관측소 CO₂ 그래프 읽는 법
장기 추세 (상승): 1958년 관측 시작 이후 CO₂ 농도가 꾸준히 증가 → 화석연료 사용 증가의 증거

계절 변동 (톱니 모양):
• 봄~여름: 북반구 식물 광합성 활발 → CO₂ 흡수 → 농도 감소
• 가을~겨울: 식물 호흡·낙엽 분해 → CO₂ 방출 → 농도 증가

시사점: 장기 상승 추세는 인간 활동(화석연료 연소)이 원인이며, 계절 변동은 생태계의 CO₂ 흡수·방출이 원인입니다.
지구 온난화의 영향
⚠️ 지구 온난화가 일으키는 변화들
• 해수면 상승: 빙하·빙상 녹음 + 해수 열팽창 → 저지대 침수, 섬나라 소멸 위기
• 극한 기상 이벤트: 폭염·폭설·폭우·가뭄 빈도 증가
• 산호 백화 현상: 수온 상승으로 공생 조류 이탈 → 산호 백화·사망
• 생물 분포 변화: 식물 개화 시기 당김, 동물 서식지 북상·고도 상승
• 농업 생산성 변화: 일부 지역 생산성 감소, 새로운 병해충 확산
대기 대순환과 기후 변화 대응
🌀 대기 대순환(Atmospheric Circulation)
위도별 에너지 불균형(적도 > 극)으로 발생하는 대규모 대기 순환.

3세포 순환:
• 해들리 세포(0~30°): 적도 가열 → 상승 → 열대 수렴대(강수 많음). 아열대 하강 → 사막 형성
• 페렐 세포(30~60°): 편서풍대. 중위도 온대 기후
• 극 세포(60~90°): 극에서 하강 → 극편동풍

기후 변화 대응 국제 협약:
• 파리 협약(2015): 지구 평균 기온 상승을 산업화 이전 대비 1.5℃ 이하로 제한
• 탄소 중립(Net Zero): 배출량 = 흡수량으로 CO₂ 순 배출 0 목표
✏️ 확인 문제 4

태양 에너지의 생성 — 수소 핵융합
☀️ 수소 핵융합 반응
태양 중심부는 약 1500만 ℃의 초고온·초고압 환경입니다. 이 환경에서 4개의 수소 원자핵(양성자)이 충돌하여 1개의 헬륨 원자핵으로 융합됩니다.

핵융합 반응식 (p-p 연쇄 반응):
4 ¹H → ¹He + 2e⁺ + 2ν + 에너지

질량 결손: 융합 후 헬륨의 질량 < 반응 전 수소 4개의 질량 합
→ 감소한 질량이 에너지로 전환됩니다.
⚡ E=mc² — 아인슈타인의 질량-에너지 등가 원리
E = mc²
E: 에너지(J)   m: 질량(kg)   c: 빛의 속도(약 3×10⁸ m/s)

빛의 속도의 제곱이 곱해지므로 매우 작은 질량도 엄청난 에너지로 전환됩니다.

예) 질량 1g이 완전히 에너지로 전환되면 약 9×10¹³ J → 히로시마 원자폭탄의 약 2배 에너지
KSTAR — 대한민국의 인공 태양
🔬 KSTAR(한국형 초전도 핵융합 연구 장치)
한국핵융합에너지연구원이 개발한 토카막(Tokamak) 방식의 핵융합 장치입니다.

• 목표: 수소 동위원소(중수소·삼중수소)를 약 1억 ℃의 플라즈마 상태로 만들어 핵융합 달성
• 의의: 핵융합 에너지는 원료인 바닷물에서 중수소를 무한 추출 가능, 방사성 폐기물이 핵분열에 비해 매우 적음
• 2024년 기준: 1억 ℃ 이상에서 48초 이상 플라즈마 유지 기록 달성

전환 경로에너지 형태사례
광합성빛에너지 → 화학에너지(포도당)식물, 조류, 남세균
화석연료태양에너지가 수억 년간 화학에너지로 저장석탄·석유·천연가스
풍력태양열 → 대기 순환 → 바람(역학에너지) → 전기에너지풍력 발전기

✏️ 확인 문제 5
태양에서 에너지가 발생하는 원리를 '핵융합', '질량 결손', 'E=mc²' 용어를 모두 사용하여 설명하시오.
정답:

태양 중심부의 초고온·초고압 환경에서 4개의 수소 원자핵이 1개의 헬륨 원자핵으로 합쳐지는 핵융합 반응이 일어납니다. 이때 반응 후 생성물(헬륨)의 질량이 반응물(수소 4개)의 질량 합보다 작아지는 질량 결손이 발생합니다. 이 감소한 질량은 아인슈타인의 E=mc² 원리에 따라 엄청난 양의 에너지로 전환되며, 이것이 태양 에너지의 근원입니다.

전자기 유도 원리
🧲 전자기 유도(Electromagnetic Induction)
코일 주위의 자기장이 변화하면 코일에 전류가 유도되는 현상입니다.

패러데이(Faraday) 법칙: 유도 기전력의 크기는 코일을 지나는 자기선속의 변화율에 비례합니다.

유도 전류를 크게 하려면:
• 자석을 빠르게 움직인다
• 코일의 감은 수를 늘린다
• 더 강한 자석을 사용한다
🔄 렌츠의 법칙
유도 전류는 자기장의 변화를 방해하는 방향으로 흐릅니다.

예) 자석의 N극을 코일에 가까이 가져가면 → 코일 내부 자기장 증가 방향 방해 → 코일이 자석 쪽을 N극으로 만드는 방향으로 전류 흐름 → 반발력 발생
발전기의 구조와 교류 생성
⚙️ 발전기(Generator) 구조
• 자석(계자): 자기장 형성
• 코일(전기자): 자기장 속에서 회전하며 전류 발생
• 원리: 코일이 자기장 속에서 회전 → 코일을 지나는 자기선속 변화 → 전자기 유도 → 전류 발생

교류(AC): 발전기에서 자연스럽게 발생하는 전류. 방향이 주기적으로 바뀜 (우리나라: 60Hz, 220V)
화력 발전과 핵발전의 에너지 전환 비교
구분에너지 전환 단계장점단점
화력 발전 화석연료 연소(화학→열) → 수증기(열→역학) → 터빈·발전기(역학→전기) 기술 성숙, 안정적 공급, 건설 비용 저렴 CO₂·미세먼지·황산화물 배출, 화석연료 고갈
핵발전 우라늄 핵분열(핵→열) → 수증기(열→역학) → 터빈·발전기(역학→전기) CO₂ 배출 없음, 연료 효율 매우 높음 방사성 폐기물 처리, 사고 위험(체르노빌·후쿠시마)
⚠️ 공통점: 둘 다 수증기로 터빈을 돌린다
화력 발전과 핵발전의 가장 큰 공통점은 열에너지로 물을 끓여 수증기로 터빈을 돌리는 방식이라는 점입니다. 에너지원(화석연료 vs 핵연료)만 다를 뿐 발전 원리는 동일합니다.
고압 송전의 원리
🔌 왜 고압으로 전기를 송전할까?
전력 손실(줄 열): P_손실 = I²R (전류²×저항)

전력(P = IV)이 일정할 때 전압(V)을 높이면 → 전류(I)가 낮아짐 → I² 에 비례하는 열 손실이 크게 줄어듦

변압기: 코일 권수비로 전압을 자유롭게 변환
• 발전소 → 송전(수십만 V 초고압) → 변전소에서 강압 → 가정(220V) 공급

핵심: 전압을 10배 높이면 같은 전력에서 전류는 1/10, 손실은 1/100로 줄어듭니다.
✏️ 확인 문제 6
발전소에서 생산된 전기를 가정까지 보낼 때 고압으로 송전하는 이유를 전력 손실과 연관지어 설명하시오.
정답:

송전선의 전력 손실은 P손실 = I²R에 의해 전류의 제곱에 비례합니다. 따라서 전류를 줄이면 전력 손실을 크게 줄일 수 있습니다.

전송하는 전력(P = IV)은 동일하게 유지하면서 전압(V)을 높이면 전류(I)는 그에 반비례하여 낮아집니다. 예를 들어 전압을 10배 높이면 전류는 1/10로 줄고, 전력 손실은 (1/10)² = 1/100로 크게 감소합니다.

따라서 발전소에서 가정까지의 긴 거리를 송전할 때 변압기로 전압을 높여(수십만 V) 고압 송전하면 열 손실을 최소화할 수 있습니다.

에너지 보존 법칙과 효율
⚡ 에너지 보존 법칙
에너지는 새로 생성되거나 소멸되지 않고, 한 형태에서 다른 형태로 전환될 뿐입니다. 에너지의 총량은 항상 일정하게 보존됩니다.

에너지 효율(%) = (유용하게 전환된 에너지 / 공급된 전체 에너지) × 100

실제 기계에서는 마찰·소음·열 등으로 일부 에너지가 소실되므로 효율은 항상 100% 미만입니다.
전기차 vs 내연기관차 — 에너지 효율 비교
구분에너지 전환 경로효율특징
내연기관차 화학에너지(연료) → 열에너지 → 역학에너지(바퀴) 약 20~30%
(나머지 70~80%는 열 손실)		 배기가스(CO₂, NOx) 배출. 엔진 마찰열 손실 큼
전기차 전기에너지(배터리) → 역학에너지(전동기) 약 80~90%
(손실이 매우 적음)		 주행 중 배기가스 없음. 회생 제동으로 에너지 회수 가능
🔋 회생 제동(Regenerative Braking)
전기차가 감속할 때 전동기가 발전기로 전환되어 역학에너지 → 전기에너지로 재전환하는 시스템.

브레이크를 밟을 때 버려지던 운동에너지를 배터리에 저장하여 재사용합니다. 이것이 전기차의 에너지 효율을 더욱 높여줍니다.

ESS와 스마트 그리드
🔋 ESS와 스마트 그리드 — 신재생 에너지의 약점 보완
신재생 에너지의 약점: 간헐성 — 날씨·시간대에 따라 발전량이 불규칙

ESS(Energy Storage System, 에너지 저장 시스템):
잉여 전력(전기가 남을 때)을 저장했다가 필요할 때 공급 → 간헐성 문제 해결

스마트 그리드(Smart Grid):
AI·통신 기술로 전력 수요·공급을 실시간으로 최적화하는 지능형 전력망
• 남는 전기는 ESS 또는 전기차 배터리에 저장
• 전기차를 전력망에 연결해 부족 시 방전 (V2G 기술)

핵심: ESS + 스마트 그리드 = 신재생 에너지의 간헐성 문제를 해결하는 핵심 기술
✏️ 확인 문제 7
어떤 전기 히터에 1000 J의 전기에너지를 공급했을 때 900 J의 열에너지가 발생하고 나머지는 소음으로 소실되었다. 이 전기 히터의 에너지 효율을 구하고, 소실된 에너지의 양과 형태를 설명하시오.
정답:

에너지 효율 = (유용하게 전환된 에너지 / 공급된 전체 에너지) × 100
= (900 J / 1000 J) × 100 = 90%

소실된 에너지: 1000 - 900 = 100 J
소실된 에너지는 소음(음에너지)의 형태로 방출되었습니다.

에너지 보존 법칙에 따라 공급된 1000 J = 열에너지 900 J + 소음에너지 100 J로 모두 보존됩니다. 단, 소음으로 소실된 에너지는 유용하게 사용되지 않으므로 효율 계산에서 제외됩니다.
