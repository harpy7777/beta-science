중단원 01 · Chemical Equilibrium
화학 평형
가역 반응과 비가역 반응의 차이를 이해하고, 화학 평형 상태의 특징과 평형 상수의 의미, 반응 지수로 반응 진행 방향을 예측하는 방법을 학습합니다.
01
Reversible Reaction · 가역 반응과 화학 평형
가역 반응과 화학 평형
정반응↔역반응 · 비가역 반응 · 동적 평형 · 정반응 속도 = 역반응 속도
▼
가역 반응과 비가역 반응
🔵 가역 반응 (Reversible Reaction)
반응 조건에 따라 정반응(반응물 → 생성물)과 역반응(생성물 → 반응물)이 모두 일어날 수 있는 반응.

화학 반응식에서 가역 반응은 ⇌ 기호로 나타냅니다.

예) 황산 구리(II) 오수화물:
CuSO₄·5H₂O(s) ⇌ CuSO₄(s) + 5H₂O(l)
(파란색)                (흰색)
⚠️ 비가역 반응 (Irreversible Reaction)
한 방향으로만 일어나는 반응. 생성물이 다시 반응물로 되돌아가기 매우 어렵습니다.

예)
• 메테인 연소: CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(l)
• 마그네슘과 염산 반응: Mg(s) + 2HCl(aq) → MgCl₂(aq) + H₂(g)
가역 반응과 비가역 반응 비교
구분가역 반응비가역 반응
정의정반응·역반응 모두 가능한 방향으로만 진행
화학 반응식 기호⇌→
예시암모니아 합성, 석회 동굴 생성, CuSO₄·5H₂O 분해메테인 연소, 마그네슘+염산 반응
화학 평형 (Chemical Equilibrium)
⚖️ 화학 평형이란?
가역 반응에서 정반응 속도 = 역반응 속도가 되어 반응물과 생성물의 농도가 변하지 않고 일정하게 유지되는 상태.

이는 반응이 멈춘 것이 아니라 동적 평형(dynamic equilibrium) 상태입니다.
⚡
동적 평형
정반응·역반응이 같은 속도로 지속적으로 진행 중
📊
농도 일정
반응물·생성물의 농도가 변하지 않고 일정하게 유지
👁️
겉보기 정지
겉으로는 변화가 없는 것처럼 보이지만 반응은 진행 중
🌡️
온도에 의존
온도가 일정하면 평형 농도(평형 상수)도 일정
NO₂ ⇌ N₂O₄ 실험으로 이해하는 화학 평형
🧪 2NO₂(g) ⇌ N₂O₄(g)   (적갈색 ⇌ 무색)
NO₂만 넣은 경우: 처음엔 적갈색이 진하다가 점점 옅어지고 일정해짐
→ 정반응(NO₂→N₂O₄)이 우세하다가 평형 도달

N₂O₄만 넣은 경우: 처음엔 무색이었다가 점점 적갈색이 진해지고 일정해짐
→ 역반응(N₂O₄→NO₂)이 우세하다가 평형 도달

두 경우 모두 충분한 시간이 지나면 같은 평형 상태에 도달합니다.

정답: 정반응, 역반응
✏️ 확인 문제 2
가역 반응에서 반응물과 생성물의 농도가 변하지 않고 일정하게 유지되는 상태를 ( )(이)라고 한다.
정답: 화학 평형

화학 평형 상태에서 정반응 속도와 역반응 속도가 같으며, 겉으로는 변화가 없는 것처럼 보이는 동적 평형 상태입니다.
✏️ 확인 문제 3
화학 평형 상태에서 정반응 속도와 역반응 속도가 같다. (○, ×)
정답: ○ (맞음)

화학 평형은 정반응과 역반응이 같은 속도로 진행되는 동적 평형 상태입니다.
02
Equilibrium Constant · 평형 상수
화학 평형과 평형 상수
K = [생성물]계수 / [반응물]계수 · 온도만의 함수 · K 크기로 생성물 비교
▼
평형 상수 (Equilibrium Constant, K)
📐 평형 상수란?
일정한 온도에서 화학 반응이 평형에 도달했을 때 반응물의 농도 곱에 대한 생성물의 농도 곱의 비는 처음 농도와 관계없이 항상 일정합니다. 이 값을 평형 상수(K)라고 합니다.
aA + bB ⇌ cC + dD  →  K = [C]ᶜ[D]ᵈ / [A]ᵃ[B]ᵇ
[A],[B],[C],[D]는 각 물질의 평형 농도(mol/L) · 계수가 지수로 사용됨
평형 상수의 특징
특징내용
온도에만 의존일정한 온도에서는 처음 농도와 무관하게 항상 일정한 값
농도·압력 무관농도나 압력이 변해도 평형 상수는 변하지 않음 (온도가 같다면)
순수 고체·액체 제외순수한 고체(s)나 액체(l)는 농도가 일정하므로 평형 상수 식에 포함하지 않음
단위 생략평형 상수는 몰농도로 계산하지만 단위는 표기하지 않음
평형 상수로 알 수 있는 정보
평형 상수(K) 크기의미평형 상태
K ≫ 1생성물이 반응물보다 많음정반응 방향으로 많이 진행
K ≈ 1반응물과 생성물이 비슷중간 정도 진행
K ≪ 1반응물이 생성물보다 많음역반응 방향으로 많이 치우침
평형 상수 계산 예시
🧮 H₂(g) + I₂(g) ⇌ 2HI(g) — 평형 상수 계산
어떤 온도에서 1 L 용기에 H₂ 1 mol, I₂ 1 mol 넣어 반응 후 HI 1.6 mol 생성됨

반응표:
            H₂    +    I₂    ⇌    2HI
초기(mol): 1.0       1.0            0
반응(mol): -0.8      -0.8        +1.6
평형(mol): 0.2       0.2          1.6

평형 농도(1 L 기준): [H₂]=0.2 M, [I₂]=0.2 M, [HI]=1.6 M
K = [HI]² / ([H₂][I₂]) = 1.6² / (0.2 × 0.2) = 2.56 / 0.04 = 64

화학 반응이 평형에 도달했을 때 반응물의 농도 곱에 대한 생성물의 농도 곱의 비를 ( )(이)라고 한다.
정답: 평형 상수(K)
✏️ 확인 문제 5
반응 HNO₃(aq) ⇌ H⁺(aq) + NO₃⁻(aq)의 평형 상수식을 써 보자.
정답: K = [H⁺][NO₃⁻] / [HNO₃]

분자: 생성물의 농도를 각 계수의 거듭제곱으로 곱함 → [H⁺]¹ × [NO₃⁻]¹
분모: 반응물의 농도를 각 계수의 거듭제곱으로 곱함 → [HNO₃]¹
✏️ 확인 문제 6
평형 상수에 대한 설명으로 옳지 않은 것은?
① 온도가 일정할 때 일정한 값을 갖는다.
② 평형 상수는 온도 외에도 농도와 압력의 영향을 받는다.
③ 순수한 고체나 액체의 농도는 평형 상수 계산 시 생략한다.
④ K가 크면 생성물이 반응물보다 많은 상태에서 평형에 도달한다.
⑤ 처음 농도에 관계없이 같은 온도에서는 항상 일정한 K값을 갖는다.
정답: ②

평형 상수(K)는 온도에만 의존합니다. 농도나 압력이 변해도 온도가 동일하다면 평형 상수는 변하지 않습니다. 농도나 압력은 반응 지수(Q)를 변화시켜 평형의 이동 방향만 바꿀 뿐입니다.
03
Reaction Quotient · 반응 지수
화학 반응의 진행 방향 예측
반응 지수(Q) vs 평형 상수(K) 비교 · Q<K → 정반응 · Q>K → 역반응
▼
반응 지수 (Reaction Quotient, Q)
📊 반응 지수(Q)란?
반응 지수(Q): 화학 반응을 관찰하는 시점의 반응물과 생성물의 농도를 평형 상수식에 넣어 계산한 값.

평형 상수(K)와 형태는 동일하지만, 평형 농도가 아닌 현재 농도를 사용합니다.
Q와 K의 비교로 반응 진행 방향 예측
조건반응 진행 방향이유
Q < K→ 정반응 우세생성물을 더 만들어야 평형 도달 (Q를 K까지 증가시켜야 함)
Q = K→ 평형 상태이미 평형 도달, 정반응·역반응 속도 동일
Q > K→ 역반응 우세생성물을 줄여야 평형 도달 (Q를 K까지 감소시켜야 함)
Q < K → 정반응  |  Q = K → 평형  |  Q > K → 역반응
반응은 Q = K 가 될 때까지 진행하여 평형에 도달합니다
계산 예시
🧮 N₂(g) + 3H₂(g) ⇌ 2NH₃(g), K=200
1 L 용기에 N₂, H₂, NH₃를 각각 0.1 mol 넣었을 때 반응 방향은?

Q = [NH₃]² / ([N₂][H₂]³) = (0.1)² / (0.1 × (0.1)³) = 0.01 / 0.0001 = 100

Q(100) < K(200) → 정반응이 우세하게 진행하여 NH₃가 더 생성됩니다.
✏️ 확인 문제 7
반응물과 생성물의 현재 농도를 평형 상수식에 넣어 계산한 값을 ( )(이)라고 한다.
정답: 반응 지수(Q)
✏️ 확인 문제 8
어떤 온도에서 A(g) ⇌ B(g) 반응의 평형 상수(K)가 1일 때, 처음 농도가 [A]=1 mol/L, [B]=2 mol/L이면 어느 방향으로 반응이 진행하는가?
정답: 역반응이 우세하게 진행

Q = [B]/[A] = 2/1 = 2
Q(2) > K(1) → 역반응이 우세하게 진행하여 [B]는 감소하고 [A]는 증가합니다.
🚀 심화 문제
어떤 온도에서 PCl₃(g) + Cl₂(g) ⇌ PCl₅(g) 반응의 K=50 이다. 같은 온도에서 [PCl₃]=0.31, [Cl₂]=0.68, [PCl₅]=5.21 mol/L 일 때 반응의 진행 방향을 구하시오.
정답: 이미 평형 상태

Q = [PCl₅] / ([PCl₃][Cl₂]) = 5.21 / (0.31 × 0.68) = 5.21 / 0.2108 ≈ 24.7 ≠ 50

※ 실제 계산 시 Q ≈ 24.7로 K=50보다 작으므로 정반응이 우세합니다.
만약 Q=K=50이 나왔다면 이미 평형 상태입니다. 반응 지수를 통해 반응 방향을 판단합니다.

중단원 02 · Le Chatelier's Principle
화학 평형 이동
르샤틀리에 원리를 이해하고 농도·압력·온도 변화에 따른 화학 평형 이동 방향을 예측하며, 생명 현상과 산업 현장에서의 적용 사례를 학습합니다.
01
Concentration Change · 농도 변화와 평형 이동
화학 평형 이동과 농도 변화
조건 변화 → 평형 깨짐 → 새로운 평형 · 반응물↑ → 정반응 우세 · 생성물↑ → 역반응 우세
▼
화학 평형 이동 (Equilibrium Shift)
🔵 화학 평형 이동이란?
화학 반응이 평형 상태일 때 농도·압력·온도 같은 조건이 달라지면 평형이 깨집니다. 이에 따라 조건의 변화를 감소시키는 쪽으로 정반응 또는 역반응이 우세하게 진행해 새로운 평형을 이루는 것을 화학 평형 이동이라고 합니다.

생성물 농도 감소→ 정반응 우세생성물 농도가 증가하는 방향으로 이동
다이크로뮴산 칼륨 실험 예시
🧪 Cr₂O₇²⁻(aq) + H₂O(l) ⇌ 2CrO₄²⁻(aq) + 2H⁺(aq)
                    (주황색)                                         (노란색)

• NaOH(aq) 첨가: OH⁻가 H⁺와 반응 → [H⁺] 감소 → 정반응 우세 → 노란색으로 변화
• HCl(aq) 첨가: H⁺ 증가 → [H⁺] 증가 → 역반응 우세 → 주황색으로 변화

✏️ 확인 문제 10
화학 평형 상태에서 반응물을 첨가하면 어느 반응이 우세하게 진행하여 새로운 평형에 도달하는가?
정답: 정반응이 우세하게 진행

반응물 농도가 증가하면 이를 감소시키는 방향(정반응)이 우세하게 진행하여 새로운 평형에 도달합니다.
02
Pressure Change · 압력 변화와 평형 이동
압력 변화와 화학 평형 이동
압력↑ → 기체 분자 수 감소 방향 · 반응 전후 기체 분자 수 동일 → 압력 무관
▼
압력 변화와 화학 평형 이동
💨 기체 반응에서 압력 변화의 효과
기체는 압력이 변하면 부피(농도)가 달라집니다. 기체의 화학 반응이 평형 상태일 때:

• 압력 증가: 압력을 낮추는 방향, 즉 전체 기체 분자 수가 감소하는 방향으로 평형 이동
• 압력 감소: 압력을 높이는 방향, 즉 전체 기체 분자 수가 증가하는 방향으로 평형 이동

🌡️ 온도와 화학 평형

농도와 압력의 변화는 반응 지수(Q)만 바꾸고 평형 상수(K)는 그대로 유지됩니다.
그러나 온도 변화는 평형 상수(K) 자체가 달라집니다. 이것이 온도 변화가 나머지 변화들과 결정적으로 다른 점입니다.

• 얼음물(온도 낮춤): 온도 높이는 방향 = 발열 반응(역반응) 우세 → 붉은색으로 변화
• 뜨거운 물(온도 높임): 온도 낮추는 방향 = 흡열 반응(정반응) 우세 → 파란색으로 변화
르샤틀리에 원리 (Le Chatelier's Principle)
📋 르샤틀리에 원리 (1884년 발표)
"화학 반응이 평형 상태일 때 외부 조건(농도·압력·온도)이 변하면, 그 변화의 영향을 줄이는 방향으로 반응이 진행하여 새로운 평형을 이룬다."

• 농도 증가 → 그 물질의 농도를 감소시키는 방향으로 이동
• 압력 증가 → 기체 분자 수를 감소시키는 방향으로 이동
• 온도 증가 → 온도를 낮추는(흡열 반응 방향으로) 이동

2NO₂(g) ⇌ N₂O₄(g)   (정반응: 발열 반응)

ㄱ. 온도를 낮춘다.   ㄴ. Cl₂(g)를 첨가한다.   ㄷ. PCl₅(g)을 첨가한다.   ㄹ. 기체의 압력을 낮춘다.
정답: ㄷ, ㄹ

ㄱ. 틀림 — 정반응이 흡열 반응이므로, 온도를 낮추면 역반응(발열)이 우세
ㄴ. 틀림 — Cl₂(생성물) 첨가 → 생성물 농도 증가 → 역반응 우세
