화석과 지질 시대
🪨 화석(Fossil)이란?
화석: 지질 시대에 살았던 생물의 유해나 흔적이 지층에 남아 있는 것.

일반적으로 생물의 뼈나 껍질 같은 단단한 부분이 보존되지만, 공룡·새의 발자국, 생물이 뚫은 구멍, 알, 배설물도 화석이 될 수 있습니다.

화석으로 알 수 있는 것:
• 과거 생물의 구조와 특징
• 생물의 진화 과정
• 생물이 살았던 당시의 지구 환경

화석으로 환경을 알 수 있는 원리:
현재 생물의 서식 환경을 과거에도 적용 (동일 과정설)
• 삼엽충 화석 → 퇴적 환경·생김새로 바다에 살았음을 추정
• 산호 화석 → 현재 산호 서식 환경처럼 수온 높고 수심 얕은 바다 추정
📅 지질 시대(Geological Time Scale)
지구가 탄생한 후부터 현재까지 지질학적 활동이 일어나고 있는 시대. 화석에 나타나는 생물계의 큰 변화를 기준으로 구분합니다.

대멸종(Mass Extinction) — 원인 가설 4종 비교
☄️ 대멸종이란?
대멸종: 지구 환경이 갑작스럽게 변화하여 많은 생물이 짧은 기간 동안 광범위한 지역에서 멸종하는 것. 지질 시대 동안 총 5번 발생했습니다.
가설주요 내용타당성 근거
기후 변화설지구 기온의 급격한 변화(온난화 또는 빙하기) → 생물 서식 환경 파괴지질 기록에서 급격한 온도 변화 확인
해양 무산소설해저 무산소 상태 발생 → 해양 생물 대량 폐사특정 지층에서 무산소 환경 퇴적 증거
화산 폭발설대규모 화산 폭발 → 화산재 햇빛 차단, 산성비, 기온 변화데칸 고원(인도) 대규모 현무암 분출 흔적
소행성 충돌설거대 소행성 충돌 → 먼지 구름, 햇빛 차단, 기온 하강, 먹이사슬 붕괴칙술루브 크레이터(멕시코), 이리듐 고농도 지층
⚠️ 가설 타당성 평가 기준
가설의 타당성을 평가할 때 고려할 기준:
① 가설을 뒷받침하는 과학적 증거가 있는가?
② 전 지구적인 환경 변화를 초래하였는가?
③ 다른 과학적 사실에 어긋나지 않는가?
④ 생물을 멸종할 만큼 위력적이었는가?

현재 학계: 단일 원인보다 복합적 환경 변화가 대멸종을 초래했다는 견해가 우세합니다.
🌱 대멸종 이후 생물다양성은 어떻게 변했나?
대멸종에서 살아남아 새로운 환경에 적응한 생물은 오랜 시간에 걸쳐 다양한 종으로 진화했습니다.

예) 중생대 말 공룡 멸종 → 포유류가 비어 있는 생태 공간에서 폭발적으로 다양화 → 신생대 포유류의 전성시대

→ 이 과정이 지질 시대 동안 반복되면서 오늘날의 생물다양성이 형성되었습니다.

다음 지질 시대를 오래된 것부터 순서대로 나열하고, 각 시대의 대표 화석 생물을 1가지씩 쓰시오.
신생대 · 선캄브리아시대 · 고생대 · 중생대
정답: 선캄브리아시대 → 고생대 → 중생대 → 신생대

• 선캄브리아시대: 스트로마톨라이트
• 고생대: 삼엽충
• 중생대: 암모나이트 (또는 공룡)
• 신생대: 화폐석 (또는 매머드)

② (전 지구적 변화) ✅ 거대 소행성 충돌로 인한 먼지 구름은 전 지구적 기온 하강과 광합성 중단을 일으킬 수 있습니다.

변이(Variation)란?
🦋 변이 — 진화의 재료
변이: 같은 종의 개체 사이에 나타나는 형질의 차이.

변이가 생기는 원인: 주로 유전자의 차이 때문.
• 오랫동안 축적된 돌연변이
• 유성생식 과정에서 생식세포의 다양한 유전자 조합

비상 교과서 변이 사례:
• 무당벌레의 다양한 딱지날개 무늬와 색
• 기린의 다양한 털 무늬와 색
• 호랑나비의 다양한 날개 무늬와 색
자연선택(Natural Selection)
🌿 자연선택의 정의와 과정
자연선택: 생물 무리에서 환경에 적응하기 유리한 형질을 가진 개체가 그렇지 않은 개체에 비해 더 잘 살아남아 자손을 더 많이 남기는 과정
① 변이 존재
같은 종 개체 간 형질 차이
→
② 포식(생존 경쟁)
불리한 형질 개체 높은 비율로 소멸
→
③ 자연선택
유리한 형질 개체 생존·번식
→
④ 형질 전달
자손에게 유리한 유전자 전달
→
⑤ 진화
세대 반복 → 유리 형질 비율 증가
뻥튀기 과자 모의실험 — 자연선택 이해
🎨 비상 교과서 자연선택 모의실험 원리
실험 설계: 빨간색/초록색 도화지 위에 빨강·주황·노랑·초록색 뻥튀기 과자 각 10개씩 올려놓고, 눈에 가장 먼저 띄는 것부터 집어서 제거(포식 모의)

비유 관계:
• 과자 색의 다양성 → 변이
• 과자를 집어내는 것 → 포식(자연선택)
• 남은 과자 비율에 맞춰 과자 추가 → 증식
• 도화지 색 바꾸기 → 환경 변화

예측 결과:
빨간 도화지 → 빨간색 과자 많이 남음 (보호색 효과)
초록 도화지 → 초록색 과자 많이 남음

결론: 어떤 환경에서 유리한 형질이 다른 환경에서는 불리할 수 있습니다. 자연선택의 방향은 환경에 따라 달라집니다.
다윈의 자연선택설 — 기린의 진화 사례
단계내용
① 변이 존재기린 집단에 목 길이가 짧은 것부터 긴 것까지 다양했음
② 생존 경쟁먹이(높은 나뭇잎)를 놓고 경쟁 발생. 목이 짧은 기린은 높은 곳 잎을 먹기 불리 → 죽음
③ 자연선택목이 긴 기린만 살아남아 자손을 남김
④ 진화이 과정이 반복 → 목이 긴 기린의 비율이 세대를 거치며 증가 → 진화
생물의 진화 — 갈라파고스 핀치
🐦 갈라파고스 핀치가 보여주는 진화와 생물다양성
남아메리카 대륙에서 한 종의 핀치 무리가 갈라파고스 제도로 이동 → 섬마다 서로 다른 환경(먹이) → 각 섬의 환경에 적응한 방향으로 자연선택

섬별 핀치 부리 적응 사례:
• 곤충이 많은 섬 → 뾰족하고 가느다란 부리
• 단단한 씨앗이 많은 섬 → 크고 튼튼한 부리
• 선인장이 많은 섬 → 선인장 가시보다 긴 부리
• 부드러운 열매가 많은 섬 → 큰 부리

→ 오랜 시간 동안 서로 다른 환경에 적응하여 여러 종으로 다양해진 것. 이처럼 진화의 과정을 통해 생물다양성이 형성됩니다.
⚠️ 핵심 구분: 생물의 진화(Evolution)
진화: 오랜 시간 동안 여러 세대를 거치면서 생물이 변화하는 현상.

개체 수준 ✗ → 집단(개체군) 수준 ✅
진화는 한 개체가 변하는 것이 아니라 집단 내 유전자 빈도가 세대를 거치며 변화하는 것입니다.
생물다양성의 3가지 요소
🧬
유전적 다양성
같은 종에서 형질을 결정하는 유전자의 다양한 정도
예) 무당벌레 다양한 딱지날개 무늬
🐟
종다양성
일정한 지역에 사는 생물종의 다양한 정도
(종수 + 균등도 모두 고려)
🌲
생태계다양성
어느 지역에 존재하는 생태계의 다양한 정도
사막·초원·삼림·강·바다 등
생물다양성의 가치와 감소 원인, 보전
항목내용
가치• 생태계 안정적 유지의 원천 (먹이그물 복잡성 유지)
• 식량·의약품·의복 등 생물자원 확보
• 유전적 다양성이 높으면 병충해·기후 변화에 강한 품종 개발 가능
감소 원인• 서식지 파괴 및 단편화 (가장 주요 원인): 도로·주택 개발로 서식지 분절
• 불법 포획과 남획
• 환경 오염과 기후 변화
• 외래생물 유입 (뉴트리아, 황소개구리, 가시박 등)
보전 방법• 서식지 복원·보존, 생태통로 설치
• 야생 생물 불법 포획 금지
• 국제 협약: 생물다양성협약(CBD), CITES(멸종위기종 거래 금지), 람사르협약(습지 보호)
• 외래생물 도입 전 생태계 영향 철저 검증

다음은 갈라파고스 제도의 핀치에 대한 학생들의 대화이다. 옳지 않은 부분을 찾아 올바르게 고치시오.

학생 A: "섬마다 먹이의 종류가 달라서 핀치의 부리 모양과 크기가 다양해."
학생 B: "오랫동안 서로 다른 환경에 적응하면서 부리 모양이 다양해졌지만, 핀치는 모두 같은 종이므로 생물종이 다양해진 것은 아니야."
정답: 학생 B의 발언이 틀림

수정: 갈라파고스 제도의 핀치들은 오랜 시간 동안 섬마다 다른 환경에 적응하면서 서로 다른 종으로 다양해진 것입니다. 같은 조상에서 출발했지만 지금은 여러 개의 다른 종으로 진화·분화되었습니다.

학생 A ✅: 섬마다 먹이 종류가 달라 자연선택의 방향이 달라진 것은 맞습니다.
✏️ 확인 문제 4
뻥튀기 과자 모의실험에서 빨간색 도화지를 초록색으로 바꾸면 남아있는 과자의 색 비율은 어떻게 변하는가? 이를 자연선택과 관련지어 설명하시오.
정답:

빨간 도화지에서는 빨간색 과자가 많이 남아 있었지만, 도화지를 초록색으로 바꾸면 초록색 과자의 비율이 점점 높아지고 빨간색 과자의 비율은 낮아집니다.

이는 자연선택과 동일한 원리입니다. 빨간 도화지라는 환경에서 생존에 유리했던 형질(빨간색)이 초록색 도화지라는 새로운 환경에서는 불리해지고, 초록색이 유리해지기 때문입니다.

결론: 자연선택의 방향은 환경에 따라 달라지며, 이것이 다양한 환경에서 다양한 생물이 진화하게 된 원인입니다.

▼
자연과 인류 역사를 바꾼 화학 반응
🌍 산소가 관여하는 세 가지 핵심 반응
광합성, 철의 제련, 화석 연료의 연소는 모두 산소가 이동한다는 공통점이 있습니다. 이처럼 산소가 이동하는 반응을 산화·환원 반응이라고 합니다.

• 남세균의 광합성: CO₂ + H₂O → 포도당 + O₂ → 대기 산소 증가, 오존층 형성
• 철의 제련: 산화철 → 순수한 철 추출 → 철기 시대 개막, 문명 발달
• 화석 연료 연소: 탄소(C) + O₂ → CO₂ + 열 → 산업혁명, 현대 문명
산화와 환원 — 두 가지 정의
정의 기준산화 (Oxidation)환원 (Reduction)
산소의 이동 물질이 산소를 얻는 반응
예) 2CuO + C → 2Cu + CO₂에서 탄소(C)가 산화		 물질이 산소를 잃는 반응
예) 위 반응에서 산화구리(CuO)가 환원
전자의 이동 물질이 전자(e⁻)를 잃는 반응
예) Mg → Mg²⁺ + 2e⁻ (마그네슘이 산화)		 물질이 전자(e⁻)를 얻는 반응
예) O₂ + 4e⁻ → 2O²⁻ (산소가 환원)
⚠️ 핵심 법칙: 산화와 환원은 항상 동시에 일어난다
2CuO + C → 2Cu + CO₂ 에서
• 탄소(C): 산소를 얻음 → 산화
• 산화구리(Ⅱ)(CuO): 산소를 잃음 → 환원

한 물질이 산화되려면 반드시 다른 물질이 산소(또는 전자)를 받아 환원되어야 합니다. 산화와 환원은 반드시 짝으로 동시에 발생합니다.
전자의 이동으로 본 산화환원 — 비산소 반응도 설명 가능
⚡ 질산은 수용액 + 구리 선 반응 분석
반응식: 2Ag⁺ + Cu → 2Ag + Cu²⁺

• 구리(Cu): Cu → Cu²⁺ + 2e⁻ → 전자를 잃음 → 산화
  → Cu²⁺이 수용액에 녹아들어 수용액이 푸른색으로 변함

• 은 이온(Ag⁺): Ag⁺ + e⁻ → Ag → 전자를 얻음 → 환원
  → 구리 선 표면에 은(Ag)이 석출됨

전자의 이동 방향: Cu(구리 선) → Ag⁺(수용액 속 은 이온)
산화환원 반응의 이용 — 철의 제련과 수소 연료 전지
이용 사례화학 반응식산화된 물질환원된 물질
철의 제련
(용광로)		 1단계: 2C + O₂ → 2CO (코크스 산화)
2단계: Fe₂O₃ + 3CO → 2Fe + 3CO₂		 CO (일산화탄소 → CO₂) Fe₂O₃ → Fe (순수한 철)
수소 연료 전지 2H₂ + O₂ → 2H₂O (화학에너지 → 전기에너지) H₂ (수소 → H₂O) O₂ (산소 → H₂O)
광합성 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂ H₂O → O₂ (산화) CO₂ → 포도당 (환원)
메테인 연소 CH₄ + 2O₂ → CO₂ + 2H₂O CH₄ → CO₂ (산화) O₂ → H₂O (환원)

검은색 산화 구리(Ⅱ)와 탄소를 혼합하여 가열하는 실험(2CuO + C → 2Cu + CO₂)에 대한 설명으로 옳은 것을 모두 고르시오.
ㄱ. 연결된 석회수는 변화가 없다.
ㄴ. 시험관 속에서 붉은색 물질이 생성된다.
ㄷ. 탄소가 환원되어 이산화탄소가 발생한다.
ㄹ. 산화구리(Ⅱ)는 산소를 잃어 구리로 환원된다.
정답: ㄴ, ㄹ

ㄱ ✗: CO₂가 생성되어 석회수가 뿌옇게 변합니다.
ㄴ ✅: 환원된 구리(Cu)가 붉은색 물질로 생성됩니다.
ㄷ ✗: 탄소(C)는 산소를 얻어 산화됩니다. (탄소 → CO₂)
ㄹ ✅: 산화구리(Ⅱ)는 산소를 잃어 구리로 환원됩니다.

산성·염기성이 나타나는 이유 — 이온 확인 실험
🔬 비상 교과서 이온 확인 실험
산성을 나타내는 이온 확인:
질산칼륨 수용액에 적신 푸른색 리트머스 종이 위에 묽은 염산에 적신 실을 올리고 전류를 흘려주면 → 리트머스 종이가 (-)극 쪽으로 붉게 변함
→ 산성은 양이온(H⁺) 때문에 나타남

염기성을 나타내는 이온 확인:
수산화나트륨 수용액에 적신 실 사용 → 리트머스 종이가 (+)극 쪽으로 푸르게 변함
→ 염기성은 음이온(OH⁻) 때문에 나타남
중화 반응의 원리
💧 중화 반응(Neutralization)
중화 반응: 산과 염기가 반응하여 물(H₂O)이 생성되는 반응

핵심 알짜 이온 반응식: H⁺ + OH⁻ → H₂O

수소 이온과 수산화 이온의 반응 개수비: 1:1

혼합 후 용액의 성질:
• H⁺ 남음 → 산성
• OH⁻ 남음 → 염기성
• H⁺ = OH⁻ 완전 반응 → 중성 (중화점)
중화열과 온도 변화 — 비상 교과서 실험 해석
🌡️ 묽은 염산 + 수산화나트륨 수용액 혼합 (A~E 홈판 실험)
비상 교과서 탐구에서는 묽은 염산과 NaOH 수용액의 부피를 달리하여 혼합함:

홈 A: 염산 2mL + NaOH 10mL → 염기성 (NaOH 과잉)
홈 B: 염산 4mL + NaOH 8mL → 염기성
홈 C: 염산 6mL + NaOH 6mL → 중성 (완전 중화 → 온도 최고)
홈 D: 염산 8mL + NaOH 4mL → 산성
홈 E: 염산 10mL + NaOH 2mL → 산성

온도 변화 패턴:
• 중화 반응이 일어날 때 중화열 발생 → 온도 상승
• 반응한 H⁺와 OH⁻ 수가 많을수록 중화열이 많이 발생
• 홈 C(완전 중화)에서 온도 최고. A→C는 온도 상승, C→E는 온도 하강
중화 반응의 이용 사례
사례문제 상황중화 방법
충치 예방세균이 음식 분해 → 산성 물질 → 치아 부식염기성 치약으로 양치 → 산성 중화
중성지 제조근대 종이(산성지) → 시간 지나면 황변·파손펄프를 염기로 중화하여 중성지 제조 → 보존 수명 연장
생선 비린내 제거트리메틸아민(염기성) → 비린내 원인레몬즙(산성)으로 염기성 중화
위산 과다(제산제)위산(HCl, 산성) 과다 → 속쓰림염기성 제산제 복용 → 위산 일부 중화
토양·호수 산성화산성비·비료 → 토양·호수 산성화 → 생물 피해염기성 석회 가루 살포 → 산성 중화
✏️ 확인 문제 6
같은 농도의 묽은 염산과 수산화나트륨 수용액을 다음과 같이 혼합하였다. 혼합 용액의 최고 온도가 가장 높은 것을 고르고, 그 이유를 설명하시오.
(가) 염산 40mL + NaOH 20mL   (나) 염산 30mL + NaOH 30mL   (다) 염산 20mL + NaOH 40mL
정답: (나) 염산 30mL + NaOH 30mL

중화열의 양은 반응하는 H⁺와 OH⁻의 수(몰수)에 비례합니다.

(가): 염산 20mL 분량만 중화 (NaOH가 부족) → 20mL 분량의 중화열
(나): 염산 30mL ↔ NaOH 30mL 완전 중화 → 30mL 분량의 중화열 (가장 많음)
(다): NaOH 20mL 분량만 중화 (염산이 부족) → 20mL 분량의 중화열

(나)에서 H⁺와 OH⁻가 가장 많이 반응하므로 중화열이 가장 많이 발생하여 온도가 가장 높아집니다.

• 열에너지를 흡수하면 → 주변 온도 하강
에너지를 방출하는 변화 vs 흡수하는 변화
구분변화 종류구체적 사례주변 온도

산화칼슘과 물을 이용한 음식 조리 — 비상 교과서 탐구
🍳 가열 장치 없이 음식 조리하기
화학 반응식: CaO + H₂O → Ca(OH)₂ + 열에너지

산화칼슘(CaO)과 물이 반응하면 100℃ 이상의 강한 발열 반응 발생.

탐구 설계 (비상 교과서 예시 - 달걀 삶기):
① 비커에 산화칼슘 50g이 든 팩을 넣고 물 200mL 추가
② 발생하는 열로 팩 위의 달걀을 70℃ 이상 15분 유지하여 조리
③ 전기·불 없이 음식 조리 가능 (전투식량·야외 발열 도시락 활용)

안전 주의: CaO + H₂O 반응 시 많은 수증기와 열 발생 → 화상 위험. 보안경·실험용 장갑·면장갑 착용 필수.
생명 현상과 지구 현상에서의 에너지 출입
🌊 자연 속 에너지 출입의 역할
• 세포 호흡(발열): 포도당 + O₂ → CO₂ + H₂O + 에너지 → 생명 활동에 필요한 에너지 공급
• 광합성(흡열): 빛에너지 흡수 → 화학에너지(포도당) 저장 → 모든 생명의 에너지 원천
• 태풍 발달(발열): 바다에서 수증기 증발(흡열) → 상승 기류에서 응결(발열) → 열에너지 방출하며 태풍 발달
• 도로 물 뿌리기(흡열): 물의 기화 → 주변 열에너지 흡수 → 시원해짐

[실험] 나무판 위에 물을 뿌린 뒤 질산암모늄과 수산화바륨이 든 삼각 플라스크를 올려놓고 두 물질을 잘 섞은 후 들어올리면 나무판이 플라스크에 달라붙은 채로 들어 올려진다.
정답:

질산암모늄과 수산화바륨이 반응할 때 주변으로부터 열에너지를 흡수하는 흡열 반응이 일어납니다. 이로 인해 유리 기구 주변의 온도가 0℃ 이하로 낮아지고, 유리 기구와 나무판 사이에 뿌려놓은 물이 얼어 나무판이 유리 기구에 달라붙게 됩니다.
