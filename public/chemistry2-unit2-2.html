소단원 01 · Oxidation–Reduction Reaction
산화·환원 반응
전자의 이동과 산화수 개념, 산화수 변화로 산화·환원 판단, 산화제·환원제 구분, 반쪽 반응식을 이용한 산화·환원 반응식 완성법을 학습합니다.
01
Oxidation & Reduction · 전자 이동
산화·환원 반응과 전자의 이동
산화 = 전자 잃음 · 환원 = 전자 얻음 · 산화와 환원은 항상 동시에 발생
▼
산화·환원의 정의
⚡ 산화·환원이란?
반응물 사이에 전자를 주고받는 반응을 산화·환원 반응이라고 합니다.

• 산화 (Oxidation): 전자를 잃는 것 → 산화수 증가
• 환원 (Reduction): 전자를 얻는 것 → 산화수 감소

핵심 원칙: 산화와 환원은 항상 동시에 일어납니다. (전자를 잃는 반응이 있으면 전자를 얻는 반응도 반드시 존재)
Mg 연소 반응 예시
🔥 2Mg(s) + O₂(g) → 2MgO(s)
• Mg: 전자 2개를 잃음 → Mg²⁺로 산화됨 → 산화
• O₂: 전자 2개를 얻음 → O²⁻로 환원됨 → 환원

이온 결합 물질에서는 전자의 이동이 명확하게 드러납니다.
✏️ 확인 문제 1
다음 반응에서 산화된 물질과 환원된 물질을 전자의 이동으로 각각 설명해 보자.
Zn(s) + CuSO₄(aq) → ZnSO₄(aq) + Cu(s)
정답:

산화된 물질: Zn
Zn → Zn²⁺ + 2e⁻ : 아연이 전자 2개를 잃고 아연 이온으로 산화됨

환원된 물질: Cu²⁺ (CuSO₄ 속 구리 이온)
Cu²⁺ + 2e⁻ → Cu : 구리 이온이 전자 2개를 얻어 구리로 환원됨
02
Oxidation Number · 산화수 규칙 7가지
산화수의 정의와 결정 규칙
전기음성도 큰 원자가 공유 전자쌍 독점 가정 · 7가지 규칙 · 화합물 합=0
▼
산화수(Oxidation Number)란?
🔢 산화수의 정의
화합물을 구성하는 원자 중 전기음성도가 큰 원자가 공유 전자쌍을 모두 가진다고 가정할 때 각 원자가 가지는 전하입니다.

→ 공유 결합 물질에서도 산화·환원을 명확히 설명할 수 있게 해 주는 개념
산화수 결정 규칙 7가지
규칙내용예시
규칙 1원소(단체)를 구성하는 원자의 산화수 = 0H₂, O₂, Cu, Zn → 각 원자 산화수 0
규칙 2단원자 이온의 산화수 = 이온의 전하Na⁺: +1, Mg²⁺: +2, Cl⁻: −1
규칙 3화합물에서 F의 산화수 = −1 (항상)HF: F = −1
규칙 4화합물에서 H의 산화수 = +1 (금속 수소화물 제외: −1)HCl: H = +1, NaH: H = −1
규칙 5화합물에서 O의 산화수 = −2 (과산화물 제외: −1)H₂O: O = −2, H₂O₂: O = −1
규칙 6화합물 구성 원자의 산화수 합 = 0CO₂: (+4)×1 + (−2)×2 = 0
규칙 7다원자 이온 원자들의 산화수 합 = 이온의 전하SO₄²⁻: (+6) + (−2)×4 = −2
산화수 계산 예제
🧮 HNO₃에서 N의 산화수
H = +1, O = −2 (규칙 4, 5)
화합물 전체 합 = 0 (규칙 6)
(+1) + N + (−2)×3 = 0
N − 5 = −1 → N = +5 → N의 산화수: +5
🧮 CH₄에서 C의 산화수
H = +1 (규칙 4)
C + (+1)×4 = 0 → C = −4
(탄소가 수소보다 전기음성도 크므로 공유 전자쌍을 모두 가짐)
✏️ 확인 문제 2
다음 물질에서 각 원자 또는 이온의 산화수를 구해 보자.
(1) Ag    (2) Ca²⁺    (3) HNO₃    (4) MnO₄⁻
정답:

(1) Ag: 0 — 원소(단체)이므로 산화수 = 0

(2) Ca²⁺: +2 — 단원자 이온의 산화수 = 이온의 전하

(3) N: +5 — H = +1, O = −2이므로 (+1) + N + (−2)×3 = 0 → N = +5

(4) Mn: +7 — O = −2, 이온 전하 = −1
Mn + (−2)×4 = −1 → Mn = −1 + 8 = +7
03
Oxidizing/Reducing Agent · Half-Reaction
산화제·환원제와 반쪽 반응식
산화제=자신은 환원 · 환원제=자신은 산화 · 반쪽 반응식 4단계 완성법
▼
산화제와 환원제
⚗️ 산화제 vs 환원제
산화제 (Oxidizing Agent): 자신은 환원되면서 다른 물질을 산화시키는 물질
  → 산화수가 감소하는 원자를 포함

환원제 (Reducing Agent): 자신은 산화되면서 다른 물질을 환원시키는 물질
  → 산화수가 증가하는 원자를 포함
Fe₂O₃ + CO 반응에서 산화제·환원제
🏭 Fe₂O₃(s) + 3CO(g) → 2Fe(s) + 3CO₂(g)
Fe의 산화수: +3 → 0 (감소) → Fe₂O₃ = 산화제
C의 산화수: +2 → +4 (증가) → CO = 환원제

※ 같은 물질(SO₂ 등)도 반응에 따라 산화제/환원제 모두로 작용 가능
반쪽 반응식 4단계 완성법
📋 반쪽 반응식 완성 4단계
1단계: 산화 반응식과 환원 반응식으로 분리
2단계: 각 반쪽 반응식에서 반응 전후 원자 수 맞추기
3단계: 각 반쪽 반응식에서 잃거나 얻은 전자 수 표기
4단계: 산화에서 잃은 전자 수 = 환원에서 얻은 전자 수가 되도록 계수 맞추고, 두 식 더하기
KI + Cl₂ → KCl + I₂ 반응식 완성 예제
🔬 반쪽 반응식 완성 과정
산화 반응식: 2I⁻(aq) → I₂(s) + 2e⁻   (I: −1 → 0, 산화수 증가)
환원 반응식: Cl₂(g) + 2e⁻ → 2Cl⁻(aq)   (Cl: 0 → −1, 산화수 감소)

두 식의 전자 수가 같으므로 그대로 더하면:
전체: 2KI(aq) + Cl₂(g) → 2KCl(aq) + I₂(s)
✏️ 확인 문제 3
다음 산화·환원 반응에서 산화제와 환원제로 작용한 물질을 찾아보자.
2Fe₂O₃(s) + 3C(s) → 4Fe(s) + 3CO₂(g)
정답:

Fe의 산화수: +3 → 0 (감소) → Fe₂O₃이 환원됨 → Fe₂O₃ = 산화제
C의 산화수: 0 → +4 (증가) → C가 산화됨 → C = 환원제

화학 전지
화학 에너지를 전기 에너지로 전환하는 화학 전지의 원리, 볼타 전지와 다니엘 전지, 일차 전지와 이차 전지, 다양한 실용 전지의 특징을 학습합니다.
01
Chemical Cell Principle · 볼타·다니엘
화학 전지의 원리 — 볼타 전지와 다니엘 전지
화학 에너지 → 전기 에너지 · (−)극=산화·전자 방출 · (+)극=환원·전자 수용
▼
화학 전지의 정의
🔋 화학 전지 (Chemical Cell)
화학 전지: 산화·환원 반응을 이용하여 화학 에너지를 전기 에너지로 전환하는 장치

구성 요소 3가지:
① (−)극 — 산화 전극: 전자를 잃는 산화 반응 발생
② (+)극 — 환원 전극: 전자를 얻는 환원 반응 발생
③ 전해질: 이온이 이동할 수 있는 용액 (전기 회로 완성)
볼타 전지 (Voltaic Cell)
⚡ 볼타 전지 구조 — Zn판 + Cu판 + H₂SO₄(aq)
(−)극
Zn판 (아연): Zn(s) → Zn²⁺(aq) + 2e⁻   산화
아연이 전자를 잃고 이온화 → 전자가 도선을 통해 (+)극으로 이동
(+)극
Cu판 (구리): 2H⁺(aq) + 2e⁻ → H₂(g)   환원
수소 이온이 전자를 받아 H₂ 기체 발생
⚠️ 볼타 전지의 단점 — 분극 현상
분극 (Polarization): (+)극(구리판)에서 발생한 H₂ 기체가 전극 표면을 덮어 환원 반응을 방해 → 전압 저하

→ 이 문제를 해결한 것이 다니엘 전지
다니엘 전지 (Daniell Cell)
🔋 다니엘 전지 구조 — Zn|ZnSO₄(aq) ∥ CuSO₄(aq)|Cu
(−)극
Zn판: Zn(s) → Zn²⁺(aq) + 2e⁻   산화
아연판 질량 감소, ZnSO₄(aq)에 담금
(+)극
Cu판: Cu²⁺(aq) + 2e⁻ → Cu(s)   환원
구리판 질량 증가, CuSO₄(aq)에 담금
전체 반응: Zn(s) + Cu²⁺(aq) → Zn²⁺(aq) + Cu(s)
염다리 역할: 두 수용액 연결, 이온 이동으로 전하 균형 유지 (음이온→(−)극, 양이온→(+)극)
✏️ 확인 문제 4
다니엘 전지에서 전류가 흐를 때 아연판과 구리판의 질량은 각각 어떻게 변하는지 설명해 보자.
정답:

아연판 (−)극: Zn → Zn²⁺ + 2e⁻ 산화 반응 → 아연 원자가 이온으로 용해 → 아연판 질량 감소

구리판 (+)극: Cu²⁺ + 2e⁻ → Cu 환원 반응 → 구리 이온이 금속으로 석출 → 구리판 질량 증가

소단원 03 · Standard Reduction Potential
표준 환원 전위
반쪽 전지의 표준 환원 전위(E°) 개념, 표준 수소 전극 기준, 표준 전지 전위 계산법(E°전지 = E°환원 − E°산화)을 학습합니다.
01
Standard Reduction Potential · E° 개념
표준 환원 전위와 표준 수소 전극
E° 크면 환원 잘 됨 · 표준 수소 전극 E° = 0.00 V · 기준으로 측정
▼
표준 환원 전위(E°)란?
📊 표준 환원 전위 (Standard Reduction Potential, E°)
서로 다른 반쪽 전지를 연결할 때 산화·환원 반응의 경향성을 비교하는 척도입니다.

• E° 클수록: 환원되기 쉬움 → (+)극(환원 전극)이 됨
• E° 작을수록: 산화되기 쉬움 → (−)극(산화 전극)이 됨

측정 기준 — 표준 수소 전극:
25℃, H⁺ 농도 1 M, H₂ 기체 압력 1 atm, 백금(Pt) 전극 → E° = 0.00 V
Cu 반쪽 전지와 Zn 반쪽 전지 측정
🔬 표준 수소 전극과 Cu 반쪽 전지 연결 (전위차 = 0.34 V)
표준 수소 전극에서 산화 (H₂→2H⁺+2e⁻): E° = 0.00 V
구리 반쪽 전지에서 환원 (Cu²⁺+2e⁻→Cu): E° = ?

측정 전위차 = 0.34 V → Cu 반쪽 전지 E° = +0.34 V
🔬 표준 수소 전극과 Zn 반쪽 전지 연결 (전위차 = 0.76 V)
아연 반쪽 전지에서 산화 (Zn→Zn²⁺+2e⁻): E° = ?
표준 수소 전극에서 환원 (2H⁺+2e⁻→H₂): E° = 0.00 V

측정 전위차 = 0.76 V → Zn 반쪽 전지 E° = −0.76 V

Cell Potential · 전지 전위 계산
표준 전지 전위 계산
E°전지 = E°환원 전극 − E°산화 전극 · 전지 표기법 · 전압 높이려면 E° 차이 크게

표준 전지 전위 계산 공식
E°전지 = E°환원 전극 − E°산화 전극
환원 전극 = (+)극     산화 전극 = (−)극
다니엘 전지 표준 전지 전위 계산
🧮 Zn | Zn²⁺(aq) ∥ Cu²⁺(aq) | Cu 전지
환원 전극 (+)극: Cu²⁺ + 2e⁻ → Cu   E° = +0.34 V
산화 전극 (−)극: Zn → Zn²⁺ + 2e⁻   E° = −0.76 V

E°전지 = 0.34 − (−0.76) = 0.34 + 0.76 = 1.10 V
전지 표기법
📝 전지 표기 규칙
왼쪽: 산화 전극 구성 물질    | : 서로 다른 상 경계
오른쪽: 환원 전극 구성 물질    ∥: 염다리

예) 다니엘 전지: Zn(s) | Zn²⁺(aq) ∥ Cu²⁺(aq) | Cu(s)
✏️ 확인 문제 6
산화 전극에서 Ni(s)→Ni²⁺(aq)+2e⁻, 환원 전극에서 Cu²⁺(aq)+2e⁻→Cu(s) 반응이 일어나는 전지의 표준 전지 전위를 구해 보자.
정답: E°전지 = 0.60 V

E°(Cu²⁺/Cu) = +0.34 V (환원 전극)
E°(Ni²⁺/Ni) = −0.26 V (산화 전극)

E°전지 = E°환원 − E°산화 = 0.34 − (−0.26) = 0.60 V
✏️ 확인 문제 7
다니엘 전지에서 구리 반쪽 전지(Cu|CuSO₄(aq)) 대신 사용했을 때 표준 전지 전위를 높일 수 있는 반쪽 전지를 표에서 모두 찾아보자.
(가) Fe(s)|FeSO₄(aq)   (나) Ag(s)|Ag₂SO₄(aq)   (다) Ni(s)|NiSO₄(aq)
정답: (나) Ag 반쪽 전지

소단원 04 · Electrolysis
전기 분해
전기 에너지로 비자발적 산화·환원 반응을 일으키는 전기 분해의 원리, 용융액과 수용액의 전기 분해, 도금 원리와 산업적 활용을 학습합니다.
01
Electrolysis Principle · 전기 분해 원리
전기 분해의 원리 — 용융액과 수용액
(+)극=산화 전극 · (−)극=환원 전극 · 비자발적 산화·환원 반응 유도
▼
전기 분해(Electrolysis)란?
⚡ 전기 분해
물질에 전기 에너지를 가하여 비자발적인 산화·환원 반응을 일으키는 것

화학 전지와 반대 방향:
• 화학 전지: 화학 에너지 → 전기 에너지 (자발적 반응)
• 전기 분해: 전기 에너지 → 화학 에너지 (비자발적 반응)

전기 분해 장치에서 극의 역할:
(+)극 = 산화 전극: 음이온·물이 전자 잃음 → 산화
(−)극 = 환원 전극: 양이온·물이 전자 얻음 → 환원
NaCl 용융액의 전기 분해
🧪 NaCl(l) 전기 분해 (다운스 전지)
(+)극
2Cl⁻(l) → Cl₂(g) + 2e⁻   산화
염화 이온이 전자를 잃어 Cl₂ 기체 발생
(−)극
2Na⁺(l) + 2e⁻ → 2Na(l)   환원
나트륨 이온이 전자를 얻어 나트륨 금속 생성
전체: 2Na⁺(l) + 2Cl⁻(l) → 2Na(l) + Cl₂(g)
CuSO₄ 수용액의 전기 분해
🔵 수용액 전기 분해 — 이온과 H₂O 모두 참여 가능
수용액 전기 분해 시 전해질의 이온과 H₂O 중 더 쉽게 산화·환원되는 물질이 반응합니다.

(+)극: 2H₂O(l) → O₂(g) + 4H⁺(aq) + 4e⁻   (H₂O 산화, SO₄²⁻보다 쉽게 산화)
(−)극: 2Cu²⁺(aq) + 4e⁻ → 2Cu(s)   (Cu²⁺ 환원, H₂O보다 쉽게 환원)

전체: 2Cu²⁺(aq) + 2H₂O(l) → 2Cu(s) + O₂(g) + 4H⁺(aq)
물의 전기 분해
💧 2H₂O(l) → 2H₂(g) + O₂(g)
순수한 물은 전류 흐름 없음 → 소량의 KNO₃ 등 전해질 첨가하여 전기 분해

(+)극(산화): 2H₂O(l) → O₂(g) + 4H⁺(aq) + 4e⁻
(−)극(환원): 4H₂O(l) + 4e⁻ → 2H₂(g) + 4OH⁻(aq)

→ H₂와 O₂가 2:1 부피비로 발생
✏️ 확인 문제 8
물에 황산 나트륨(Na₂SO₄)을 녹여 전기 분해했을 때 (+)극과 (−)극에서 각각 생성되는 물질은 무엇인가?
정답:

Na₂SO₄ 수용액에서 Na⁺, SO₄²⁻, H₂O 존재

(+)극: SO₄²⁻보다 H₂O가 산화되기 쉬움
2H₂O → O₂(g) + 4H⁺ + 4e⁻ → O₂ 기체 발생

(−)극: Na⁺보다 H₂O가 환원되기 쉬움
4H₂O + 4e⁻ → 2H₂(g) + 4OH⁻ → H₂ 기체 발생

Electroplating & Refining · 도금과 정제
전기 분해의 활용 — 도금과 구리 정제
도금: 물체=(−)극, 도금 금속=(+)극 · 구리 정제: 불순물 포함 구리=(+)극
▼
금속 도금 (Electroplating)
✨ 도금이란?
물체 표면에 금속을 얇게 입히는 것. 부식 방지, 심미적 가치 향상에 활용

도금 설치 방법:
• 도금할 물체 → (−)극 연결 (금속이 환원되어 석출)
• 도금시킬 금속판 → (+)극 연결 (금속이 산화되어 이온 공급)
🥈 은도금 예시 — K[Ag(CN)₂](aq) 용액
(+)극
Ag(s) → Ag⁺(aq) + e⁻   산화
은판에서 은이 이온으로 용해
(−)극
Ag⁺(aq) + e⁻ → Ag(s)   환원
은 이온이 환원되어 물체 표면에 석출
구리 정제 (Copper Refining)
🏭 전기 분해로 구리 정제
불순물(Zn, Fe, Ni 등)을 포함한 구리에서 순수한 구리를 분리

설치:
• 불순물 포함 구리 → (+)극
• 순수한 구리 → (−)극
• CuSO₄(aq) 전해질

원리: (+)극에서 불순물과 Cu 모두 산화되어 이온화되지만,
(−)극에서는 표준 환원 전위가 큰 Cu²⁺만 환원되어 석출
Zn, Fe, Ni → Cu보다 E° 작음 → 이온 상태 유지 → 제거됨
Ag, Au, Pt → Cu보다 E° 크고 산화 안됨 → (+)극 아래 침전
✏️ 확인 문제 9
물체의 표면에 금속을 도금할 때 물체를 전원 장치의 (−)극에 연결하는 까닭은 무엇인가?
정답:

(−)극은 환원 전극이므로 금속 이온이 전자를 얻어 금속으로 환원됩니다.

도금할 물체를 (−)극에 연결하면 용액 속 금속 이온(예: Ag⁺)이 물체 표면에서 환원되어 금속 상태로 석출되기 때문입니다.

▼
세포호흡과 광합성
🌱 세포호흡 (Cell Respiration)
C₆H₁₂O₆(aq) + 6O₂(g) → 6CO₂(g) + 6H₂O(l)

• 포도당(C₆H₁₂O₆): 탄소의 산화수 증가 → 환원제로 작용, 산화됨
• O₂: 산소의 산화수 0 → −2 (감소) → 산화제로 작용, 환원됨
☀️ 광합성 (Photosynthesis)
식물은 빛에너지를 이용하여 CO₂와 H₂O로부터 포도당 합성

• CO₂: 탄소의 산화수 감소 → 산화제로 작용, 환원됨
• H₂O: 산소의 산화수 증가 → 환원제로 작용, 산화됨

∴ 광합성은 세포호흡의 역반응
바이타민 C의 항산화 작용
💊 바이타민 C (Vitamin C) — 항산화제
우리 몸에서 생성되는 활성 산소는 세포를 손상시키는 산화제입니다.

바이타민 C는:
• 전자를 내어주는 특성 → 쉽게 산화됨
• 활성 산소와 결합하여 빠르게 제거 → 산화 반응 억제 (항산화 작용)
• 즉, 바이타민 C = 환원제로 작용
질소 고정과 암모니아 합성
🌾 질소 고정 (Nitrogen Fixation)
공기 중 N₂를 NH₃ 등의 질소 화합물로 변환하는 과정

N₂(g) + 3H₂(g) → 2NH₃(g)

• N₂: 질소의 산화수 0 → −3 (감소) → 산화제로 작용, 환원됨
• 뿌리혹세균이 자연적으로, 하버-보쉬법으로 산업적으로 실시
