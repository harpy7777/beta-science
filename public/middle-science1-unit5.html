Force Definition · 힘의 정의
힘의 정의와 알짜힘
과학에서 힘의 의미 · 화살표 표시 3요소 · 알짜힘 계산 · 힘의 평형
▼
과학에서 말하는 힘이란?
💡 힘의 정의
과학에서 말하는 힘이란 물체의 모양이나 운동 상태(속력·운동 방향)를 변화시키는 원인이다.

힘의 크기 단위: N (뉴턴, Newton)
→ 영국 물리학자 뉴턴(Newton, I., 1642~1727)의 이름에서 유래
힘을 화살표로 나타내는 방법
📍
작용점
힘이 작용하는 지점 (●로 표시)
📏
힘의 크기
화살표의 길이로 표시 (클수록 길게)
➡️
힘의 방향
화살표가 가리키는 방향
나란하게 작용하는 두 힘의 알짜힘
두 힘의 방향알짜힘의 크기알짜힘의 방향예시
같은 방향 두 힘의 크기를 더한 것 두 힘과 같은 방향 30N + 15N = 45N
반대 방향 큰 힘에서 작은 힘을 뺀 것 큰 힘의 방향 30N - 15N = 15N (큰 힘 방향)
⚖️ 힘의 평형 (Equilibrium)
한 물체에 두 힘이 작용할 때 알짜힘이 0이면 두 힘은 힘의 평형을 이룬다.

힘의 평형 조건: 두 힘의 크기가 같고, 방향이 서로 반대이며, 같은 직선 위에 있어야 한다.

예: 책상 위 화분 → 중력(아래) = 책상이 떠받치는 힘(위) → 알짜힘 = 0 → 정지
✏️ 확인 문제 1
물체에 3N과 2N의 힘이 반대 방향으로 작용할 때 알짜힘의 크기와 방향은?
정답:

• 크기: 3N - 2N = 1N
• 방향: 3N 힘의 방향 (더 큰 힘의 방향)

💡 반대 방향으로 나란하게 작용하는 두 힘의 알짜힘 = 큰 힘 − 작은 힘, 방향은 큰 힘쪽
02
Gravity · 중력
중력 — 지구가 물체를 당기는 힘
중력의 방향 · 무게 vs 질량 비교 · 천체에 따른 중력 차이
▼
중력이란?
🌍 중력 (Gravity)
지구가 물체를 당기는 힘.

방향: 항상 지구 중심 방향(연직 아래 방향)으로 작용

예: 공을 가만히 놓으면 아래로 떨어짐, 번지 점프, 달이 지구 주위를 도는 것, 지구에 공기가 존재하는 것
무게 vs 질량 비교
구분무게 (Weight)질량 (Mass)
정의물체에 작용하는 중력의 크기물질이 가진 고유한 양
단위N (뉴턴)kg, g
측정 도구용수철저울, 가정용 저울양팔저울, 윗접시저울
장소에 따른 변화변한다 (중력이 다르므로)변하지 않는다
지구에서 관계질량 1 kg → 무게 약 9.8 N
🌕 달에서의 무게와 질량
달의 중력은 지구 중력의 약 1/6
→ 달에서 무게는 지구의 약 1/6로 줄어든다
→ 달에서 질량은 지구와 똑같다 (질량은 장소와 무관)
✏️ 확인 문제 2
질량이 10kg인 물체를 달에 가져갔을 때 질량과 무게는 각각 어떻게 되는가? (달의 중력은 지구의 1/6)
정답:

• 질량: 10 kg (변하지 않음 — 질량은 장소에 관계없이 일정)
• 무게: 지구에서 10 kg × 9.8 N/kg = 98 N의 1/6 = 약 16.3 N

💡 무게는 중력의 크기이므로 중력이 작은 달에서는 작아지지만, 질량은 물질의 고유한 양이므로 달에서도 변하지 않는다.
03
Elastic Force · 탄성력
탄성력 — 원래 모양으로 되돌아가려는 힘
탄성·탄성력의 정의 · 크기와 방향 · 생활 속 탄성력 이용
▼
탄성과 탄성력이란?
🌀 탄성력 (Elastic Force)
• 탄성: 변형된 물체가 원래 모양으로 되돌아가려는 성질
• 탄성력: 변형된 물체가 원래 모양으로 되돌아가려는 힘
• 탄성체: 탄성을 가지는 물체 (용수철, 고무줄, 고무 띠 등)
탄성력의 크기와 방향
📐 탄성력의 특징
• 방향: 탄성체가 원래 모양으로 되돌아가려는 방향 (변형 방향과 반대 방향)
  예: 용수철을 오른쪽으로 당기면 탄성력은 왼쪽으로 작용

• 크기: 탄성체가 변형된 정도(늘어난 길이)에 비례
  → 더 많이 늘일수록 탄성력이 커진다
탄성력을 이용하는 예
📎 집게 🖊️ 볼펜 용수철 💇 머리끈 📱 스마트기기 거치대 🏃 장대높이뛰기 🏹 활과 화살 🎵 기타 줄
✏️ 확인 문제 3
고무줄을 양쪽에서 잡아당길 때 고무줄에 작용하는 탄성력의 방향을 설명하시오.
정답:

탄성력은 고무줄이 원래 모양으로 되돌아가려는 방향, 즉 당기는 힘의 반대 방향으로 작용한다.

→ 왼쪽으로 당기는 손에는 오른쪽 방향의 탄성력이, 오른쪽으로 당기는 손에는 왼쪽 방향의 탄성력이 작용한다.
04
Friction · 마찰력
마찰력 — 운동을 방해하는 힘
마찰력의 정의·방향·크기 요인 · 마찰력을 크게/작게 이용하는 예
▼
마찰력이란?
🛑 마찰력 (Friction)
두 물체의 접촉면에서 물체의 운동을 방해하는 힘.

방향: 물체의 운동 방향(미끄러짐)과 반대 방향으로 작용
마찰력의 크기에 영향을 주는 요인
요인관계이유
접촉면의 거칠기 거칠수록 마찰력 크다 빙판(매끄러움) < 흙길(거침)
물체의 무게 무거울수록 마찰력 크다 가벼운 상자 < 무거운 상자 끌기
마찰력의 이용 사례
🏂
마찰력 작게 이용
스노보드 — 눈 위에서 마찰력이 작아 쉽게 미끄러짐
🚲
마찰력 크게 이용
자전거 제동 장치 — 마찰력 커져 바퀴 멈춤
🥌
마찰력 조절 이용
컬링 — 브러시로 얼음판의 마찰력을 조절하여 스톤 방향·속도 제어
✏️ 확인 문제 4
마찰력의 크기는 접촉면이 ___수록, 물체의 무게가 ___수록 크다. 빈칸을 채우시오.
정답:

마찰력의 크기는 접촉면이 거칠수록, 물체의 무게가 무거울수록 크다.

→ 빙판에서 잘 미끄러지는 이유: 빙판은 매끄러워 마찰력이 작기 때문
→ 무거운 짐이 끌기 힘든 이유: 무거울수록 마찰력이 크기 때문
05
Buoyancy · 부력
부력 — 물체를 밀어 올리는 힘
부력의 정의·방향·크기 결정 요인 · 배·잠수함·헬륨풍선의 원리
▼
부력이란?
🌊 부력 (Buoyancy)
액체나 기체가 그 속에 있는 물체를 위로 밀어 올리는 힘.

방향: 중력과 반대 방향 (위쪽)

크기 측정법: 공기 중 용수철저울 눈금 − 물속 용수철저울 눈금 = 부력의 크기
부력의 크기에 영향을 주는 요인
📦 물에 잠긴 부피 ↑ → 부력 ↑
물에 잠긴 물체의 부피가 클수록 부력이 크다.

예: 금속 덩어리를 물에 넣으면 가라앉음 (잠긴 부피 작아 부력 < 중력)
→ 같은 금속으로 큰 배 모양을 만들면 물에 뜸 (잠긴 부피 커져 부력 ≥ 중력)
부력과 중력의 관계
상태조건예시
물 위에 뜸부력 ≥ 중력배, 튜브, 구명조끼
공중에 떠 있음부력 = 중력 (기체 중)헬륨 풍선, 비행선
가라앉음부력 < 중력금속 덩어리
🚢 잠수함의 원리
잠수함의 부피가 일정하므로 부력의 크기는 변하지 않는다. 잠수함 내부의 공기탱크에 바닷물을 채워 중력 크기를 조절하여 가라앉거나 뜬다.

• 가라앉기: 공기탱크에 바닷물 채움 → 중력 > 부력
• 떠오르기: 압축 공기로 바닷물 밀어냄 → 중력 < 부력
✏️ 확인 문제 5
화물선이 비어 있을 때와 짐을 가득 실었을 때, 두 경우에 작용하는 부력의 크기를 비교하고 그 이유를 설명하시오.
정답:

짐을 가득 실은 화물선에 더 큰 부력이 작용한다.

이유: 짐을 실으면 화물선이 무거워져 물에 더 깊이 잠기므로 물에 잠긴 부피가 커진다. 부력의 크기는 물에 잠긴 부피가 클수록 크므로 부력이 더 크게 작용한다.

(화물선은 항상 중력 = 부력인 힘의 평형 상태에서 떠 있으므로, 짐이 더 무거우면 더 깊이 잠겨 부력도 커진다.)

힘과 운동
알짜힘과 운동 상태 변화, 힘의 방향에 따른 속력·방향 변화, 힘의 평형 상태를 학습합니다.
01
Net Force & Motion · 알짜힘과 운동
물체에 작용하는 힘과 운동 상태 변화
알짜힘의 방향에 따른 속력·방향 변화 3가지 경우
▼
알짜힘이 작용하면 운동 상태가 변한다
🎯 핵심 원리
물체에 알짜힘(≠0)이 작용하면 물체의 속력이나 운동 방향이 변한다.

반대로, 알짜힘이 0이면 물체는 운동 상태가 변하지 않는다 (정지 또는 등속 직선 운동).
알짜힘 방향에 따른 운동 변화 3가지
1
운동 방향과 같은 방향으로 알짜힘 작용
속력만 빨라진다 (운동 방향 변하지 않음)
예: 공을 가만히 놓아 아래로 떨어질 때 — 중력이 운동 방향(아래)과 같은 방향 → 점점 빨라짐
예: 쇼트트랙에서 뒤 선수가 앞 선수를 밀 때 → 속력 빨라짐
2
운동 방향과 반대 방향으로 알짜힘 작용
속력만 느려진다 (운동 방향 변하지 않음)
예: 공을 위로 던졌을 때 — 중력이 운동 방향(위)의 반대 방향 → 점점 느려짐
예: 잔디 위를 굴러가는 공 — 마찰력이 운동 반대 방향 → 느려져 정지
3
운동 방향과 나란하지 않은 방향으로 알짜힘 작용
비스듬한 방향: 속력과 운동 방향 모두 변한다
수직 방향: 운동 방향만 변한다 (속력 변하지 않음)
예: 비스듬히 던진 농구공 → 속력과 방향 모두 변함
예: 인공위성 — 중력이 운동 방향에 수직 → 속력 일정, 방향만 계속 바뀜
알짜힘 방향속력 변화운동 방향 변화대표 예
운동 방향과 같음빨라짐없음자유낙하, 쇼트트랙 밀기
운동 방향과 반대느려짐없음위로 던진 공, 잔디 위 공
비스듬한 방향변함변함비스듬히 던진 공, 그네
수직 방향없음변함인공위성, 회전목마
✏️ 확인 문제 6
다음 운동에서 속력과 운동 방향의 변화를 쓰시오.

(가) 비스듬히 차올린 공의 운동
(나) 회전목마의 운동
(다) 아래로 떨어지는 다이빙 선수의 운동
정답:

(가) 속력과 운동 방향 모두 변함
→ 중력이 비스듬한 방향으로 작용하기 때문

(나) 운동 방향만 변함 (속력은 일정)
→ 원의 중심 방향으로 힘이 작용 (운동 방향에 수직)

(다) 속력만 빨라짐 (운동 방향 변하지 않음)
→ 중력이 운동 방향(아래)과 같은 방향으로 작용
02
Force Equilibrium · 힘의 평형
힘의 평형 — 알짜힘 = 0, 운동 상태 불변
힘의 평형 조건 · 평형 사례 · 정지 vs 등속운동
▼
힘의 평형이란?
힘의 평형 → 알짜힘 = 0 → 운동 상태 변하지 않음
두 힘의 크기가 같고, 방향이 반대이며, 같은 직선 위에 있을 때
힘의 평형 조건
⚖️ 평형의 3가지 조건
① 두 힘의 크기가 같다
② 두 힘의 방향이 서로 반대이다
③ 두 힘이 같은 직선 위에 있다

→ 세 조건을 모두 만족해야 평형!
힘의 평형 사례
🪴
책상 위 화분
중력(아래) = 책상의 수직 항력(위) → 알짜힘 0 → 정지
🎈
공중에 뜬 헬륨 풍선
중력(아래) = 부력(위) → 알짜힘 0 → 정지
📦
밀어도 안 움직이는 상자
미는 힘 = 마찰력 → 알짜힘 0 → 정지 유지
⚖️
수직추가 달린 실
중력(아래) = 실이 추를 당기는 힘(위) → 정지
💡 평형 상태 = 반드시 정지는 아니다
알짜힘 = 0이면 운동 상태가 변하지 않을 뿐이다.
→ 이미 정지해 있으면: 계속 정지
→ 이미 운동 중이면: 등속 직선 운동 (같은 속도로 계속 운동)

예: 마찰이 없는 빙판에서 일정한 속도로 미끄러지는 스케이터 → 알짜힘 = 0 → 등속 운동
✏️ 확인 문제 7
바닥에 놓인 물체를 힘껏 밀었는데 물체가 움직이지 않았다. 이때 물체에 작용하는 힘의 평형 관계를 설명하시오.
정답:

물체를 미는 힘과 물체에 작용하는 마찰력이 서로 힘의 평형을 이루기 때문이다.

• 미는 힘의 방향: 앞쪽
• 마찰력의 방향: 뒤쪽 (운동 방향 반대)
• 두 힘의 크기가 같고 방향이 반대 → 알짜힘 = 0 → 운동 상태 변하지 않음(정지 유지)
💡 서술형 심화
인공위성은 지구 주위를 일정한 속력으로 원운동한다. 이때 인공위성의 속력과 운동 방향은 어떻게 변하는지, 작용하는 힘과 관련지어 설명하시오.
예시 답안:

인공위성에는 지구 중력이 운동 방향에 수직인 방향(지구 중심 방향)으로 작용한다.

• 속력: 힘이 운동 방향과 수직이므로 변하지 않는다
• 운동 방향: 힘이 계속 운동 방향과 수직으로 작용하므로 계속 바뀐다

결과적으로 인공위성은 속력은 일정하고 방향만 계속 변하는 등속 원운동을 한다.
