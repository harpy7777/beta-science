  • 에너지 문제: 태양광·풍력 등 신재생 에너지 기술로 화석연료 의존도 감소
  • 환경 문제: 탄소 포집·저장(CCS) 기술, 플라스틱 분해 미생물 개발
  • 보건 문제: mRNA 백신(코로나19), 유전자 치료, AI 진단 보조 시스템
  • 식량 문제: GMO·수직 농장·스마트팜으로 식량 생산 효율 향상
  • 생활 안전: 지진 조기 경보 시스템, AI 기반 사고 예방 기술
  • 과학의 필요성 논거: 근거 기반 의사 결정, 문제 예측·예방, 지속가능한 발전

🦠 감염병 진단
감염병 진단과 과학의 유용성
PCR · 항원항체 반응 · 코로나19 대응 사례
핵심 학습 내용
  • PCR 진단: 병원체의 특이적 DNA/RNA를 증폭하여 검출 → 고감도·고정확도
  • 항원·항체 반응: 병원체 항원에 특이적인 항체와 결합 → 신속항원검사(RAT) 원리
  • 항원 vs 항체 검사: 항원(현재 감염 여부) vs 항체(과거 감염 이력·면역 상태)
  • 유전정보 분석: 변이 바이러스 추적, 전파 경로 역학 조사에 활용
  • mRNA 백신: 바이러스 유전 정보를 이용해 면역 반응 유도 → 빠른 개발 가능
  • 빅데이터+AI: 확진자 동선 추적, 전파 예측, 의약품 개발 기간 단축

📡 데이터 수집
데이터 수집과 피지컬 컴퓨팅
센서·MCU·IoT 기반 실시간 환경 데이터 탐구
핵심 학습 내용
  • 피지컬 컴퓨팅: 물리적 환경의 데이터를 수집·처리·출력하는 시스템
  • 구성 요소: 센서(입력) → MCU/컴퓨터(처리) → 출력 장치(LED·모터·디스플레이)
  • 환경 센서 종류: 온도·습도·CO₂·미세먼지(PM2.5)·소음·조도 센서
  • 사물 인터넷(IoT): 사물이 인터넷에 연결되어 실시간 데이터 교환
  • 스마트팜: 센서로 온도·습도·이산화탄소 농도를 자동 조절 → 최적 생육 환경 유지
  • 탐구 활동: 교실 미세먼지 농도를 1주일간 측정·분석 → 개선 방안 도출

🗄️ 빅데이터
빅데이터 활용 사례와 문제점
보건·환경·교통 활용 vs 개인정보·편향 데이터 문제
핵심 학습 내용
  • 빅데이터의 4V: Volume(대용량)·Variety(다양성)·Velocity(속도)·Veracity(정확성)
  • 보건 분야: 전자 의무 기록(EMR) 분석으로 맞춤 치료, 감염병 예측 시스템
  • 환경·기상 분야: 기상 위성 데이터 분석으로 태풍·폭염 예보 정확도 향상
  • 교통 분야: 교통 흐름 빅데이터로 신호 최적화, 사고 예측
  • 문제점: 개인정보 유출, 편향 데이터로 인한 차별적 알고리즘, 디지털 격차
  • 공공 데이터 포털: 정부 빅데이터를 공개하여 민간에서 활용 가능

🦾 AI와 로봇
인공지능과 로봇의 발전
머신러닝·딥러닝·자율주행·산업 로봇의 영향
핵심 학습 내용
  • 머신러닝: 데이터에서 패턴을 학습하여 예측·판단하는 AI 기법
  • 딥러닝: 인공신경망을 다층으로 쌓아 이미지·음성·언어 인식에 활용
  • 의료 AI: X-ray·CT 영상 분석으로 암 조기 진단, 의약품 개발 기간 단축
  • 자율주행: 카메라·LiDAR·레이더로 주변 감지 → AI가 실시간 판단·제어
  • 산업 로봇: 반복적·위험한 작업 대체 → 생산성 향상, 일자리 구조 변화
  • 사회적 영향: 효율성·안전성 향상 vs 일자리 감소·윤리 문제
세특 서술 예시
"딥러닝 기반 의료 AI의 진단 정확도 데이터를 분석하고, 의료진과 AI의 협력 가능성을 탐구하였으며, 자율주행 자동차의 사고 책임 귀속 문제를 법적·윤리적 관점에서 논증함."

🌐 발전과 한계
과학기술의 발전이 가져온 변화와 한계
디지털 격차 · 기술 의존성 · 일자리 변화 분석
핵심 학습 내용
  • 긍정적 변화: 의사소통 편리, 정보 접근 용이, 생산성 향상, 의료·교육 품질 향상
  • 일자리 변화: 단순 반복 직업 감소 vs 새로운 기술 직업 창출 (직업 구조 변화)
  • 디지털 격차: 기술 접근성의 사회·경제적 차이 → 정보 불평등 심화
  • 기술 의존성: 스마트폰·소셜 미디어 중독, 개인 정보 처리 능력 약화
  • 환경 문제: 전자 폐기물(e-waste), 데이터센터의 막대한 전력 소비
  • 해결 방안: 디지털 리터러시 교육, 포용적 기술 정책, AI 윤리 가이드라인
